# 付费锁定点分析 + 投稿分类建议

## 一、付费锁定点分析

### 触发时机
Step 3 生成发布素材时，在候选书名/简介/封面提示词之后执行。

### 分析流程

```
1. 读取全部章节的末尾3行（章尾钩子），记录每章的"悬念强度"
   悬念强度评估标准：
   - ★★★★★：生死悬念/重大真相即将揭露/关键人物状态剧变
   - ★★★★☆：强烈好奇心（"他是谁？""里面有什么？"）
   - ★★★☆☆：情感钩子（甜蜜/心疼/期待）
   - ★★☆☆☆：平淡过渡
   - ★☆☆☆☆：无钩子

2. 识别"情绪投资完成点"——读者已经：
   - 认识并喜欢主角团（通常前3-5章）
   - 经历过至少一次情感高潮（心疼/心动/紧张）
   - 被核心悬念钩住（主线谜团展开）

3. 计算最佳锁定位置：
   推荐锁定点 = 情绪投资完成后的第一个 ★★★★★ 悬念章末尾
   免费占比目标：25%-35%（行业最佳实践）

4. 输出三个方案：
   - 推荐方案：最佳平衡点（免费占比 ~30%）
   - 保守方案：多给几章免费（~25%处的强悬念点）
   - 激进方案：少给几章免费（~20%处的强悬念点，需确保情绪投资已够）

5. 统计数据：
   - 免费区章数和字数
   - 付费区章数和字数
   - 免费占比百分比
   - 锁定点章尾原文（悬念钩子）
   - 首个付费章标题和内容概述（说明付费后立刻获得什么）
```

### 输出格式（写入 publish-materials.md）

```markdown
## 付费锁定建议

### 推荐方案
- 免费章节：第1-{N}章（{free_words}字，占比{percent}%）
- 锁定点：第{N}章末 "{最后一句话}"
- 首个付费章：第{N+1}章 "{章名}"（{内容概述}）
- 理由：{为什么这里是最佳锁定点}

### 保守方案
- 免费章节：第1-{M}章（{free_words}字，占比{percent}%）
- 锁定点钩子："{最后一句话}"

### 激进方案
- 免费章节：第1-{K}章（{free_words}字，占比{percent}%）
- 锁定点钩子："{最后一句话}"
- 风险：{可能的风险，如读者投入不够深}

### 平台设置建议
- 单章定价：最低档（降低首次付费心理门槛）
- 首个付费章标题建议：{更有悬念感的替代标题}
```

## 二、投稿分类建议

### 触发时机
Step 3 生成发布素材时，与付费锁定分析一起执行。

### 分析流程

```
1. 读取 meta.json 的 genre、style、world 字段
2. 读取 config/style.md 了解语言风格
3. 读取前3章确认：
   - 主视角角色性别（男/女）→ 决定男频/女频
   - 核心卖点类型（爽/甜/虐/燃/悬疑）
   - 爽点来源（打脸/逆袭/恋爱/权谋/升级）

4. 判断频道：
   - 女主视角 + 感情线为主 + 甜宠/言情/宫斗 → 女频
   - 男主视角 + 升级/战斗/商战为主 → 男频
   - 双视角/群像 → 看核心读者画像（谁更想翻下去）

5. 匹配百度作家平台分类体系：

   女频一级分类：
   - 都市言情（甜宠/豪门/职场/婚恋）
   - 古代言情（宫斗/种田/穿越/重生）
   - 幻想言情（仙侠/玄幻/异能/星际）
   - 悬疑言情（推理/灵异/探案）

   男频一级分类：
   - 都市（异能/商战/兵王/神医）
   - 玄幻（东方玄幻/异界大陆/高武）
   - 仙侠（修真/仙侠/洪荒）
   - 历史（架空/穿越/争霸）
   - 科幻（未来/星际/末世）
   - 悬疑（灵异/探险/推理）

6. 输出判断依据表：
   | 维度 | 本书特征 | 指向 |
   |------|---------|------|
   | 视角 | {主视角角色+人称} | {男频/女频} |
   | 核心卖点 | {卖点描述} | {分类} |
   | 情感线权重 | {占比描述} | {分类} |
   | 爽点类型 | {爽点描述} | {分类} |
   | 男/女主塑造 | {塑造特点} | {分类} |
```

### 输出格式（写入 publish-materials.md）

```markdown
## 投稿分类建议

### 分类结论
- 频道：{男频/女频}
- 一级分类：{如 都市言情}
- 二级分类：{如 甜宠}
- 推荐标签：{标签1}、{标签2}、{标签3}、{标签4}、{标签5}

### 判断依据
| 维度 | 本书特征 | 指向 |
|------|---------|------|
| 视角 | ... | ... |
| 核心卖点 | ... | ... |
| 情感线权重 | ... | ... |
| 爽点类型 | ... | ... |
| 男/女主塑造 | ... | ... |

### 百度作家平台操作指引
1. 登录 zuojia.baidu.com
2. 创建作品 → 选择频道：{男频/女频}
3. 一级分类：{分类}
4. 二级分类：{分类}
5. 添加标签：{标签列表}
6. 付费设置：免费章节 第1-{N}章，第{N+1}章起付费
```
