# 记忆压缩（通用）

适用于 /novel、/short 的定期记忆压缩（每 10 章）和全书完成时的强制记忆重建。

## 触发时机
- 写作阶段每 10 章自动调用
- 全书完成时强制执行
- 用户手动调用

## 流程

```
1. 读取 recent-context.md 中最早的 10 章内容
2. 压缩为 ≤500 字摘要 → 写入 context/archives/stage-{NNN}-{NNN}.md
3. 从 recent-context.md 中移除已归档内容
4. 重建 memory-outline.md：
   读取所有 archives/ + 当前 recent-context.md → 提炼为 ≤2000 字全书大纲
5. 汇报压缩结果
```

## 压缩优先级（从高到低）
1. 影响后续剧情的关键事件（不能丢）
2. 角色状态/关系变化（不能丢）
3. 伏笔埋设和回收（不能丢）
4. 新确立的世界规则（不能丢）
5. 具体对话和场景细节（可丢，只保留结果）

## memory-outline.md 必须包含
- 故事进展摘要（第1章到当前的整体走向）
- 关键转折点列表
- 当前局势描述
- 核心角色状态表
- 活跃伏笔列表（**从 foreshadowing.md 同步**，foreshadowing.md 是权威源）
- 已建立的世界规则
