# 打脸桥段模板

## 结构公式
1. **蔑视** — 对方看不起主角（嘲笑、羞辱、下结论）
2. **加码** — 对方加大筹码（当众侮辱、拉帮结伙、威胁）
3. **反转** — 主角展示真实实力/身份
4. **震惊** — 围观者集体惊呆、对方面色大变
5. **碾压** — 主角轻松解决，越轻松越爽
6. **余波** — 消息扩散、更大势力注意到

## 升级变体
- **连环打脸**: 打完小的来大的，一浪接一浪
- **代理打脸**: 不是主角亲自动手，手下就够了
- **身份打脸**: 不用动手，亮身份直接吓瘫
- **预言打脸**: 之前有人说主角不行，结果事实证明

## 关键技巧
- 蔑视阶段要足够过分，读者才爽
- 反转要干脆，不要拖泥带水
- 配角的反应是爽感倍增器
