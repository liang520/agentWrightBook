# 穿越重生机制

## 与 soul-transmigration.md 的区别
soul-transmigration 侧重灵魂转换的"机制"（怎么穿的）。本模板侧重穿越后的"生存策略"（穿过去后怎么活）。

## 穿越类型
- **魂穿**：现代灵魂进入古代/异世界某人身体，继承原主记忆
- **穿书**：穿进自己看过的小说，知道剧情走向
- **重生**：死后回到过去某个时间点，带着前世记忆
- **时空穿梭**：物理穿越，带着现代物品/知识

## 开局模式（常见起点）
- **冷宫废后**：被废弃的皇后/妃子，从最低谷翻身
- **流放路上**：全家被流放，在困境中求生
- **替身/假千金**：发现自己是冒牌货，面临被揭穿
- **退婚/和离**：被退婚或主动和离，独立求生
- **濒死重生**：前世死亡瞬间重回过去，带着仇恨/遗憾

## 金手指类型
- **前世记忆**：知道未来走向/关键事件/人物命运
- **原著剧情**：穿书者知道小说结局和关键转折
- **现代知识**：医术/科学/商业知识在古代/异世降维打击
- **随身空间**：详见 golden-finger.md
- **系统面板**：详见 golden-finger.md

## 穿越后的核心矛盾
- 原主遗留的人际关系（仇人/盟友/婚约）
- 隐藏现代人身份 vs 行为举止露出破绽
- 利用先知优势 vs 蝴蝶效应改变原有剧情
- 融入新世界规则 vs 用现代思维改变规则

## 常见叙事节奏
1. 醒来/发现穿越 → 梳理原主记忆和当前处境
2. 低调生存 → 试探周围人的态度
3. 小展身手 → 用金手指解决第一个危机
4. 引起注意 → 被人怀疑/被大佬注意
5. 站稳脚跟 → 建立自己的势力/资源
6. 逆转命运 → 改变原主/原著中的悲惨结局
