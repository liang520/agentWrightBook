# 契约婚姻/退婚桥段模板

## 退婚开场公式
1. **当众羞辱** — 对方家族当众退婚/悔婚
2. **落魄主角** — 主角此时处于人生低谷
3. **立誓** — 主角发誓要让对方后悔
4. **崛起** — 主角获得机缘开始变强
5. **再遇** — 变强后再遇前未婚妻/夫家
6. **后悔** — 对方肠子都悔青了

## 契约婚姻公式
1. **被迫结合** — 因为某种原因必须假结婚/契约婚姻
2. **互看不顺眼** — 初期互相嫌弃
3. **日久生情** — 通过日常相处产生感情
4. **身份曝光危机** — 假关系差点被拆穿
5. **真心告白** — 从假变真

## 关键技巧
- 退婚场景要够打脸、够屈辱
- 后悔的转折要有层次，不是一下子
- 读者要能从第一章就预见到对方会后悔（期待感）
