# 复仇回归桥段模板

## 结构公式
1. **回忆伤痛** — 闪回/内心独白，交代被害经过
2. **低调归来** — 主角以普通身份回到伤心地
3. **旧人重逢** — 遇到当年害过自己的人（对方认不出/看不起）
4. **试探** — 小展身手，引起注意但不暴露
5. **布局** — 一步步瓦解对方的根基
6. **摊牌** — 亮明身份或最终一击
7. **报应** — 对方跪求/崩溃/被毁

## 变体
- **温水煮蛙**: 不直接报复，让对方慢慢感受到恐惧
- **以牙还牙**: 对方当年怎么害主角的，主角原样奉还
- **放过式复仇**: 已经赢了但选择放过，精神碾压

## 关键技巧
- 伤痛回忆要足够惨，读者才会代入仇恨
- 复仇过程要有策略感，不是蛮力
- 摊牌时刻要有仪式感
