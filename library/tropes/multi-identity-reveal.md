# 多重马甲揭露桥段模板

## 结构公式
1. **身份铺设** -- 主角以多个不同身份（外貌/性别/身份）活跃，每个身份有独立的人际关系网
2. **线索积累** -- 通过另一角色（通常是男主/观察者视角）逐步发现不同身份间的蛛丝马迹（招式相似、气息相同、行为模式一致）
3. **部分揭露** -- 分阶段揭露（先揭一层，震惊一次），每次揭露都是一个高潮点
4. **全面揭穿** -- 通过某个无法伪装的证据（信物认主/灵兽认主/独有技能）一次性揭开所有身份
5. **情感爆发** -- 观察者/男主的情感从震惊 -> 愤怒 -> 心痛 -> 理解 -> 接受
6. **关系重构** -- 所有人重新认识主角，人际关系洗牌

## 关键技巧
- 每层马甲揭露的间距要控制好，不能太快（失去悬念）也不能太慢（读者忘了）
- 最终全面揭穿必须有仪式感（如天殇剑飞向苏晚认主）
- 揭露后的情感戏比揭露本身更重要——"你一直在我身边却不告诉我"的心碎感
- 用男主视角写马甲揭露更有冲击力（他以为爱的人和恨的人是不同的人）

## 适用题材
修仙马甲文、穿书、重生、都市异能

## 经典范式
moonlight-veil: 苏晚（病弱女子）-> 妖璃（妖族公主）-> 苍刀（金发男子）-> 妖族女王
揭露顺序：苍刀先被认出 -> 恩人身份确认 -> 冰翎认主暗示 -> 天殇剑飞向认主全面揭穿
