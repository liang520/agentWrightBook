# 直播掩护桥段模板

## 结构公式
1. **异常事件发生** -- 超自然/异能事件在公共场合发生，面临暴露风险
2. **主角开播** -- 以"拍电影/特效展示/科幻片"为幌子开启直播
3. **真打假戏** -- 在直播中进行真实战斗，观众以为是特效
4. **流量暴涨** -- 直播内容过于逼真引发全网讨论（真假之争反而带更多流量）
5. **官方配合** -- 权力机构顺水推舟将事件定性为"影视活动"
6. **隐藏收益** -- 除掩护外还获得额外好处（人气/认可/资源）

## 核心逻辑
现代社会中，"真实"反而不被相信。越是真实的超自然画面，越会被当成高级CG。利用这一心理盲区，将真实战斗包装为娱乐内容。

## 适用条件
- 世界观为"现代+超自然"混合，普通人尚不知超自然的存在
- 主角或盟友有官方/资本背景可以配合
- 需要延缓超自然事件暴露的时间线

## 变体
- **纪录片型**: 以纪录片形式记录异能世界，观众以为是伪纪录片
- **游戏直播型**: 以VR游戏为幌子展示异世界场景
- **短视频型**: 碎片化传播，每个片段都像是特效短片

## 关键技巧
- 主角在镜头前必须保持"表演"感，偶尔打破第四面墙跟观众互动
- 考据党的拆解讨论是很好的世界观展开手段
- 掩护终有失效之日——这本身就是一个伏笔
