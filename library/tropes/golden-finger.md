# 金手指/外挂类型

## 随身空间
- **形态**：戒指/手镯/胎记/意识空间
- **功能**：存储物资、种植灵药/粮食、时间加速修炼
- **限制**：不能存活物（或有等级限制）、进出有条件（不能被人看到）
- **升级**：空间随主角实力/使用频率扩大，解锁新功能（灵泉、药田、小屋）
- **叙事作用**：解决物资匮乏、私藏宝物、关键时刻救命

## 系统面板
- **类型**：签到系统、任务系统、商城系统、抽奖系统
- **核心循环**：发布任务 → 完成 → 获得奖励 → 实力提升
- **限制**：任务有时限/惩罚、积分不够买不了好东西
- **叙事作用**：给主角明确的成长路线和阶段性目标

## 神医能力
- **来源**：前世/传承/觉醒/系统赠送
- **能力**：诊脉、针灸、配药、手术，能治普通医院治不了的病
- **进阶**：从治普通病 → 治绝症 → 治内伤/中毒 → 起死回生
- **叙事作用**：救人积累人脉、获取回报、展现实力、打脸庸医

## 玄学风水能力
- **类型**：看风水、算命、驱邪、画符、相面
- **来源**：家族传承/前世修为/天生灵眼
- **商业化**：开店收费、直播看风水、给富豪调风水
- **叙事作用**：神秘感、精准预言制造悬念、驱邪场面制造爽感

## 前世记忆
- **优势**：知道关键事件时间线、知道谁是敌人谁是贵人
- **局限**：蝴蝶效应导致事件偏移、记忆不完整/模糊
- **叙事张力**：已知命运 vs 试图改变命运、是否告诉别人真相
