# 拍卖会桥段模板

## 结构公式
1. **入场** — 描写拍卖会的规格和氛围，暗示水深
2. **热身** — 几件普通物品，主角观察/小试身手
3. **目标出现** — 主角真正想要的东西登场
4. **竞价** — 和对手互相加价，气氛紧张
5. **危机** — 价格超出预算/对手用手段（身份压制、威胁）
6. **破局** — 主角用意想不到的方式拿下（隐藏身份、贵人相助、以物换物）
7. **散场冲突** — 拍卖结束后对手不服，场外动手

## 变体
- **鉴宝型**: 主角识破别人看不出的宝物，低价捡漏
- **搅局型**: 主角故意抬价让对手高价买废物
- **被迫型**: 主角不想暴露但被逼出手

## 关键技巧
- 物品描写要有吸引力，让读者也想要
- 竞价节奏要快，一个来回就是一个小高潮
- 场外冲突衔接下一个剧情
