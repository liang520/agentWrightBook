# 男主角色原型库

## 隐忍复仇型
- **性格特征**: 表面低调隐忍，内心坚定，有仇必报但不冲动
- **说话方式**: 简洁，不废话，偶尔冷嘲
- **常见动机**: 家族被灭、被陷害、被退婚
- **口头禅示例**: "欠我的，总要还的"
- **适用题材**: 都市、玄幻、复仇

## 霸道强势型
- **性格特征**: 自信到近乎自负，实力碾压，不屑解释
- **说话方式**: 命令式，短句多，居高临下
- **常见动机**: 守护、征服、证明
- **口头禅示例**: "你不配"
- **适用题材**: 都市总裁、玄幻

## 扮猪吃虎型
- **性格特征**: 表面废物/普通人，实际深藏不露
- **说话方式**: 随和甚至有点怂，关键时刻画风突变
- **常见动机**: 低调生活被打破，被迫出手
- **口头禅示例**: "我真的只是个普通人啊"
- **适用题材**: 都市、校园、修仙

## 热血少年型
- **性格特征**: 正义感强，永不服输，重情重义
- **说话方式**: 直接，情绪外露，爱喊口号
- **常见动机**: 变强、保护同伴、追逐梦想
- **口头禅示例**: "这一拳，是为了我的兄弟！"
- **适用题材**: 热血、少年向、修仙

## 武德少年型
- **性格特征**: 勇猛讲原则，重义气，能动手绝不动口但也绝不滥用暴力，临危不惧有分寸感
- **说话方式**: 平时随和自嘲如单口相声，战斗时狂傲直接，关键时刻沉稳果断
- **常见动机**: 保护身边人、偿还恩情、证明自己不是孬种
- **口头禅示例**: "打架是为了息事宁人"
- **适用题材**: 热血+幽默混合、都市异能、现代奇幻
- **来源**: 《灵拳破天》陆争鸣

## 支持型暖男主
- **性格特征**: 不是最强但最仗义，怕鬼但冲第一个上，嘴硬心软，有自己的爱好和可爱之处
- **说话方式**: 综艺感吐槽式幽默，自嘲式打趣，紧张时口头禅爆发
- **常见动机**: 保护身边人、支持女主、在女强框架下找到自己的位置
- **口头禅示例**: "我靠！"
- **适用题材**: 女强男弱型甜宠、都市奇幻、灵异甜爽
- **来源**: beauty-mystic 沈昭宁（当红影帝+沈家后人，怕鬼但仗义，爱做微缩模型）

## 深情魔尊型
- **性格特征**: 外表冷酷霸道、嗜杀果断，内心极致深情，为一人压制本性百年。杀伐不留情但有底线（对恩人/爱人）
- **说话方式**: 冷冽简短，关键时刻一字千钧，用"本尊"自称增强威压感
- **常见动机**: 守护白月光、寻找恩人真相、逆天改命
- **口头禅示例**: "本尊说了算"
- **成长弧线**: 失控堕魔 -> 追寻 -> 守护 -> 心碎 -> 并肩
- **适用题材**: 玄幻、修仙、马甲文（男主视角）
- **来源**: moonlight-veil 凌渊

## 腹黑算计型
- **性格特征**: 表面温文尔雅，内心步步为营，喜欢布局
- **说话方式**: 礼貌得体，话里有话
- **常见动机**: 夺权、复仇、掌控一切
- **口头禅示例**: "一切都在意料之中"
- **适用题材**: 权谋、宫斗、都市商战
