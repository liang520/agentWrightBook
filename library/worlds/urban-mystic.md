# 都市玄学体系模板

## 核心设定
现代都市中存在灵修/驱邪/风水等超自然力量，普通人不知情或半信半疑。主角以玄学能力在都市中行走，与娱乐圈/商界/日常生活交织。

## 力量体系
1. 灵力 -- 基础能量，用于驱邪/布阵/治疗
2. 灵眼/天眼 -- 看见灵体/邪物的能力
3. 法器 -- 剑/符/阵等工具
4. 灵宠 -- 修行妖物，辅助战斗/侦查
5. 入定/打坐 -- 恢复灵力的方式

## 敌人体系
- 游魂野鬼（低级）
- 百年以上厉鬼/妖物（中级）
- 噬魂魔/至邪之物（高级）
- 养小鬼/阴灵偶等人为邪术

## 势力结构
- 灵修者（散修/宗门残余）
- 风水师/道士（民间半吊子）
- 不知情的普通人（绝大多数）
- 利用邪物的商界/权贵

## 资源体系
- 灵石/灵器 -- 蕴含灵力的宝物
- 药浴/灵药 -- 恢复灵力
- 符咒 -- 守灵符/驱邪符/入梦符
- 结界/灵阵 -- 防御/封印

## 常见场景
- 娱乐圈片场/综艺录制中突遇邪物
- 富豪别墅暗藏邪祟
- 古宅/民宿/温泉的闹鬼事件
- 拍卖会上拍到灵器/灵石
- 驱邪收费创业

## 核心冲突来源
- 灵力恢复缓慢vs频繁遭遇威胁
- 隐藏身份vs被迫展示能力
- 人界规则（法律/舆论）vs超自然行事
- 与邪物交易的反噬

## 代表案例
- beauty-mystic《绝色替身靠灵修日入千万》：灵修宗主穿越娱乐圈+都市驱邪+甜宠
