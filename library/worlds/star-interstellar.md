# 星际种田体系模板

## 社会阶层
1. 贫民区居民（无光脑、无公民权利保障）
2. 平民（普通公民，基本权利）
3. 贵族（世袭或封赏，享有特权和优待）
4. 军部高官（军衔带来的权力和地位）
5. 王室/皇室（最高统治者）

## 性别体系（ABO变体）
- **雄性**：传统意义上的"男性"角色，社会地位略高
- **雌性**：可怀孕的性别，包括男性雌（外表为男，可受孕）
- 赐婚/婚姻需要国王祝福（贵族阶层）
- 军婚不可悔婚

## 势力结构
- **军部**：分多个军区（三军部等），有中将/少将等军衔
- **研究院**：科研机构，含制药部、灵植部等
- **药厂**：商业实体（松氏药厂等），与研究院竞争人才
- **贵族家族**：世家大族，掌控经济资源
- **王室**：国王统治，举办庆典，赐福赐婚

## 资源/技能体系
- **灵植**：含有灵气和毒素的药用植物（龙舌草、白芨、香附子等）
- **药剂等级**：1级→高级，由检测仪自动评定
- **制药方式**：精神力解析+煎煮/合成，药方为核心竞争力
- **光脑**：手腕式智能终端，通讯/搜索/购物
- **星网**：互联网等价物，可开店铺/直播
- **空间钮**：随身存储空间
- **治疗仪/治疗舱**：医疗设备

## 常见场景
- 国王庆典/宴会、军部基地/医疗室、研究院实验室、星网店铺、贫民区酒馆、边境战场

## 核心冲突来源
- 异兽入侵（边境战争）、药品匮乏、药方争夺、贵族vs平民阶级矛盾、商业竞争（药厂vs独立制药师）
