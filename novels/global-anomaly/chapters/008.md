# 第8章 好狗胆，我的妹妹也敢动

眨眼间，陆沉来到陆远父母的住处——临海市一片老旧小区。
为防惊吓到人，他带着陆晓陆阳降落在隐蔽处，然后跨步走进小区。
一打量，不由皱眉。
这是一片没有电梯的老楼房，环境极差，卫生糟糕，苍蝇乱飞。
难怪方媛看不上陆远，住在这种地方跟贫民窟没什么区别，生来就低人一等。
陆晓小手一指："爸爸，你看，爷爷在那边下棋！"
陆沉看过去，不禁苦笑。陆远的父亲心态倒是真好——在这种环境里，还笑呵呵地和几个老人杀着象棋。
几人围坐石桌旁，气氛激烈。
"老陆，你怎么总爱悔棋？我这步卧槽马加马后炮，你本来死定了。你这一悔棋，我又得重新布阵。"
对弈的白胡子老头不停摇头，一脸恼怒。
陆父呵呵一笑："又不是正式比赛，悔棋很正常嘛，你刚才不也悔了。"
"哼！将军！"白胡子老头撅着嘴，重重提车将军。
陆父不慌不忙，上了一个连环马挡住。
"爷爷，你看谁来了！"
陆晓蹦蹦跳跳跑过去，一下翻上爷爷的肩膀。
陆父高兴得皱纹都堆满了："哈哈，小晓、小阳你们来了。你奶奶在烙葱油饼呢，一会儿回家吃。"
陆晓抱着他的头往后一转："爷爷，你往后看。"
陆父转头，看到陆沉时愣住了。手里的棋子啪地掉在棋盘上。
"远儿，你回来了？你这几个月都跑哪去了？"
陆沉看着老人佝偻的背影和满头白发，心里也是一阵酸楚。
片刻间，竟自然地适应了儿子这个角色。
"爸，说来话长，回家再细说吧。"
陆父点点头，歉意地对棋友们说："大伙儿对不起，我得回家了，这局算我输。"
也有人窃窃私语："听说老陆这个儿子嗜赌成性，半年前失踪，是跑出去躲债了吧。"
"嘘，小声点，人家听见不好。"
陆父耳朵不聋，自然听见了，嘴角微微一抽。
看远儿这穿着，估计也没挣到什么钱。算了，人活着就行，其他都不重要。
陆沉当然也听得一清二楚。在异能纪元，这种嚼舌根的人不打残也得让他长个记性。但在地球还是算了。
他暗中催动了一个小异能——操控四周飞鸟，召到几个老头头顶。
"给他们洗个脸，清醒清醒。"
鸟雀们乖乖听令，纷纷对准目标排放。
"哎呦！哪来的这么多鸟！"
"太缺德了，我眼睛都睁不开了！"
那个骂得最欢的大爷，一只乌鸦直接飞到他嘴边精准投放。大爷神情呆滞，不知该张嘴还是闭嘴。
陆父带着陆沉来到一单元6楼，掏钥匙开门。
一个将近六旬的老太太正在打扫卫生，看到陆沉和孩子们，顿时语无伦次。
"天哪，远儿！你回来了！"
她扔掉毛巾，冲上来捶陆沉的胸口，笑中带泪。
这半年儿子失踪，把她又急又愁，人都老了不少。
陆沉拍了拍她的肩："妈，别哭了，我不是回来了嘛。"
他整合了陆远的部分记忆——陆远和母亲最亲。还有一个读高三的妹妹陆瑶，和他感情也很好。陆父不善言辞，但对儿子也不差。
总之，这是一个温暖的家庭。没想到陆远被方媛肢解后，这个家就失去了核心。
陆沉暗叹一口气，招呼陆晓陆阳过来见奶奶。
"你们先坐着看会电视聊聊天，我去做午饭，再烙些孩子爱吃的葱油饼。"
陆母擦干眼泪，乐呵呵地去了厨房。
陆沉看看墙上的钟——才上午十点多。陆母太高兴了，忘了时间就开始做饭。
他也不阻止。他是孤儿出身，从来没感受过亲情的温暖。如今体会到了，竟让他坚硬如铁的心有了些许柔软。
觉醒者之路最忌亲情牵绊。但人非草木，岂能无情？
爷俩简单聊着。陆沉编了个合理的说辞——说自己去国外跟人合伙搞投资，几个月赚了些钱，在临海市锦澜府给老人买了别墅。
老陆头被吓得够呛——投资这么好赚钱？几个月就能买别墅？
"远儿，你那还差人不？我看我行不行？"他活动了一下筋骨，表示自己还不到六十，能干活。
陆沉憋住笑：我这都是编的，上哪给您找投资去？
"等有空再带您去开开眼界，您去了至少也是个顾问级别的。"
老陆头更高兴了，翻出一瓶珍藏的好酒。
吃过午饭、享受过一番亲情之后，陆沉动员老两口搬家。只带必要物品，别墅那边一应俱全。
老两口虽然期待住大别墅，但对老房子还是很留恋，商量着能不能再住段时间。
陆沉想了想，都依了他们。
正在这时，陆父的手机响了。是女儿陆瑶打来的。
"爸爸，我踩脏了一个人的鞋子，对方要我赔一万块！不然就把我带走……呜呜呜……"
陆瑶在电话那头失声痛哭。
什么？陆父惊怒交加——光天化日竟有人敢敲诈欺负他闺女？
"瑶儿别怕，爸和你哥马上来！那帮王八蛋敢乱来，老子和他们拼了！"
"什么？哥哥回来了？"陆瑶又惊又喜。哥哥一向疼她，而且打架也厉害，要是来了，这帮人肯定要遭殃。
电话那头传来几个男人调戏的声音——
"小妞，快催家里送钱来，不然哥几个可要动手了。"
"还别说，这妞长得真正点，不知床上功夫怎么样。"
陆沉杀意骤起。
好狗胆。连我SSS级大能的妹妹都敢动？嫌命长了吧。
"爸，把地址告诉我，我马上去。你和妈带着孩子待在家里，我一个人搞得定。"
陆父半信半疑，把学校地址说了。
陆沉抓了个葱油饼边吃边出门。
关上门，意念一动，一道快如闪电的虚影原地消失。
陆父不放心，打开门要跟着出去——外面空无一人。
远儿呢？跑这么快？
这孩子快三十的人了，还是这么心急火燎的。
