## 角色性格档案

### 姜北辰（主角）
- **前世**：灵星天剑宗元婴天才弟子，被七大仇敌宗门联手偷袭身死
- **核心驱动**：报仇雪恨 + 保护身边人 + 重返巅峰
- **性格深层**：表面霸道不可一世，实则极重感情；对敌人绝不手软，对亲近之人温柔护短
- **成长弧线**：从孤狼式复仇者 → 有家有伙伴的领袖
- **战斗风格**：碾压式打击，以绝对实力说话，不喜欢拖泥带水

### 沈星澜（女主）
- **身份**：灵星修仙者，流落蓝星后成为天武联盟总盟主
- **核心驱动**：回灵星报仇（九幽殿毁她师门） + 守护同伴
- **性格深层**：外表清冷不近人情，内心重情重义；对强者敬佩，对弱者怜悯
- **感情发展**：从互相试探 → 同道互认 → 并肩作战中产生感情
- **战斗风格**：冰系能力为主，凌厉精准，和姜北辰形成冰火互补

### 烈羽（灵宠）
- **身份**：大乘境九头火凤，被封印在地下火海中
- **核心驱动**：恢复自由 + 跟着强者吃香喝辣
- **性格深层**：嘴上嫌弃实际忠诚，自尊心极强但能屈能伸
- **功能**：战力补充（大乘境）+ 信息源 + 气氛调节担当
- **特殊设定**：可化为人形（红发少年），贪吃好斗

### 魏玄冥（终极BOSS）
- **身份**：黑玄秘境之主，灵星第一修士，准仙人境
- **核心驱动**：独占灵星资源飞升仙界
- **性格深层**：将所有人视为棋子，包括自己的手下
- **威胁层级**：全书最高，直到终战才正面对决
