# 全书记忆大纲 — 寒渊觉醒（全37章，已完结）

## 故事进展摘要
姜北辰（玄元界青霄剑宗元婴天才）被七宗联盟偷袭身死，神魂穿越至蓝星云溪镇冰柜中，借夺舍秘法重组姜远志肉身重生。获九霄塔升至化神初期，收服大乘境灵宠烈羽，灭血魔宗/霍家，教全家修炼。发现暗渊碎片（域外邪物血祭阴谋），闭关半年闯九霄塔九层获九天玄功，化神巅峰。拒绝仙门飞升（获30日令牌），灭天武联盟韩子墨，收服天武联盟。追踪暗渊至护国寺击碎碎片灭神念，烈羽清12个血祭点。与沈星澜结盟，经东海虚空裂隙回玄元界，五行仙元觉醒。灭幽冥谷先遣队，赶赴凌霄阁救白凤鸣。沈星澜杀殷无为复仇，横扫东域建玄元同盟。回青霄剑宗认师顾长渊，灭七宗联盟报仇。重塑本体肉身，归还姜远志肉身。攻入幽冥禁地，金瞳投诚带路，一剑碎灭准仙人境魏玄冥。四人飞升仙界，百年后姜北辰成仙帝。
