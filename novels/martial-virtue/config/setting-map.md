# 设定映射表

## 世界观

| 原作设定 | 新设定 | 说明 |
|---------|--------|------|
| 帝制现代国家 | 宁朝联邦（现代帝制国家） | 保留帝制+现代科技混合设定 |
| 平行异世界（三年后的毁灭未来） | 裂隙之境（三年后的崩坏未来） | 每晚午夜传送 |
| 死亡国度 | 冥渊界 | 亡灵和死亡的终极领域 |
| 死亡魔窟 | 深渊魔窟 | 必死者被传送到的封印空间 |
| 罪狱 | 劫牢 | 囚禁觉醒者的超维空间 |
| 镜像世界 | 映界 | 用于训练的虚拟战斗空间 |
