### 主线弧线

1. **【起】原作第1-5章 → 新作第1-3章**：强者苏醒
   - MC 万年沉睡后苏醒在现代都市，发现异能界秩序崩坏
   - 发现规则被暗中篡改，旧盟友留下求救信号
   - 分化多重身份渗透各方势力

2. **【承】原作第6-18章 → 新作第4-11章**：潜伏调查
   - 以不同身份混入异能组织、地下势力、官方机构
   - 揭露"寄生体"式的暗中操控手段
   - 结识搭档（尘钰→顾行舟），建立信任
   - 发现旧友被分魂封印的阴谋

3. **【转】原作第19-33章 → 新作第12-21章**：夺权清理
   - 惩治堕落异能者（长生城事件→长安小区权贵事件）
   - 推翻腐败领袖，以伪装身份上位
   - 正面对决暗影BOSS（逃入时间缝隙→潜逃）
   - 盟友道心转变

4. **【承二】原作第34-42章 → 新作第22-26章**：人间百态
   - 海神事件→海尊异变事件
   - 花蕖线→华秋边缘人物救赎线（精简）
   - BOSS反击，据点被毁

5. **【合】原作第43-54章 → 新作第27-32章**：终局
   - 设局引蛇出洞
   - BOSS真身揭露
   - 上古真相（善恶起源）
   - 终局对决 + 大梦反转
