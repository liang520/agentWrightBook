# 全书记忆大纲

## 世界规则
- 苍穹界修仙世界，天道有自主意志，暗中操控所有人命运
- 渡劫飞升是谎言——天道用天雷抹杀渡劫者，吸收其力量
- 造化天书是世界根源法器，可改写天道规则
- 天道通过"天选之女"姜映雪积蓄力量，姜映雪死后目标转向苏晚
- 灵兽契约认灵魂不认肉身，无法伪造

## 主线弧线
凌渊视角：失控堕魔（ch1-5）-> 追寻恩人（ch6-11）-> 守护妖璃（ch12-20）-> 入魔屠戮（ch21-22）-> 被苏晚唤回（ch23）-> 真相大白（ch27-28）-> 心碎与三月之约（ch29-30）-> 逆天道（ch31-33）-> 并肩（ch34）

## 角色终态
- 凌渊：魔尊，与苏晚大婚，守护两界
- 苏晚（妖璃/苍刀/慈济天尊）：造化天书认主，成为两界维系者
- 沈青鸿：从光门获救，情敌→盟友，继续经营沧澜城
- 姜映雪/姜瑶：ch22被凌渊亲手杀死，天道消散后气运归零
- 慕容净：修为尽废，无名小庙敲木鱼
- 冰翎：苏晚的灵兽，ch18复活认主

## 核心情感线
凌渊对苏晚：白月光执念 -> 追查恩人 -> 暗中守护 -> 入魔失控 -> 真相震撼 -> 心碎但不放手 -> 逆天改命 -> 并肩

## 关键转折点
1. ch1：苏晚祭阵而死，凌渊堕魔
2. ch11：凌渊确认妖璃=恩人
3. ch21-22：姜映雪掘墓→凌渊入魔屠戮
4. ch27：天殇剑认主，所有马甲揭开
5. ch29：苏晚坦白"我利用了你"
6. ch33：天道对决，造化天书认主
7. ch34：天道消散，大婚
