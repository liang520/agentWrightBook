# 情节骨架

## 原作概况
- 原作：白月光的千层马甲
- 总章数：28章（含前言+番外）
- 总字数：约211369字
- 目标字数：45000-50000字（硬上限50000，统计口径见CLAUDE.md"字数统计规范"）
- **视角改编**：从女主叶玫视角 → 男主魔尊裴羡视角
- **风格改编**：言情甜宠 → 热血燃向男频

## 主线弧线（魔尊视角重构）

1. 【起·白月光陨落】新1-5章
   - 裴羡在神宗压制魔族血脉，守护心上人叶玫。叶玫以身祭阵，死在他怀里
   - 封印崩碎、堕魔，灭妖族泄愤。在妖城遭遇神秘金发男子"狂刀"——实力深不可测，言辞刺激，将他引离妖城
   - 被天雷削弱，坠落荒野，被一个"不知天高地厚"的女子救了半壶水
   - 暗自追踪此女，发现她行事诡异、战力超群，却与他恩人气息相似

2. 【承·暗中守护】新6-14章
   - 华城考核赛：裴羡以斗笠男子暗中观察"妖九"。她以一阶药师身份carry全场——他既欣赏她的顽强，又被她的战斗方式震撼
   - 荒山决战：柳萌萌（女主分身）追杀妖九。裴羡出手灭佛子神识、钉死柳萌萌。各路大能围攻，他一人屠尽
   - 神宗生活：裴羡化名入道宗，暗中守护。年亚澜现身引发争锋。裴羡认出岳丈叶剑，公开护持第五峰
   - 心中矛盾：他对叶玫的执念，与对妖九日益增长的关注——她的骨气、她的狡黠、她身上那股熟悉的气息

3. 【转·真相与背叛】新15-25章
   - 妖九身上的秘密越来越多：她筑基悟出"生道"、搜刮君家宝库、与年亚澜的暧昧
   - 叶玫坟被柳梦梦掘开——裴羡彻底入魔，亲手杀女主、杀佛子、屠正道联军
   - "叶玫"现身制止他——她用叶玫的声音、叶玫的银镯、叶玫的无锋剑。他的魔化被她一句话瓦解
   - 裴羡得知妖九=叶玫=狂刀——所有的恩、仇、爱、恨，都是一个人。震惊、狂喜、痛苦交织
   - 叶玫坦白一切皆有算计——裴羡心碎但仍不离不弃

4. 【合·逆天改命】新26-34章
   - 妖族复兴大战：裴羡带魔族助力，与叶玫联手打造妖城
   - 须弥山秘境：叶玫独入险境，裴羡暗中跟随
   - 天道真相：造化玉牒=世界剧本，天道企图吞噬叶玫
   - 最终决战：裴羡在叶玫意识将散时赶到，为她输入力量，助她滴血认主造化玉牒
   - 天道意志消散，叶玫开辟上层位面，裴羡与她并肩而立
