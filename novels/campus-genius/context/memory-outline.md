# 全书记忆大纲

## 故事完整摘要
沈知意（华清化学系学霸）死于实验室爆炸后灵魂附身到锦城六中差生顾念念身上。以学霸实力逆袭，从全校倒数到五市联考第一，渣爹饭桌被打脸。经历月考作弊冤案（Ch8→Ch16平反）、保温杯砸头魂体分离（Ch17）、原主灵魂回归夺回身体（Ch19-20）等危机。钱若薇因嫉妒犯下图钉/保温杯/推人三宗罪被揭露（Ch24）。通过奥数集训营与裴砚舟（前世同学）重逢，追查前世死因：何暮晚挑唆秦瑞阳以为沈知意偷听其贪污秘密，秦瑞阳制炸药同归于尽（Ch27-29）。高考后进入华清，直面何暮晚获得坦白录音（Ch37），裴砚舟以一年药物使其精神崩溃（Ch39）。韩教授将沈知意前世科研项目交还（Ch40）。墓地夺回骨灰盒告别前世，回母校演讲，与陆时屿牵手，接受新身份好好活着（Ch41）。
