# 全书记忆大纲

## 核心设定
灵修宗宗主苏清微千年前以身铸造镇魔剑，残魂穿越为娱乐圈黑料小花苏瑶。与原身达成魂约：替她完成执念（赚十亿）。镇魔剑上四颗灵石（蓝/赤/金/翠）存有残魂碎片，须全部归位才能灵肉合一。

## 核心人物
- **苏瑶**（女主）：千年宗主残魂寄凡胎，淡然威严+不谙现代，口头禅"我从不食言"
- **沈昭宁**（男主）：当红影帝+沈家后人，怕鬼但仗义，爱做微缩模型，口头禅"我靠"
- **小翠/阿翠**：百年翠青蛇妖，灵宠，胆小嘴硬贪吃，叫苏瑶"姐姐"
- **方远**：沈昭宁经纪人/发小，搞笑担当
- **陆青衡**：苏瑶经纪人，半敌半友
- **乔安**：苏瑶助理，暖心

## 主线弧线（34章）
1. **相遇+对抗**（ch1-6）：苏瑶拔剑取石、被迫恋综、赵宏达台球赌约
2. **恋综录制**（ch7-20）：竹溪温泉山居三天——救小旭、抓蛇名场面、赤灵石归位、周大爷支线、行李箱偷渡蛇
3. **回城+玉兰庄**（ch21-27）：收小翠、鸿门宴斩噬魂魔、赵宏达自首、金灵石回收、发现十亿执念
4. **创业+收官**（ch28-34）：解约创业玄学大师、噬魂魔报复剑魂降世、四石归位、跨年斩魔魂约解除、表白大结局

## 感情线
互相嫌弃 -> 被迫合作好奇 -> 恋综心动 -> 玉兰庄生死相依 -> 暗恋升温 -> 跨年表白牵手

## 世界规则
- 灵力可驱邪/开灵眼/凝神入定/御剑飞行
- 魂约束缚苏瑶灵魂在凡胎中，完成执念方可解除
- 镇魔剑只有沈家后人+灵修血脉可拔出
- 灵石归位可增强灵力，四石齐则人剑合一
- 噬魂魔以人血祭换取窥探欲望的能力
