# 角色映射表

## 映射原则
- 所有原作角色名在成稿中**绝对不能出现**
- 新名字符合都市异能/无限流风格，音韵区别于原作
- 性格保留原作核心特质，说话方式和口头禅做差异化处理

---

## 核心角色

### 1. 清沂 → 陆知渊（主角/受）
- **性格**：表面咸鱼摆烂、实则心思极深，擅长伪装弱小。骨子里倔强，不愿被任何人安排命运。创伤后应激导致失忆，对过去有直觉性的恐惧和回避。
- **说话方式**：懒洋洋的语气，喜欢用反问句堵人。紧张时反而更平静，像是情绪断联。
- **口头禅**："……随便吧。" / "你们忙，我先躺了。"
- **特征**：左耳太阳图腾耳夹（映射为：左耳银色日轮耳钉）。看似F级废物，实为隐藏的最强者。

### 2. 妄悠 → 沈夜霖（主CP/攻）
- **性格**：冷淡疏离，对大多数人漠不关心，唯独对陆知渊有深入骨髓的执念与温柔。表面是系统高层/执行者，实际为了保护陆知渊自毁了一切。怕黑。
- **说话方式**：简短克制，不说多余的话。但在陆知渊面前偶尔流露出孩子气的一面。
- **口头禅**："别动。" / "……闭嘴，让我待一会儿。"
- **特征**：修长苍白，气质阴冷。降级后与陆知渊共享生命值。

### 3. 江黎歌 → 程暮声（队友/系统员工）
- **性格**：表面温和有礼、好脾气，实际是系统派来的观察者，有自己的立场和秘密。对陆知渊有复杂的任务关系（保护+监视）。后期逐渐被陆知渊打动，从旁观者变为真正的同伴。
- **说话方式**：礼貌得体，偶尔用敬语制造距离感。紧急时刻会露出利落干脆的一面。
- **口头禅**："这个嘛……我只是建议。" / "请不要误会，我没有恶意。"

### 4. 塞西瑞浮 → 裴洛珈（副本1核心boss）
- **性格**：双生灵魂合一的咒术师/系统NPC。亦正亦邪，行事极端但有自己的逻辑。弟弟杀死哥哥后灵魂融合，精神不稳定。
- **说话方式**：时而温柔时而暴戾，两种人格交替出现。喜欢用宗教式的措辞。
- **口头禅**："来，告诉我你的罪。" / "你看，这不就对了吗？"

### 5. 羽季凉生 → 霍辞舟（NPC盟友/5年前队友）
- **性格**：沉默寡言、重义气，5年前与陆知渊并肩作战的战友。为破阵甘愿自取心脏，有舍身成仁的觉悟。
- **说话方式**：极少开口，一旦说话必有分量。语气平淡如陈述事实。
- **口头禅**："……你不记得了。" / "走吧。"

### 6. 祝依然 → 纪星萤（小女孩队友）
- **性格**：看似天真柔弱的少女，实际内心坚韧。在红缘副本中是关键角色，有超出年龄的冷静。
- **说话方式**：说话轻声细语但逻辑清晰，偶尔冒出老成的判断。
- **口头禅**："我没事的，哥哥。" / "……我觉得不对劲。"

### 7. 俞筱 → 温绥宁（队友）
- **性格**：理性冷静的异能者，有独特的辅助型能力。戏份精简，主要在红缘副本中发挥作用。
- **说话方式**：条理分明，像在做分析报告。
- **口头禅**："从概率上讲……" / "等等，让我算一下。"

### 8. 塞西瑞仪 → 裴洛珈（合并）
- **处理方式**：不单独出场。作为裴洛珈"双子哥哥"的身份，仅通过回忆闪回和裴洛珈的精神分裂表现带出。
- **在文中的存在形式**：裴洛珈偶尔切换的另一种人格语气即为"哥哥"的残留。

### 9. 塞西芙丝 → 殷夕照（修女/NPC线索角色）
- **性格**：副本1中的NPC，以童谣和支线任务引导玩家。形象为苍白的修女，语气空灵。
- **说话方式**：用歌谣般的韵律说话，断断续续。
- **口头禅**："……安息吧……尘归尘……"

### 10. 塞西芙霓 → 殷朝暮（圣女/回忆篇反派）
- **性格**：5年前回忆篇中的关键反派。狂信者，为"教义"不择手段。是围攻往生风铃战队的主要推手。
- **说话方式**：狂热而笃定，有布道者的节奏感。
- **口头禅**："这是神的旨意。" / "你们都会被净化的。"

## 次要角色

### 11. 鸢鸢 + 生生 → 苏衍（合并为一个综合队友）
- **性格**：开朗话多的队友，兼具原作两人的功能。在5年前回忆篇中出场，是往生风铃战队的成员。
- **说话方式**：语速快，喜欢插话。
- **口头禅**："等等等等，你说什么？！"

### 12. 顾余 → 楚言（5年前队友/牺牲者）
- **性格**：温厚可靠的前辈，战队的定心骨。挡剑牺牲是陆知渊创伤的核心。
- **说话方式**：语气温和，像兄长。临终遗言关于戒烟约定。
- **口头禅**："别抽了，说好的。"

### 13. 黄宇豪 → 贺庭肃（红缘副本反派）
- **性格**：红缘副本中的反派NPC/玩家，阴险狡诈。
- **说话方式**：虚伪客气，笑里藏刀。

### 14. 三号复制体 → 三号切片
- **性格**：陆知渊的"咸鱼"人格切片，最具代表性。自愿赴死完成"埋葬自我"任务。
- **说话方式**：比本体还懒，说话像在梦游。

### 15. 五号复制体 → 五号切片
- **性格**：陆知渊的"自毁"人格切片，代表自杀倾向。是副本最危险的心理关卡。
- **说话方式**：低沉平静，逻辑清晰但结论极端。

### 16. 小妄悠 → 小时候的沈夜霖
- **处理方式**：红缘副本中出现的童年版沈夜霖，暗示陆知渊失忆真相。
- **关键台词**："……懦夫。"
