# 情节骨架：短篇改编版

## 一、主线弧线（4个副本+结局）

### 副本1：教堂圣歌（核心副本，保留最完整）
- **开局**：清沂进入废物回收系统，排名倒一，抽到F级废柴道士（减速debuff）
- **建立关系**：遇江黎歌（系统派来的保护者），伪装小白兔
- **解谜主线**：塞西芙丝支线任务三段（墓碑→尸骨→埋葬自我）
- **boss战**：塞西瑞浮（双生子/咒术师/系统NPC）
- **复制体实验**：穿越多条世界线，亲手"埋葬"自己的人格切片
- **虫人街道**：战斗+羽季凉生线+心脏收集+破阵
- **回忆篇**：5年前往生风铃战队、顾余牺牲、反击教廷、异能觉醒
- **通关**：拒绝自杀诱导，活着躺进棺材通关

### 副本2：极乐园+系统总部
- **过渡**：进入安全区，扭蛋抽到死神级道具（系统总部参观券）
- **系统总部S级任务**：妄悠正式登场，指头游戏互爆身世
- **绑定**：妄悠降为玩家加入队伍，生命值共享
- **炸楼通关**：精神疲劳值逼近极限，用奇行种爪按下核心按钮炸楼

### 副本3：红缘（妄悠故乡）
- **设定**：古代战乱世界，分为拐卖/演戏/收集镜片三条任务线
- **小妄悠**：本世界的童年妄悠，暗示清沂"懦夫"和失忆真相
- **镜子**：三组镜片拼合→清沂被吸入记忆世界

### 最终回忆+结局
- **记忆恢复**：清沂回忆起全部往事——往生风铃、献祭阴谋、妄悠自毁
- **最终战**：与妄悠潜入湖底炸毁邪神心脏
- **结局**：清沂回归现实世界，妄悠通过电脑发来消息"我回来了"

## 二、保留支线

### 1. 感情线（清沂×妄悠）
- 妄悠暗中保护→正式登场→互相试探→游戏绑定→记忆回收→HE
- **压缩策略**：删除重复的暧昧暗示，保留关键节点（游戏、绑定、怕黑、最终吻）

### 2. 身世线（清沂的真实身份）
- 半神半人实验体→全服第一→献祭阴谋→失忆→觉醒
- **压缩策略**：将分散的信息集中在2-3个关键揭示点

### 3. 复制体线（埋葬自我）
- 5个复制体代表清沂不同性格面→逐一"杀死"→心理成长
- **压缩策略**：保留3号（咸鱼）和5号（自杀）两个最有代表性的，其余用叙述带过

## 三、删除/弱化支线

### 1. 钟于对手线 → 删除
- 原作中钟于多次挑衅、污蔑清沂，但对主线推进贡献极低
- **处理**：仅在开局保留1-2句提及作为背景噪音，不展开冲突

### 2. 玉佩线（红缘副本） → 大幅压缩
- 黄宇豪用玉佩指控祝依然的情节过于拖沓
- **处理**：直接跳过玉佩对峙，以叙述交代冲突结果

### 3. 惩罚细节 → 弱化
- 回收惩罚的恐怖描写（花从肚子里长出来等）可精简
- **处理**：一句话交代惩罚机制和结果，不详写过程

### 4. 羽季凉生回忆篇 → 大幅压缩
- 原作用Ch19-25（7章）讲述5年前故事，占比过大
- **处理**：压缩为2章，保留核心事件（遇清沂、顾余牺牲、反击、觉醒），删除大量对话

### 5. 虫人街道细节 → 压缩
- 多轮虫人战斗重复性高（Ch14-18共5章）
- **处理**：压缩为2章，合并战斗场景，快速推进到破阵

### 6. 圣水试炼 → 删除
- 极乐园内的硫酸杯筛选对主线贡献低
- **处理**：直接跳过
