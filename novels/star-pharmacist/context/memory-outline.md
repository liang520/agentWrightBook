# 全书记忆大纲

## 作品信息
- 书名：星域灵药师[重生]
- 类型：BL星际种田/重生/甜宠爽文（短篇改编）
- 总章数：34章 | 总字数：约47480字
- 主线：沈逸辰重生→改变命运→研发灵药→揭露裴氏→与凌霄寒双向奔赴

## 世界规则
- 星域联邦，灵脉性别体系（灵雄/灵雌）
- 灵药体系：灵草→炼制→灵药（一阶至至阶），灵毒是核心矛盾
- 军事体系：星将/辉将，第三舰队为核心舞台
- 灵契军约不可废（军婚设定）
- 星皇府为最高权力

## 主线脉络
- 起（1-5）：重生→求娶凌霄寒→古方→遗产→灵网店铺
- 承（6-13）：研究所→药理辩论→灵药工坊→裴氏暗流
- 转（14-26）：星兽狂潮→问题灵药→解毒研究→星尘区枪战→怀孕→失子
- 合（27-34）：解毒成功→裴氏覆灭→嘉奖→婚礼→双重生揭晓

## 核心角色终态
- 沈逸辰（灵雄）：功勋药师，与凌霄寒婚礼圆满
- 凌霄寒（灵雌）：晋升辉将，双重生者，灵毒已清
- 霍远征：双腿义肢但继续指挥，特等军功章
- 苏锦屿：首席军医，全程医疗核心
- 顾明澜/叶澄：药厂合伙人，后勤保障
- 程远山/韩守正：药植特研组教授，学术支撑
- 宋暮白：完全康复，与柳寒烟结婚
- 陆泽言：被捕 | 江如玉：失势
- 裴氏家族：全面覆灭

## 情感主线
- 灵暖石贯穿全书（第2→5→9→14→15→26→33→34章）
- "这辈子不会再让你受委屈"反复呼应（第1→19→26→33章）
- 双重生揭晓为全书最大反转（第34章）
