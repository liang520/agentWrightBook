## 一、硬伤修复（4项）

### H1. details-lock"相恋七年"与正文"共修两年"矛盾

**问题**：details-lock 锁定"姜璃与黎沐辰相恋年数=七年"，但第1章写"共修灵术已近两年"。"七年"实际是裴霁寒的暗恋年数。

**修改**：
- 文件：`config/details-lock.md`
- 原文：`| 姜璃与黎沐辰相恋年数 | 七年 |`
- 改为：`| 姜璃与黎沐辰共修年数 | 近两年（青梅竹马多年，正式共修灵术近两年） |`
- 搜索关键词同步更新

### H2. 第6章记忆场景错误——"醉仙楼包间"应为"洞府"

**问题**：第6章回忆写"换来的是醉仙楼包间里那副不堪入目的画面"，但第1章明确写的是在黎沐辰的**洞府**中撞见。

**修改**：

- 原文：`上一次她全心全意地交付，换来的是醉仙楼包间里那副不堪入目的画面。`
- 改为：`上一次她全心全意地交付，换来的是那间洞府里荧光明灭的双修法阵。`

### H3. 道侣三约第一条"随母姓"被推翻缺乏过渡

**问题**：第10章明确约定"灵胎随母姓"，第26章直接命名"裴乐安"随父姓，中间无任何讨论或内心交代。

**修改**：
- 文件：`chapters/026.md`，在命名"裴乐安"的段落前插入姜璃内心过渡
- 在姜璃说出"裴乐安"之前，加2-3句：

```
她想过用"姜"——当初那三条约定里有一条写得明明白白。可现在她看着眼前这个人，看着他笨拙地雕花灯、认真地给她量灵脉温度、每天早起比她先到厨房的样子，觉得那三条约定是她穿着盔甲定下的规矩。

盔甲早就脱了。有些规矩也该跟着一起脱。
```

### H4. 道侣三约第二条"不公开"被打破无反应

**问题**：第22章裴霁寒在订婚宴当众说"我的道侣"，姜璃无任何反应，好像三约第二条从未存在。

**修改**：
- 文件：`chapters/022.md`，在裴霁寒说"我的道侣"（第27行）之后，加姜璃的反应
- 原文第31行后插入：

```
姜璃的心跳漏了一拍。道侣三约里白纸黑字写着"不公开"，他就这么当着满堂人说出来了——甚至没有提前跟她商量。

她本该生气的。但看着他平静如常的侧脸，她发现自己一点都不想否认。
```

## 二、重要问题修复（8项）

### I1. 中段（14-17章）节奏塌陷

**问题**：三路审查均指出14-17章缺乏外部冲突，"甜蜜日常"推动力不足，是最可能的弃书区间。

**修改方案**：在第15章（玄霜山）中加入一场微型危机。不需要大改，在第15章中段插入约150字的情节：

```
裴掌门在书房对裴霁寒提起，天玄宗有几位长老私下议论灵契之事，认为"散修出身的外人不宜绑定掌门嫡传的灵脉"，建议趁灵胎未成之前安排解契。裴霁寒的反应是沉默了三息后说了一句："谁提的，让他来找我。"

姜璃隔着一道屏风听到了这段对话。她没有进去，转身回了客房。那天晚上她没有去找裴霁寒同住。
```

这段插入的作用：
1. 制造新的外部威胁（宗门内部反对声）
2. 暗示姜璃的不安全感仍在（虽然感情在升温但"被排斥"的恐惧没有消失）
3. 为第17章"主动同住"制造更强的反差和动力（她克服了恐惧）

### I2. 反派傅渊过于扁平

**问题**：傅渊只有"阴险"一个维度，动机不清，阴谋太容易被破解。

**修改方案**：在第22章傅渊出场时（第47行之后），加入约100字补充动机层次：

```
姜璃注意到傅渊看向傅灵儿的眼神里有一瞬极其复杂的东西——不像兄长对妹妹的关心，更像一个被遗忘的人看着家族唯一的宠儿。她后来才知道，傅渊是庶出，傅灵儿是嫡出。紫金阁的一切本该属于傅灵儿，他所做的每一步棋都是为了证明自己配得上"大公子"三个字。
```

### I3. 第25章商会高潮过于单薄

**问题**：全书阴谋线唯一高潮，傅渊被揭穿后几乎不挣扎就败退。

**修改方案**：在第25章第43行（傅渊说"空口——"被打断）之后，扩充约200字的对抗：

```
"这些记忆石从何而来？"傅渊强撑着语调，"谁能保证不是天玄宗自己布的阵、自己拍的影像？制阵铭文可以仿造——"

"铭文能仿，手法仿不了。"姜璃从袖中取出第四样东西——一枚从阵盘上取下的灵丝节点，"紫金阁的阵法师用的是左旋走线，这是贵阁独门手法。在座若有精通阵法的前辈，可当场验证。"

她的目光扫过长桌另一侧。两位来自散修联盟的阵法长老已经站起身，走向中央的公用放映阵仔细端详。

片刻后其中一人抬头，声音不大但足够清晰："左旋走线，确实是紫金阁一脉的技法。"

傅渊的脸色一层层褪下去，最后变成一种毫无血色的灰白。他的手在桌面下攥了又松、松了又攥，指甲嵌进了掌心。
```

### I4. AI高频重复句式

**问题**："嘴角弯了"19次、"灵纹脉动/温/亮"26次、"耳根红"7次、"沉默几息"12次。

**修改方案**：全书搜索替换，每种表达保留5-6次，其余替换为多样化描写：

| 原句式 | 保留次数 | 替换方向 |
|--------|---------|---------|
| 嘴角弯了/弯了弯 | 保留6次 | 眼尾微收、唇角一动、面上不显但声音软了、鼻尖轻哼了一声 |
| 灵纹亮了/温了/脉动 | 保留8次 | 改用角色行为暗示（他的手指收紧了一下、她不自觉地摸了摸腕间） |
| 耳根/耳尖红了 | 保留3次 | 脖颈微烫、目光躲了一下、声音突然低了半调 |
| 沉默了几息/片刻 | 保留5次 | 直接接动作（他没答话，转身去倒了杯水）、停顿用标点（破折号、省略号）表达 |

具体替换将在各章中逐一执行，不做批量replace_all以防误伤。

### I5. 姜璃对旧情心理创伤处理不足

**问题**：七年感情崩塌后的内心挣扎（对亲密关系的恐惧、对判断力的怀疑）几乎没有展开。

**修改方案**：在两处加入内心独白——

1. **第5章**，在张灵医确诊后姜璃沉默时，加1句：
```
她忽然想，自己是不是天生看不准人——两年共修看不出黎沐辰是什么东西，一夕灵契就把自己绑给了另一个她根本不了解的人。
```

2. **第10章**，在姜璃提出道侣三约前，加1句：
```
这三条规矩听起来冷情得很。但她在黎沐辰身上学到的唯一一件事就是——不留后路的信任，是这世上最贵的赌注。她赌不起第二次。
```

### I6. 黎沐辰过于苍白 + 第30章收束无重量

**问题**：黎沐辰全书几乎不存在，被清理出师门只用裴霁寒一句转述带过，缺乏仪式感。

**修改方案**：在第30章（`chapters/030.md`）第27-34行之间，替换姜璃的回忆段落为更有重量的版本：

```
"黎沐辰和周蓉被清理出师门了。你爹的意思——品行有亏，不宜留在门中。"

姜璃擦裴乐安脸的手停了一息。

苏婉后来告诉她，黎沐辰离开青云宗那天，走的是后山小路。没有行囊，没有回头。周蓉倒是闹了一场，说这一切都是姜璃的报复，被两个师叔架出了山门。

她没有去看。

但那天晚上她一个人在丹房里坐了很久，盯着丹炉里安静的灵火。两年共修留下的不只是右腕上那道早已消失的灵痕，还有一种"我是不是永远看不准人"的自我怀疑。这种怀疑在她遇到裴霁寒之后并没有消失，只是被一点一点、一天一天地覆盖住了。

黎沐辰走了。那个怀疑也跟着一起走了。

不是因为报复的快感，是因为她终于确认——她看准了裴霁寒。
```

### I7. 裴掌门夫妇分居线有头无尾

**问题**：第15章引入"秘密分居一年"，后续只在第17章轻触一下就消失了。

**修改方案**：在第30章末尾（裴乐安抓手指段落后），加2句简短收束：

```
"对了，"裴霁寒的语气变得轻松了一些，"我娘搬回主院了。说是要就近看孙女。我爹……没有反对。"

姜璃笑了一声。有些路绕了一大圈，到头来还是走回原处。
```

### I8. 第22章叙述者过度解读潜台词

**问题**：第41行叙述者跳出来解释姜璃夸傅灵儿礼服的"真实含义"，削弱读者自行感受的空间。

**修改方案**：
- 文件：`chapters/022.md` 第41行
- 删除整段解释：`这是一句正常的夸奖。但在"裴霁寒与傅灵儿曾有议亲传闻"的背景下，由裴霁寒现任道侣说出来，分量就不同了——它意味着"我知道你们的过去，但我不在意"。`
- 替换为通过角色反应暗示：`傅灵儿的目光在姜璃脸上停了一瞬，像是在确认这句话背后有没有藏着别的东西。`

## 四、不修改的项目（降级为Suggestion）

以下问题经评估，属于"可以更好但不影响发布"的优化项，本次不修改：

1. 灵胎线过于顺滑（中间无危机）— 增加危机需大幅改动多章，性价比低
2. 裴霁寒修为突破未体现 — 非核心，一句话交代即可但不紧急
3. 苏巧角色存在感薄弱 — 合并角色设计的固有限制
4. 苏婉语言过于现代 — 属于风格一致性的细节优化
5. 做饭/灵米糊的现代感 — 小说卖点之一是烟火气，刻意仙侠化反而失味
6. 第7章字数超标 — 不影响阅读体验
