# 情节骨架

## 原作概况
- 原作：揣崽后和死对头结婚了
- 总章数：55章
- 总字数：188,668字
- 目标字数：45,000-50,000字（硬上限 50,000）
- 改编方向：仙侠女主言情（原作BL/ABO → 仙侠修真 + 女主视角）

## 主线弧线

1. 【起】第1-10章：被师兄背叛 → 意外与死对头灵脉共鸣 → 发现灵契绑定（不可逆）→ 决定保留灵契
   - 原作映射：被劈腿 → 醉酒发生关系 → 发现怀孕 → 手术台反悔决定生子
   - 仙侠化：女主被青梅竹马师兄背叛（发现师兄与同门师姐私通修炼禁术），悲愤之下灵力暴走，被宿敌（大宗门嫡传）救下，双修化解灵力暴走时意外产生灵契共鸣（不可逆的道侣契约）

2. 【承】第11-25章：被迫结为道侣 → 同居（宗门赐居所）→ 日常相处 → 男主表白 → 女主接受追求 → 双向表白
   - 原作映射：同居 → 商业合作 → 车祸暴露 → 逼婚 → 领证 → 表白 → 暗恋揭秘
   - 仙侠化：灵契暴露后宗门规矩必须结道侣，女主约法三章（只是名义道侣）；合作炼丹/开发秘境；女主灵力暴走暴露灵契给长辈→长辈推动正式结契

3. 【转】第26-45章：感情升温 → 求契（道侣结契大典）→ 婚姻公开 → 敌对宗门阴谋 → 女主智斗取证击溃阴谋
   - 原作映射：H市旅行 → 求婚 → 付家订婚宴官宣 → 付余阴谋 → 智斗取证
   - 仙侠化：宗门圣地感悟 → 道侣结契大典 → 敌对势力利用灵矿坍塌事故制造舆论→ 女主以幻境重现术取证反击

4. 【合】第46-55章：灵契结果显现（灵胎/传承降世）→ 满月认祖归宗 → 敌对势力终局 → 大婚
   - 原作映射：产前 → 生产 → 满月礼反击 → 付家终局 → 婚礼
   - 仙侠化：灵契催生灵胎 → 灵胎降世 → 认祖大典上敌对势力捣乱被反击 → 大宗门正式大婚
