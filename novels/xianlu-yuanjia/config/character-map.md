# 角色映射表

## 主要角色

### 姜璃（原：江栎）— 女主
- **原设**：男，Omega，婚庆公司总经理，江家独子
- **新设**：女，散修出身后入青云宗，炼丹天赋极高但灵根资质一般（单火灵根）
- **年龄**：19岁（筑基中期）
- **外貌**：杏眼微挑，肤白如玉，身形纤细但不柔弱，右腕有一道浅金色灵痕（旧灵术共修留下）
- **性格档案**：
  - 核心特质：嘴硬心软、护短、爱告状、好面子但内心敏感
  - 处事方式：表面强势实则缺安全感，被背叛后更加防备，用"毒舌"掩饰脆弱
  - 说话方式：语速快，喜欢反问句，生气时语调上扬，害羞时声音变小
  - 口头禅："你以为你是谁？""少来这套""……关你什么事"
  - 情感弧线：被渣→封闭内心→被男主真诚打动→逐渐敞开→主动表白
- **能力**：炼丹术（青云宗同辈第一）、基础剑术、灵力感知敏锐
- **别名/称呼**：璃儿（长辈叫）、姜师妹（同门叫）、小丫头（裴霁寒偶尔叫）

### 裴霁寒（原：沈霁昀）— 男主
- **原设**：男，Alpha，沈氏集团继承人
- **新设**：男，天玄宗掌门之子，剑修天才（天生剑体），金丹中期
- **年龄**：22岁
- **外貌**：剑眉星目，身形修长，气质冷冽如霜雪，常着白衣佩长剑"寒渊"
- **性格档案**：
  - 核心特质：表面冷漠寡言、实则细心温柔、暗恋多年隐忍不发、对姜璃有绝对耐心
  - 处事方式：对外冷面无情做事果决，对姜璃则变成"嘴笨心细"型
  - 说话方式：话少，每句都在点子上，表白时反而结巴，紧张时习惯握剑柄
  - 口头禅："嗯。""……我负责。""别乱跑。"
  - 情感弧线：暗恋七年→终于有机会接近→小心翼翼追求→被接受→守护
- **能力**：天玄剑法（宗门最强）、精通阵法、灵力雄厚
- **别名/称呼**：霁寒（长辈叫）、裴师兄（同辈叫）、裴大少/死对头（姜璃叫）

### 苏婉（原：苏时）— 女主闺蜜
- **原设**：男，江栎好友
- **新设**：女，青云宗内门弟子，姜璃同期师姐，后入姜璃炼丹坊做管事
- **年龄**：20岁
- **性格档案**：
  - 核心特质：仗义直爽、嘴快心热、八卦属性满点、对姜璃极其护短
  - 说话方式：大嗓门，爱用反问，生气时直接动手
  - 口头禅："我跟你说！""他活腻了是吧？""你信不信我揍他？"
- **别名/称呼**：婉姐（姜璃叫）、苏师姐（同门叫）

### 黎沐辰（原：李然）— 渣男师兄
- **原设**：男，江栎前男友
- **新设**：男，青云宗外门弟子，姜璃青梅竹马师兄，与周蓉私通修炼禁术
- **年龄**：21岁
- **性格档案**：
  - 核心特质：表面温文尔雅，实则利用姜璃资源修炼，懦弱不敢担当
  - 说话方式：温和表象下满是狡辩
- **别名/称呼**：沐辰师兄（姜璃以前叫）、黎师兄（同门叫）

### 周蓉（原：邹融）— 渣女师姐
- **原设**：男，李然新欢，江栎大学室友
- **新设**：女，青云宗内门弟子，姜璃同门师姐，与黎沐辰私通，后与傅渊勾结
- **年龄**：21岁
- **性格档案**：
  - 核心特质：表面和善实则嫉妒心强，善于伪装，挑衅时笑着说刺人的话
  - 说话方式：阴阳怪气，喜欢用"姜师妹"称呼强调优越感
- **别名/称呼**：周师姐（同门叫）

## 重要配角

### 姜夫人（原：江母/容容）
- **新设**：姜璃母亲，青云宗长老之妻，与裴夫人是旧交
- **性格**：温婉但有主见，疼爱女儿，擅长从女儿角度劝说

### 裴夫人（原：沈母/高阿姨）
- **新设**：裴霁寒母亲，天玄宗掌门夫人，极力撮合二人
- **性格**：热情大方，喜欢姜璃，会站在姜璃这边"教训"儿子

### 姜长老（原：江父）
- **新设**：姜璃父亲，青云宗执事长老，管理宗门炼丹事务

### 裴掌门（原：沈父）
- **新设**：裴霁寒父亲，天玄宗掌门，严肃但内心关心儿子
- **秘密**：与裴夫人已秘密分居一年（分房修炼），但对外维持关系

### 苏巧（原：苏巧+方远，合并）
- **新设**：苏婉的师姐，已结道侣，姜璃的好友团成员
- **性格**：成熟稳重，偶尔调侃

### 傅灵儿（原：付欢）
- **新设**：紫金阁（敌对势力）阁主之女，曾与裴霁寒有过议亲传闻
- **性格**：外柔内刚，被家族控制，最终在姜璃帮助下脱离家族

### 傅渊（原：付余）
- **新设**：紫金阁大公子，阴险狡诈，为夺灵矿控制权设计陷害天玄宗
- **性格**：城府极深，手段阴毒

### 林启（原：林齐）
- **新设**：傅灵儿未婚夫，阵法修士，内心正直
- **性格**：沉稳寡言，暗中保护傅灵儿

### 闵长老（原：闵祁）
- **新设**：青云宗长老，被姜长老调来分管姜璃的炼丹事务，后与姜璃和解

### 裴乐安（原：沈乐安/安安）
- **新设**：姜璃与裴霁寒的灵契之子，灵胎降世
- **别名/称呼**：安安、乐安
