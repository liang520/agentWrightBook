# 角色映射表

## 核心角色

### 沈廷 → 陆昭
| 项目 | 原作 | 新作 |
|------|------|------|
| 姓名 | 沈廷 | 陆昭 |
| 别名/称呼 | 沈侍巾、沈侍君、沈承使、沈侧君 | 七殿下、昭王、陆七、小七 |
| 身份 | 最低位侍巾→侧君 | 废物七皇子→郡王→亲王 |
| 真实身份 | 现代爱豆/穿越者 | 现代音乐人/穿越者 |
| 原型 | 扮猪吃虎型 + 热血少年型 | 同 |
| 性格 | 看似蠢但真诚，嘴毒但仗义，暴躁但护短 | 同保留 |
| 说话方式 | 现代口语混入古风场景，爱吐槽，关键时嘴硬 | 同保留 |
| 口头禅 | "离谱""你们这儿玩儿得挺花啊""去他妈的" | "离谱""这朝堂比娱乐圈还黑""管他什么东西" |
| 成长线 | 侍巾→侧君→独立宫斗 | 废皇子→郡王→亲王→权倾朝野 |

**性格档案**：
陆昭的核心矛盾是"现代人思维 vs 古代权谋规则"。他不蠢，只是不按古人套路出牌。他的直觉远强于分析能力——看不穿阴谋但总能在关键时刻做出正确选择。对认定的人掏心掏肺，对欺负弱者的人毫不留情。演技极差但真诚打动人。在感情上迟钝但一旦认定就死心塌地。

|------|------|------|

| 身份 | 大雍女帝/穿越者 | 周国女将军/驻京使臣/穿越者 |

| 说话方式 | 简洁有力，偶尔毒舌，说"笨蛋"时带温柔 | 同保留 |
| 成长线 | 女帝→回现代→导演 | 敌国使臣→被召回→重返大雍→联姻 |

**性格档案**：
萧凛是全书最冷静的存在。她用导演的视角看待这个世界的"剧本"，能跳出局势做判断。她对陆昭的态度从"烦他笨"到"离不开他的真诚"。她不会直接表达感情，但行动处处为他铺路。嘴上骂着"笨蛋"，手上从不停下保护。
