# 情节骨架

## 原作概况
- 原作：《我是男主，我在宫斗》
- 总章数：59章（含2个番外）
- 总字数：204,511字
- 目标字数：45000-50000字（硬上限 50000）
- 改编方向：方案B — 穿越朝堂争霸男频

## 核心改编思路

将女尊后宫宫斗改编为架空王朝朝堂争霸：
- 女尊世界 → 标准男频架空王朝（大靖）
- 男妃争宠 → 皇子夺嫡 + 朝堂权斗
- 后宫位份升级 → 朝堂官职/爵位晋升
- 皇帝宠爱 → 军功/政绩获取权力
- 女帝萧凛 → 敌国（北燕）女将军/公主，以使臣身份驻京
- 太后干政 → 摄政王把持朝政
- 蛋生线 → 替换为伪造军情/假密旨策略

## 主线弧线

1. 【起】第1-6章（原1-12）：穿越废物皇子开局
   - 现代青年穿越成大靖最不受待见的七皇子，被扔到偏远封地
   - 偶遇同为穿越者的北燕女将军（驻京使臣），互认身份结为同盟
   - 与摄政王势力初次对抗，在朝会上意外展示才能
   - 暗恋女将军的自觉，但身份悬殊

2. 【承】第7-20章（原13-39）：朝堂争锋 + 权力攀升
   - 下毒案揭开朝堂暗流：外族间谍（贺兰线）暴露
   - 假密旨计策引诱摄政王一党自相残杀
   - 宇文家（摄政王外戚）覆灭的连环计
   - 与女将军确认感情

3. 【转】第21-29章（原40-51）：女将军被迫归国 + 独自征战
   - 中秋宴遇刺 → 女将军被北燕召回
   - 男主失去最大助力，独自面对朝堂残局
   - 原主记忆觉醒获得家传兵法/谋略加持
   - 连续击破容昭（暗卫线）、宇文策（最后的外戚）
   - 宇文策心死而亡

4. 【合】第30-34章（原52-57+新编）：出征重逢 + 登顶
   - 主动请缨出使/出征北燕边境
   - 通过周王考验，与女将军重逢
   - 粉碎最后一个朝堂阴谋
   - 联姻结盟 + 封王拜相 + 权倾朝野
