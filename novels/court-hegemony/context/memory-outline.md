# 全书记忆大纲

## 故事进展摘要
现代音乐人陆昭穿越成大靖废物七皇子，与同为穿越者（前世导演）的北燕女将军萧凛结盟对抗摄政王陆恪。经千秋宴策论一战成名，暗查西戎间谍贺兰烈并设"地狱戏"将其彻底拿下。封郡王后，与摄政王之子宇文策展开暗斗：伪造军情逼摄政王亮私兵底牌，遭宇文策纵火毁证反击，金銮殿以陆家军十万兵马要挟获缓刑，最终当庭揭底翻盘。晋封亲王后与萧凛互表心意，但中秋宫宴遇刺后萧凛被急召回北燕并失忆。陆昭孤军奋战中原主残魂觉醒传授家传兵法与暗桩网络，设局揭露内奸容昭、引发韩明远倒戈致王党分裂、宇文兄弟双亡。摄政王根基瓦解后，陆昭北上找萧凛，通过萧铎沙盘考验，远程粉碎通敌弹劾，联姻大典晋摄政亲王。原主残魂执念了结消散，全书以"他终于不是棋子了"收束。
