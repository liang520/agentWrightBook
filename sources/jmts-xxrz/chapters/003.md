# 第3章

叶柠才刚坐下不到十分钟，副导演就过来催：“差不多得了，也不露脸，把头发做做就行了，别化那么精细的妆耽误时间！全组人都在等着呢！”
化妆师应了一声：“就做了发型，没化妆，再五分钟就好。”
没化妆？副导看向镜子里的叶柠。
叶柠静静地坐在那，没化妆也是浓颜美人。
想到这张脸本来就该是主角，结果沦为给别人当替身。他有些可惜的看一眼叶柠，语气缓和了不少：“尽量快点吧，一会儿还要套招。”
田甜连忙答应下来。
“套招？”叶柠抬眸，“套什么招？”
田甜有点怂：“柠姐，裴哥不让我告诉你，今天的替身戏有打戏。剧组说没有合适身材的武替，要让您亲自上，他说……”助理声音越来越小，“您是有点舞蹈底子的，就是挥挥剑而已……学学就该会了。”
化妆师嗤了一声，眼神鄙夷。
娱乐圈的美丽废物称号真是名不虚传，看来以前拍打戏就只负责站那出剑哈一声，然后摆个收剑POSS吧？
原来如此。
叶柠的头发做完，化妆师只给她涂了口红，就让她去拍了。
叶柠起身跟着田甜去片场，刚出门，走廊里传来一阵杂乱的脚步声，人群簇拥着一个穿着和叶柠一样的白色古装，妆容十分精致的女生走过来。
萧袅袅。
也是这部戏现在的女主角。
看见叶柠，萧袅袅笑着热情地上前打招呼：“柠姐！没想到你真能来！我也就是和制作人提了一嘴，没想到啊……真是辛苦你了！不愧是好姐妹！”
周围人八卦的看向叶柠，按照叶柠的脾气，被这么冷嘲热讽一顿，这还能不炸？
田甜连忙上前一步，正准备找借口拉叶柠走，叶柠已经微微一笑：“不客气。不过，我们应该不熟吧？”
原主的记忆中，就没有什么关于萧袅袅的正面记jsg忆，两人每回见面，都是表面微笑客气，内心把对方贬得一无是处。
“你！”萧袅袅整个人要炸了。
“赶时间，借过。”叶柠拨开萧袅袅，从人群中走了。
八卦的人们扭头看向萧袅袅，按照萧袅袅的脾气，这还能不炸？
果然，下一秒，萧袅袅的尖叫声穿透整个片场：“叶柠！你给我等着！”
田甜心惊胆战地把叶柠领到武术指导面前，按照裴哥的交待，扭头就给卓青裴打电话，告知工作进度以及现场情况。
隔着电话，田甜都能感受到卓大经纪人的怒火。
挂了电话，她缩了缩脖子。
完了，她可能又给柠姐惹事了！
从套招到实拍，导演只给了五分钟。
五分钟，也只够武行们给叶柠做个示范，连慢动作走过场的机会都没有，就被导演喊准备。
“导演，我们这还没套好招，万一伤着人怎么办？”武术指导抗议。
虽然刀剑都没有开刃，但也是真的剑，用力之下，如果被划到，还是容易受伤。
这次的戏又是雨中打戏，招招冲着脖子去的，很容易意外划到脸。
“那你们注意点！不然要你们干什么吃的！”导演脾气暴躁地喊，随后让现场准备。
两辆洒水车架起了水枪，有工作人员将叶柠领到庭院中间，让她站在那别动。
叶柠不明所以，下一秒一场人工大雨倾盆而至。
她下意识的抬手挡了一下雨，导演举着扩音器就骂：“那个替身！他妈的要你别动你是聋了吗？”
叶柠蹙眉，她什么时候受过这样的气？
正要离开，想到和原身的约定，最终还是放下手，任凭冰冷的人造雨落在身上。
四天前，也是在这样一场大雨里面，她遇到了原身。
那天，原身受尽委屈，中毒身亡，临死前，她的执念唤醒了蓝宝石内叶宁的残魂……
他们一个想死，一个想活。
于是，一道契约，让叶宁拥有身躯，而作为交换，她答应替叶柠了却执念。
雨水顺着脸颊滑落，凡胎的身体止不住的发抖，过了有十分钟，导演才喊了卡。
田甜立刻拿着毛巾上来，给叶柠披上。
按照拍摄计划，接下来就拍摄女主的打戏，拍完就收工。
然而，叶柠这一场卡了之后，萧袅袅就直接上去拍了。
工作人员穿着雨衣，钢丝在萧袅袅头顶架了个大伞，萧袅袅站在伞下面，化妆师用水壶喷湿萧袅袅的头发和脸，就开始正式拍摄了。
“也太过分了吧！”田甜愤愤地说，“柠姐你替她淋雨，到时候播出，你的背影就会变成她的，所有都会以为是她自己敬业的淋雨的！”
叶柠喝着姜汤，淡淡道：“拿什么钱做什么事，我拿的不就是替身的工钱？”
田甜僵了一下，安慰她：“柠姐，裴哥的话，您别往心里去，他就是刀子嘴……”可也不是什么豆腐心。
否则，也不可能在柠姐最难的时候，逼她去做替身。
要知道柠姐以前可是演主角的！
替身是什么啊？
是永远不会在大屏幕上露出脸，甚至连名字都不会出现的！
这就是在逼柠姐退出娱乐圈！
不仅如此，裴哥还安排柠姐去参加恋综，那种真人秀能随意参加吗？
一旦说错一句话，做错一点，都可能会被网友攻击。就算没有做错，以现在柠姐的处境，保不准那些人就会操控节目组，对柠姐的部分恶意剪辑，到时候真是有视频有真相，有嘴都说不清！
还是和顾星辰组CP！想到顾星辰那群疯狂的粉丝们，每次柠姐蹭他点热度，粉丝们就铺天盖地的澄清加嘲讽谩骂。
现在真组CP了，柠姐还不被他们给撕了啊？
就算不被粉丝攻击，顾星辰那么讨厌柠姐，恐怕也不会给她好脸色看……
想想都替她委屈，田甜忍不住共情，眼圈瞬间红了。
“怎么了？”叶柠留意到小助理情绪不对。
田甜摇摇头，见叶柠杯子里的姜汤没剩多少，起身道：“我再去给您拿点姜汤！”
结果没过两分钟，小助理眼圈转着眼泪回来，说怕其他人不够喝，不给倒了。
就这？至于哭鼻子？
一杯姜汤而已。
叶柠示意田甜坐下休息，两人一起等接下来的拍摄。
然而，接下来的镜头一直都在拍萧袅袅，拍了有半个小时，萧袅袅过来休息。
她顶着微湿的头发，撵走田甜，在她的椅子上坐下来。
“真冷啊，”萧袅袅说着，从一溜小跑的助理手中接过毛巾，擦了擦几乎不存在的水渍，问叶柠，“柠姐，你冷不冷？啧啧，就算不冷，一直穿着湿透的衣服，也很难受吧？辛苦你了啊，柠姐。”
叶柠一言不发，只抬眸似笑非笑地看了她一眼，就继续低头滑手机。
那眼神，淡然的像是在看小孩耍把戏。
萧袅袅脸慢慢的涨红，她站起身，冷笑一声：“你就慢慢等吧，柠姐，希望你今天天黑之前，能拍到戏！”
叶柠略有些意外，问田甜：“要等这么久？”
田甜愤然：“她肯定是故意的！”
看来真要这么久，不早说？
叶柠闭目，用灵力慢慢烘干衣服的水。
五分钟后，叶柠叹了一口气，到底是新身体，灵力恢复得比以往慢多了，烘干衣服竟然要这么久，以前可都是一瞬的事。
直到剧组中午放饭了，还没轮到叶柠。
他们来之前，卓青裴就叮嘱过田甜，下午三点半要去拍恋综海报，这边务必要在两点半前结束拍摄。
眼看着已经快一点，田甜快急哭了。
好不容易耐着性子等午饭之后继续拍摄，却依旧是萧袅袅的戏份，甚至已经换了妆造，拍另一场戏。
下午两点，田甜像热锅上的蚂蚁，整个人都坐不住了。
两点二十分，叶柠看一眼手机，招呼田甜走人。
“可是，可是戏还没拍完啊……”田甜小声提醒。
“时间到了。”叶柠晃晃手机，换衣服去了。
田甜带着哭腔拨通卓青裴的电话，在裴哥的怒吼声中，壮着胆子拖着叶柠去问了导演拍摄进程，果不其然，被骂回来了。
两人出了片场，卓青裴的车刚到。
副驾驶的窗户降下，一个带着无边眼镜头发梳得一丝不苟的男人看向窗外，探寻的视线落在小助理的身上。
小助理一脸为难的摇摇头。
卓青裴一抬下巴：“上车。”
半个小时后，摄影棚内，叶柠跟着工作人员进更衣室换衣服。
卓青裴的手机嗡嗡直响，剧组副导演打来兴师问罪。
一接通，对方国骂开场，卓青裴等对方骂完才开口。
“您不用提醒我，周导，我清楚做替身这事是李总安排的，”他扶了扶眼镜，“但您也清楚，恋综也是李总的安排。”
他顿了顿，继续说：“我们不过是帮李总做事，事办成了不就行了吗？再说，我看李总的意思，替身不过是让她吃点苦头。我这边，才是……一场大戏。”
田甜拎着两袋咖啡进来，挨个给工作人员发。
卓青裴也不便再多说，他笑着答应，如果结束之后还有需要，可以让叶柠过去补那场没拍完的戏。
刚挂了电话，外面一阵喧嚣。
顾星辰带着工作人员到了，身后一群粉丝拉着横幅，在大楼外面激动得欢呼尖叫。
进来之后，顾星辰的经纪人和工作人员沟通后，接过衣服带着顾星辰去更衣室。
叶柠一出更衣室，就和顾星辰撞了个正着。
她穿着一身碎花短裙，看起来温柔可人，然而她的眼神却一点都不温柔。
“顾星辰。”她叫住顾星辰。
顾星辰回头，看向她：“嗯？”
许鸣神经紧绷，四处扫了一遍。
叶柠肯定又找人偷拍准备蹭热度！
人藏哪儿了？
周围只有叶柠的经纪人卓青裴拿着个打火机鬼鬼祟祟，许鸣眼睛一眯，一脸严肃的走过去。
“剩下的三颗宝石，尽快找齐。”叶柠说。
“我凭什么帮你找宝石？”顾星辰顿时不爽，“不对，那是我顾家的宝石！”
“剑是我的。你太爷爷昨晚没和你说？”
“……！！！”
他梦见太爷爷这事，明明只和许鸣在车上说过，她怎么知道？
难道说，许鸣这浓眉大眼的小伙子，也叛变了？！
