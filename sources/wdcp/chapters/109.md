# 第一百零九章 赠刀！

光芒一闪。
武小德落在一片满是狼藉的超市里。
嘶嘶嘶——
外面传来一阵声响，听上去像是怪物们之间的交流。
武小德朝外一看。
之间街道上满是黑色的长蛇，密密麻麻，不知绵延到什么地方去。
白虎悄然出现，低声道：
“这个世界上到处都是杀戮——我们几乎可以任意穿梭。”
“刚才是不是走的很及时？”武小德问。
“是的，你可以看看这些伥鬼。”白虎道。
在它身周，那些伥鬼一个个悄然显现。
它们脸上都浮现出痛苦的神情，身体上不时散发出一道道“滋滋滋”的声响。
血肉几乎被腐蚀一空，依稀可见白骨。
“尽管已经是弱化版的万象森罗，尽管有这么多伥鬼帮我抵挡……恐怕再来一次，就挡不住了。”
武小德喃喃道。
“它们可以恢复，不过需要时间。”白虎道。
武小德叹息一声，说道：“能找到和圣教有关的地方吗？我们去看看这个平行世界之中，圣教又如何了，也许会对我现在的局面有帮助。”
“上来吧，我带你跑一圈看看，也许能找到圣教。”白虎道。
“多谢。”武小德道。
他骑上白虎的背，白虎立刻朝前冲去。
光影一阵模糊。
武小德出现在一处树林之中。
只见这里盘踞着几头半人半马的披甲怪物，正在与一群蜥蜴争夺死尸。
——魔怪也不是团结一致的嘛。
趁着这些怪物还没回头朝这边望，武小德拍拍白虎的头道：“找个有人的地方。”
“好。”
白虎朝前一跃。
一人一虎顿时从原地消失。
下一秒。
他们出现在一处宽阔的大街上。
唰——
一根箭矢穿过街道，射在一头猴脸怪身上，将它直接钉在墙上。
这箭矢好眼熟啊。
武小德扭头一看，一下子就看见了赵芷冰。
此刻她没有戴面具。
她没有扮演S级俱乐部的团长，而是作为公主出现在这个平行世界。
在她身边站着一头仙鹤。
“小武！”赵芷冰喊出声道。
“殿下，你也来了？”武小德招呼道。
他骑着白虎跑到赵芷冰面前。
“你怎么突然出现在这里的？这位是？”赵芷冰看着白虎问。
“我和这位名为‘白’的灵有契约，它可以带我在杀戮之地穿梭，这位是？”武小德望着仙鹤道。
“羽，我的契约灵。”赵芷冰介绍道。
“你好，羽。”武小德道。
那仙鹤歪着头看了看他，点点头，没说话。
“你好，白。”赵芷冰朝白虎道。
“你好，人类。”白虎口吐人言道。
赵芷冰心中泛起一股莫名的情绪。
本以为武小德只是一名武师，要经过一段时间的培养才可以独当一面。
谁知道他有着这么多稀奇古怪的能力，还跟灵扯上了关系。
武小德倒没想这么多。
他四下一望，不远处是皇宫。
果然，赵芷冰用了空间传送仪，锁定了平行世界的皇宫。
上次跟她说过之后，她已经被皇帝赐予了权限。
——她是来读取数据的！
“我先助你一臂之力吧，晚点再去做别的事。”武小德道。
“你那边发生什么事了吗？之前我们怎么联系你，都联系不上。”赵芷冰道。
“我正在跟教宗决斗，他有一个能力可以锁定某个空间，也许是因为这样，你们就联系不上我。”武小德解释道。
赵芷冰呆了呆，忍不住道：“这下麻烦了，教宗有随时可以返回现实世界的手段，而你不得不杀够怪物才可以回去——”
“他一定在你传送离开的地方设下了埋伏。”
她竟然这么快就想到了这一点！
真是厉害啊。
武小德心中赞叹，开口道：
“没事，我有一些想法，如果实现了，胜负还难说。”
趁着四周的怪物还未出现，他浑身一震，放出数不清的魔雾将两人笼罩。
“走吧，我送你进皇宫。”武小德道。
“好！”赵芷冰露出笑容。
武小德的迷雾经受住了各种考验，一直没有被怪物破解，值得信赖。
两人穿过街道，径直来到皇宫门口。
武小德正要扫码，谁知大门里传来一道道的声音：
“恭迎公主殿下回宫。”
轰隆隆隆——
大门直接打开。
里面两排机器人跪在地下，跟人类一样行礼。
直到这时，才有一个机器人小声道：
“欢迎尚方二维码的持有者随公主归来。”
武小德耸耸肩。
好吧，排场本就不一样。
“免礼，起来做好防御工作。”赵芷冰道。
“是！”机器人们齐声应道。
赵芷冰和武小德一起走了进去。
大门在他们身后轰然合上。
“我早该来皇宫看看的……只不过一直没想到这里会是平行世界，而这个世界的地图又那么难以弄到。”赵芷冰感叹道。
“谁能想到这里会是三年后平行世界。”武小德也摇头道。
两人走进大殿。
那个头盔依然好端端的放在龙椅上。
“如果知道了这三年间会发生什么……也许我们就能更好的应对一切。”
赵芷冰直接走到龙椅上坐下来，将头盔戴在头上。
仙鹤就守在她身边。
过了一会儿。
她又拿出一个巴掌大小的黑盒子，从里面牵了一根线，连接在头盔上。
——这就是拷贝数据了。
武小德放开迷雾，彻底笼罩四周，小心翼翼的做着警戒工作。
“放心，附近没有厮杀。”仙鹤开口道。
它的目光和白虎对上。
“方圆十里内没有，十里之外有一场厮杀，是怪物跟怪物之间的食物争夺。”白虎道。
“我当然知道，准确的说，那是十二里之外的一个地方。”仙鹤道。
“确切的说，那是十二里三丈。”白虎道。
“你要这么算，应该是十二里三丈六尺。”仙鹤道。
“一个怪物死了。”白虎道。
“但是眼睛没闭上。”仙鹤道。
“心已经不跳了。”白虎道。
“你非要跟我杠是吧。”仙鹤道。
“眼睛闭上了。”白虎道。
“有意思吗？”仙鹤道。
“你们两个——够了！”武小德和赵芷冰异口同声道。
仙鹤转过身子，不看白虎。
白虎也哼了一声，转头不看仙鹤。
二十分钟后。
赵芷冰把东西收好，站起来道：“可以了。”
“好，那你先回去，我在这边找找线索，看能不能破解教宗的手段。”武小德道。
赵芷冰把他从上到下看了一遍，忽而道：“你的吹梦刀呢？”
“碎了。”
“怎么碎的？”
“教宗借了‘神’的力量来了一招，把刀弄碎了。”
“那你后面怎么打？”
“我还会拳法。”
“……你等一下。”
赵芷冰从袖子里取出一张符箓，轻轻撕碎。
嘭！
一柄长刀出现在她手中。
“此刀名为镇国，是我赵家祖先当年随身佩刀，父皇生怕我在异世界吃亏，什么都给我带上了，现在正好给你用。”赵芷冰道。
“这……会不会太贵重了。”武小德迟疑道。
“命比任何东西都重要，先活下来再说别的。”赵芷冰将刀塞在他手中。
“好。”武小德深深看了她一眼，也不再矫情。
赵芷冰转身走回宝座上，将头盔再次戴上。
“你去吧，我在这里再查查资料，希望等我回去之后，能听到你打电话过来报平安。”赵芷冰淡淡的道。
“放心，我才不会输给那个家伙。”武小德道。
他骑上白虎，身形一闪便从大殿之中消失。
他走之后。
又过了一会儿。
赵芷冰慢慢的把头盔摘下来，放在一旁。
“虽然性命最重要……”
“但本公主竟然连手下的拳师都护不住。”
“真是丢人啊。”
她紧了紧背后的长弓和箭筒，又摸了摸腰间那成捆的符箓，迈步朝大殿外走去。
轰隆隆——
大门打开。
赵芷冰走出皇宫，随手摸出一张符箓贴在身上。
她身上的气息顿时被隔绝。
她又摸出一张符箓贴在身上。
她的身形渐渐变得透明。
她又摸出一张符箓贴在身上。
她的脚步变得轻盈。
她又摸出一张符箓贴在身上。
她的四周布满了无形的防御之力。
她又摸出一张符箓贴在身上。
她又摸出一张符箓贴在身上。
她又摸出一张符箓贴在身上。
……足足贴了一百多张符。
“冰冰，可以了，再贴别人会以为你是僵尸。”仙鹤无奈的提醒道。
赵芷冰一顿。
“是的，羽，你说的对。”
她抽出一张符，朝仙鹤身上贴去。
“我不需要！”仙鹤连连朝后跳去。
“别躲，上次你来的时候，我就想要准备一张这样的符给你，它是用来滋润羽毛的。”赵芷冰道。
“真的？”仙鹤半信半疑。
“我骗你干什么，来试试。”赵芷冰将符箓贴在仙鹤背上。
符箓上顿时闪现出一抹光华。
那一片羽毛变得更白了，甚至还散发出盈盈光彩。
仙鹤大喜，连声道：“快，还有没有，多给我几张！”
“当然有，我这里有专门为你调理身体的符箓，还有护身的、抗法的、飞行速度加快的……”
一刻钟后。
一个身上贴满符箓的人，和一个浑身贴满符箓的鹤，从皇宫里走出来。
“开始战斗吧。”仙鹤说道。
它那长长的尖喙上挑着七八张符箓，风一吹，飘飘荡荡，煞是怪异。
“没错，我们需要制作一些封魔符，以后再来了，一旦发现危险，就可以立刻回去——要多捕捉一些魔怪才行。”
赵芷冰自言自语着，朝着街道尽头的一头魔怪望去。
那魔怪原本正在发呆，不知为何，忽而哆嗦了一下，背上泛起阵阵寒意。
