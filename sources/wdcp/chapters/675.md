# 第六百七十五章 时间之主！

时间之主的旗帜？
好巧。
我也有一柄！
要不要再多搞一柄来？
——这根本不用想！
但不能表现的太迫切，否则会引起它们怀疑。
“以你们的实力，想在这人类的世界弄一两件宝物，还不是手到擒来？”
武小德抱着双臂道。
男子道：“并非如此，那个地方有涌现级的术法守护，对应地水火风，至少要四个生命才可以开启。”
另一人插话道：“我们也是寻找了很久，破解了无数难题，这才找到这里。”
原来如此。
“好，我加入。”武小德道。
男子欣然道：“夺得旗帜之后，我们四个轮流用——还没给你介绍，我们现在都伪装成了人类，我的人类名字是策主，他是彪，旁边那位是枯手。”
三人一起望着武小德，想听听他的人类名字。
“你们可以叫我小明。”
武小德道。
小明？
这个名字似乎极其常见啊！
对方果然比自己更了解人类！
三位奇诡生命都在心中默默的想着。
策立刻说道：
“跟我来。”
他走到墙角，伸手一按，虚空中顿时浮现出一个满是诡异符文的传送阵。
“那位法则之主的旗帜埋藏在地下，我们必须传送下去。”彪解释道。
武小德点点头，没说话。
——现在回想起来，刚才它们一直在试探自己。
看来自己是合格了。
策主激活传送阵。
传送阵顿时冒出四道流光，分别打在四人身上。
下一瞬。
其中三人直接从原地消失。
名为“枯手”的老头儿却停留在最后，身上放出密密麻麻的蚯蚓状生物，在整个房间里蠕动了数息。
另一边。
地下。
光芒一闪，武小德顿时现出身形。
然而身边只有策主和彪，那个老头儿却没有出现。
前方是一条黑漆漆的地道。
也不知通往何方。
“还有一位呢？”
武小德问。
策主和彪相视一笑。
“他是伪装高手，负责扫尾，清除一切痕迹。”彪解释道。
武小德顿时恍然。
——刚才自己都没看出那几具尸体是假的。
这家伙可能是专精于此道的奇诡生命。
三人等了数息。
枯手不紧不慢地出现在他们眼前。
“痕迹都清理干净了，就算是其他的同类，也只能看出这里曾经有人停留，绝对找不到我们。”
他得意地说。
“出发！”
策主一马当先，走在最前面。
地道并不长。
一会儿工夫，四人就来到了一处空旷的地下岩洞。
武小德仔细打量，只见这地下岩洞并没有什么特殊的地方，唯一引起自己身上“万物要素察觉”的就是洞内的四个角落。
每个角落上空无一物，但若是注视着那片虚空，便会感应到某种让人忍不住想要顶礼膜拜的强大力量。
“四个角落，其实没什么可选，各自挑一个站着，才可以激活这里的力量。”
策主道。
武小德偷眼去看其他两人。
只见他们跟策主一样，也是一脸无所谓的表情。
“万物要素察觉”也告诉自己，这里只是一种激活仪式的固定方法。
那么——
随便站吧。
武小德走到一处角落，站定。
其他三人各自寻了一个角落站定。
四个角齐了。
他们脚下忽然浮现出淡淡的光芒。
霎时间。
光芒散开。
整个熔岩洞穴不见了。
天色昏黄。
残阳如血。
满地皆是穿着甲胄的人类尸体。
——这里是一处战场。
数不清的人类正在跟妖魔交战。
“现在怎么办？”
武小德躲避着飞来的箭矢，大声问道。
“分头找！”策主也大声道。
话音落下。
三人直接分作三个方向，瞬间飞掠而去，消失得无影无踪。
武小德微怔，旋即反应过来。
——它们肯定有某种情报！
但是它们并没有告诉自己。
显而易见，尽管它们觉得自己可以信任，也不愿意拿出情报给自己看。
兴许自己的作用只是站一个角，打开这个时空的门。
一旦它们找到那旗帜，就可以彻底抛弃自己了。
“真是用人朝前，不用人朝后啊。”
武小德感慨地摇摇头，忽而上前一步，挡在一名人族士兵背后，揍飞了一只偷袭的妖魔。
谁知他一出手，立刻引起了更多妖魔的注意。
这些妖魔浑身死灰色，被迷雾缠绕着，不时露出浑身骸骨，看上去就像是死亡世界的怪物。
前来攻击武小德的妖魔越来越多。
——然而这种烈度的战争对他而言已经毫无危险可言。
他索性抽出蝶魄刀，在妖魔近身之前就将它们切割成一片片血肉。
渐渐的。
妖魔们就不敢上来了。
“多谢阁下的救命之恩。”
获救的人族士兵感激道。
“不必客气。”武小德摆手道。
那人族士兵东张西望了一番，忽而传音道：
“阁下，您在这里等一会儿，那柄旗帜被藏的有点深，它感应到您身上的另一柄旗帜，正在跋山涉水的赶过来。”
武小德眼睛猛然睁大，猛地后退几步。
他看着那名士兵。
只见士兵大约五十岁出头的模样，双目炯炯有神，浑身气势非同凡响。
那一身拙劣的制式战甲简直委屈了他那满是粗壮肌肉的身形。
“你是何人？”
武小德沉声问道。
“我等你很久了，不过幸好我有时间，所以最终在启程前等到了你。”士兵道。
他轻轻抬起手在虚空中一按。
无形的涟漪从他手上散发出去，迅速穿过整个世界。
战场上。
一切陷入停滞。
无论是士兵还是妖魔，都保持着原本的姿势，一动不动的静止在原地。
士兵点燃一根烟，叼在嘴里，又拿出烟盒给武小德发了一根。
武小德伸手接了烟，点燃，吸了一口，继续问道：
“现在可以说了吧，阁下究竟是谁？”
士兵道：“在很久以前，其实并没有时间存在，一切都在永恒中同时发生，这种全始全终叫做圆满。”
“后来么。”
“文明作为序曲，出现在虚空之中，众生不了解这种圆满，迫切需要一种衡量标准，所以才有了时间。”
士兵深深吸了一口烟，张嘴吐出久久不散的烟圈。
武小德听到这一刻，渐渐有些明白过来。
然而不等他说什么，对方继续说了下去：
“时间就像一个圆，本是圆满无暇，但你若非要去分辨，它便会呈现如宛如一枚硬币的形态——”
“它具有两面性。”
“小武，你让‘永恒的升序’反转过来，现在又护住了‘盘古’的初序，这都是从未有人做到的事。”
“你给了我信心。”
“所以我留给你两面旗帜，希望你能用好它们，至于另外两面旗帜，我要带到时间的另一面去，帮助那条世界线上的人战斗。”
士兵说完，只见远空中有什么东西一闪而至。
——却是一面紫色的旗帜。
“您是时间之主。”
武小德道。
“是我，不过我已经等到了你，现在要离开了。”
士兵将旗帜递给他，继续道：
“记住，在释放‘超级暗影随从’的同时，使用这两柄旗帜，会有意想不到的惊喜。”
说完，他身形一闪，冲上天空，消失不见了。
武小德站在原地，手中握着那柄紫色的旗帜，一时说不出话来。
虚空中渐渐浮现出一行行冰晶小字：
“你获得了‘无间之旗’。”
“时间的被动之旗。”
“持有此旗帜，即可阻止一切时间穿越行为，令穿越者退回其来路，不得冒犯时间之威。”
“另：”
“将此旗帜与无始之旗一起使用，会有意想不到的效果。”
