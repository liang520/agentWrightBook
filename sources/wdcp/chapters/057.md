# 第五十七章 献祭

汽车发动了。
武小德看着手机，只见上面的通话已经挂断。
皇帝走了。
真是不靠谱啊。
话说回来，在某个方面，其实赵君羽还挺像他的。
武小德摇摇头，叹口气道：“老萧，我们一会儿去杀人。”
萧白红小心翼翼的道：“知道，杀人什么的都是小事，我们之间还有一件很重要的事……”
“什么事？”武小德莫名其妙。
“那个呀，就是我借给你的那个东西呀，它帮上了你的大忙。”萧白红以一种不好意思的语气提醒道。
武小德恍然大悟。
“给你。”
他掏出耳机扔过去。
萧白红立刻接了耳机，迅速的放进自己口袋里，手在口袋外按了按。
他整个人仿佛放下了一桩心事，重新露出笑容。
“杀人嘛，这种事我精通，小武你若是不想动手，我替你杀光他们。”他拍着胸脯道。
“老萧，真够意思啊，那我就不出手了。”武小德感动道。
“一个人头五百万，熟人九九折，我跟你一见如故，又一起对付过猜魔，就给你打个九八折，够意思吧。”萧白红道。
“算了……我还是自己动手吧。”武小德木然道。
这时候已经入夜。
路灯纷纷亮了起来，大街上车水马龙。
车在路上平稳行驶，很快便到了一座气势恢宏的雄壮教堂前。
武小德端详了一下教堂，开口道：
“听说我们国家现在信奉圣教的人在逐渐增加。”
“海上出现了骷髅群，到处都是地震和各种异象，这些装神弄鬼的家伙恐怕高兴坏了。”萧白红道。
他打着方向盘，绕过教堂的正门，一直来到教堂后面的小路上。
车停稳。
两人下了车，看着那高高的铁栅栏，以及栅栏里那片无人的花园和草地。
“有不少监控器。”武小德道。
“怕什么，我可是S级俱乐部的刺客，杀这些垃圾单独行动就可以了，你休息吧。”萧白红打着哈欠道。
“刚才不是说一个人头五百万吗？”武小德笑道。
“开玩笑的，我已经接了皇室的委托，怎么还会找你要钱——现在我要工作了，你如果有兴趣，可以进来参观他们的建筑和雕塑，或者去街对面吃个宵夜，等会儿来这里汇合。”
萧白红说着，跳上铁栅栏，轻飘飘的落在花园里。
刺耳的警报声顿时响起。
监控器的摄像头转过来，对准了萧白红，同时有警卫的声音从扬声器里响起：
“你是什么人？赶紧离开教堂！否则我们将立刻报警！”
萧白红冲着摄像头笑了笑，大步穿过花园，朝教堂深处走去。
武小德站在原地看着这一幕，喃喃道：
“这家伙相当嚣张啊。”
诚然，如果对方都是些普通人，甚至是一般的职业者，自己就不必为萧白红操心。
但武小德看问题的视角却和所有人都不同。
他亲眼见识过独目巨人和骨龙。
所以比起一般人来，他心中始终多了些许谨慎。
——因为这个时代已经开始展现出人类所完全没有接触过的一面。
“老萧实力不错，但……还是去看看吧。”
武小德呢喃道。
一阵雾气悄然浮现在他身周，渐渐将他彻底笼罩。
这些雾气一开始十分浓郁，几乎挡住了武小德的视线，但很快，所有的雾气渐渐融入虚空，与四周环境变得一致——
就像是变色龙一样。
亡灵之书随之翻开，显现出一行行冰晶小字：
“寂静魔雾开始与周围的环境融为一体。”
“由于寂静魔雾的死亡特性，身处魔雾中的你不会泄露任何气息。”
武小德轻轻一跃，站在铁栅栏上。
寂静魔雾跟随在他左右，缭绕不休，将他的身形彻底遮蔽住。
远远望去，铁栅栏上什么也没有。
武小德落在草坪上，抬脚就要朝教堂里走去。
他忽然又站在了原地。
一种模模糊糊的感觉浮现在心头，让他忍不住低头望向草坪。
“啊……难道因为是死亡阵营，所以我能感应到这么多的死亡么？”
他轻声呢喃着，伸出拳头，在地上用力的捶了一拳。
咚！
地面震了震。
这一拳用了巧劲，直接把泥土震开，显露出下方所埋藏的动向。
尸体。
一具又一具尸体胡乱码放在地下，早已死去多时，但却并没有腐烂。
武小德心中产生了一种别扭和不舒服的情绪。
他皱起眉头，有些不解。
亡灵之书悄然翻开，显现出一行行冰晶小字：
“你是得到死亡国度承认的亡者士兵，你能感应出死亡的特质。”
“当前尸体的灵魂并未正常前往死亡国度。”
“这对于死亡的法则是一种扭曲，因此让你自然而然的生起了感应。”
没有前往死亡国度。
那么，这些灵魂去了哪里？
武小德将手按在吹梦刀上，一步步朝院子深处走去。
萧白红去了教堂的前厅……
那么自己就从后面走，看看能发现什么。
他换了一条路，穿过无人值守的小径，一直来到教堂的一处侧门。
门是锁着的。
武小德抽出吹梦刀，沿着门缝轻轻滑下去。
两道微不可察的轻响。
推开门。
只见那门锁被斩断的地方光滑如镜。
——吹梦刀的锋利程度真的可以。
武小德走进去，将门带上，然后环顾四周。
只见这里是一处忏悔室，看上去没什么奇怪的地方。
但是武小德感应到了浓郁的死亡气息。
对于死亡阵营的他来说在，这简直犹如黑暗中的明火。
他在墙壁上找了找，很快发现了一片中空的地方。
这里一定有暗门。
但是开门的机关并不好找。
武小德索性直接抽出吹梦刀，沿着那中空的墙壁划出一个长方形。
他以手抓住墙壁上凹凸的地方，朝回一拉——
墙壁顿时被他拉了出来，轻轻放到一边。
里面果然有一条螺旋向下的台阶。
武小德向下走去，渐渐越走越深，越走越能感应到死亡的气息。
终于。
他走到了台阶的尽头。
这里又是一处教堂大殿！
想不到在教堂的地下，还藏着这么大的一处空间！
两个穿着长袍的男人站在大殿中央，正在交谈。
“情况怎么样？”那个年长的人问道。
另一个人正在手机上不断输入字符，口中说道：“武小德没死，我正在跟大家商量，看是不是请动那位地下的存在，帮我们出手杀了这个小子。”
武小德怔了怔，拿出手机打开那个群。
只见一张新的肥婆图片出现在屏幕之上：
“我提个建议，请地下的那位存在干掉武小德，大家觉得怎么样？”
啊，原来是这个人啊。
终于找到你了。
武小德站在迷雾之中，迈开脚步，无声无息的朝两人走去。
两人毫无所觉。
那个年长的男子道：“昨晚十二点，皇室在那个世界损失了好几名强者，如果这个武小德也被干掉，那皇帝身边的护卫力量就彻底无法维系了。”
“是啊，杀了他，计划才可以顺利推进。”年轻的男子也叹息道。
这时武小德已经走到了两人面前。
他抽出吹梦刀，平举，朝前轻轻一挥。
“咦？”
年轻男子的头飞起来，在半空中发现自己的视线和周围环境产生了某种不真实，忍不住发出了声音。
年长的男子脸色一变，迅速喝道：“沉睡于深星之渊的主人啊，众生与万物不过是你梦中的食粮，以此地千二百灵魂为祭，请让我借助您的伟——”
唰！
吹梦刀穿过了他的脖颈。
他的声音戛然而止。
战斗结束。
武小德正要收刀，心中忽然产生了一缕不安。
怎么回事？
他朝四周望去，只见整个大殿已经消失了。
亡灵之书迅速打开，闪现出一行行冰晶小字：
“你中招了！”
“当前战斗中，你被一种极其恐怖的献祭之术命中，请千万小心，一定不要死，否则你的灵魂将成为某种存在的食物，再也不复存在。”
世界化作了一片黑暗而寂静的深海。
在这深海之中，渐渐响起了一阵阵幽幽的女声歌唱。
两排身穿洁白长裙的女子手捧蜡烛，从海洋的深处慢慢漂浮而来。
诡异的是，她们身后都连接着大片大片的、蠕动的血管和肌肉，一直延伸到深海中数百米之外。
借着她们手中蜡烛所散发出的光，武小德可以看见远处有着一片接天连地的黑影。
——那是什么？
武小德只觉得心中冒出阵阵寒意。
他正要抽出吹梦刀，但想了想，却没有动。
自己连那两招诗句刀法都使不出来，有什么好打的？
眼下。
在这个诡异的环境里，自己唯一可以依靠的，唯有自己手上这个来自死亡国度的能力——
寂静魔雾。
想通了这一点，他索性站在原地不动。
雾气越来越多，一层又一层，将他彻底笼罩。
时间缓缓流逝。
那些女子们面无表情的站在原地，渐渐停止了歌唱，仿佛在等待着什么。
这时候武小德猛然发现了一件事。
这些诡异的女人手中并不是什么蜡烛——
它们是一个个处于燃烧中的灵魂！
这些灵魂发出痛苦的嚎叫声，却被女人们的歌声所掩盖。
女人们高举那些燃烧的灵魂，朝四周的深海照去。
为首的那个女人高唱道：“无论你在哪个世界活着，无论你是什么，在轮到你献祭的时刻，交出你的一切吧！”
所有女人跟着她唱了起来。
深海中，渐渐充满了某种暴虐而黑暗的力量。
——但是没有用。
武小德身上的寂静魔雾来自死亡国度，等同于他是亡者。
而对方的术，按照它们自身咒语的解释，只能找寻“无论你在哪个世界活着”的存在，并将之献祭。
“无论你在哪个世界活着，无论你是什么——”
这已经是相当强力的咒语了吧，因为它穿过了空间的限制，无视任何种族。
武小德心中默默想着。
一息。
两息。
三息。
时间继续流逝。
亡灵之书上再次浮现出一行小字：
“对方的术无法寻找到目标，已经卡住了，不能继续施展。”
武小德看了一眼，始终不动，也不说话。
又过了一会儿。
只见那些女人齐齐转过身，捧着那一个个燃烧的灵魂，转身而去。
女人们一去，整个世界的异象渐渐消失。
下一瞬。
武小德发现自己又回到了那个地下大殿。
四周的地面早已全部裂开。
一只只人手从地下伸出来，密密麻麻，高高的举在半空。
这些人手的掌心位置，都有着一颗眼睛，正在不断的来回寻望，似乎想要寻找什么东西。
可惜它们看不见武小德。
……刚才究竟是另一个世界，还是这些眼睛所释放出来的幻象？
武小德有些想不明白。
但他知道这个时候该做什么。
——不要动。
时间继续向前推进。
数十息之后。
这些人手没有发现任何异样，便纷纷重新缩回地下，消失不见。
武小德依然不动。
——从刚才异象发生的那一刻起，他就站在原地，一步都没有挪动，挺过了漫长的时间。
杀完人之后，他也没有将寂静魔雾收起来，依然一直保持着“假象”的笼罩。
这是为了防止出现什么无法应对的局面。
这是在干掉敌人之际，武小德所保持的一点点谨慎。
现在看来。
这是他活下来的唯一原因。
