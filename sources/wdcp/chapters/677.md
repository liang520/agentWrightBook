# 第六百七十七章 流沙！

“左右，去把这个人类杀了，头颅拿回来插在我的旗帜上！”
妖魔首领喝道。
妖魔大都长得奇形怪状。
比如这头首领——
它看上去像是人类，但浑身惨白色皮肤，胸骨和臂骨裸露在外，双眼浑浊，头生独角。
浑浊而邪恶的气息在它身上氤氲不休。
轰隆隆——
大地震动。
在首领的命令下，数支妖魔骑兵加快速度，朝着武小德发起了冲锋。
——每支骑兵队都有十几名骑兵，一共八支这样的队伍，已经足以干掉一个人类少年。
一簇一簇的术法从骑兵们手上飞出去，掠过长空，袭向武小德。
数百米开外。
武小德放松的慢跑着，仿佛并不是在迎战妖魔骑兵队，而是在做着早起的锻炼。
——直到那些术法扑面而来。
他伸出手，划过数百米长的妖魔大军，最终停留在妖魔首领身上。
霎时间。
所有追身而来的术法贴着他身子穿过去，将他身后的掩体轰成飞扬的碎泥烂砖。
一行行冰晶小字浮现：
“你发动了‘无限鉴定’。”
“你指定了战斗对象。”
“本次战斗受到‘垂钓’序列的全程见证，其他任何存在和力量不得插手和干涉。”
“——来场一对一吧。”
与此同时，武小德动了。
他的奔跑速度猛然提起来，越跑越快，于狂奔中抽出腰间的蝶魄刀，整个人如炮弹一样撞入妖魔群中。
妖魔大军中无数兵器和术法爆发开来，却无一能伤他分毫。
长刀扬起——
妖魔首领见势不妙，抢先出手，将手中长杖狠狠敲在武小德的刀脊上。
“等的就是你！”
武小德单手虚抓，竟然隔着数十米的距离，一下子就把妖魔首领拖拽到面前。
“姥流蟒”！
但见武小德一手拧住妖魔首领的脖颈，另一只手反握长刀，双手就这么一错——
妖魔首领的头颅冲天而起，无头尸体缓缓跪在他面前，血喷了好一阵才倒下。
武小德顾不上说话，目光在妖魔丛中一扫，迅速挑了一头畏缩不前的妖魔，直接发动“无限鉴定”。
一人一妖顿时进入“一对一”的状态。
那妖魔却不知道战斗已经开始。
武小德也不提醒它。
——事实上他做这件事，只是为了让自己继续保持在妖魔中不受任何攻击。
密密麻麻的刀枪剑棒，千千万万道术法。
全都无法打中他。
所以——
他伸出手，接住了那颗妖魔首领的头颅，将之插在自己的白旗上，然后高高举起。
“要摘走别人首级的人，必须先保证自己的首级不会掉。”
他淡淡说着。
因为已经试过了这么多次，始终无法打中他。
妖魔们的攻击渐渐停下。
忽然。
一道声音在武小德耳边响起：
“士兵，你前方七百米有一根蓝色的绳子，快去用力扯一下，它会把你传送到群星山脉。”
“——那里更需要你！”
武小德一惊，装作毫不在意的扫过战场。
只见一名穿着全身甲的军官倒在尸丛之中，嘴里满是鲜血，胸口插着一柄骨杖，仿佛已经死去多时。
然而细看之下便会发现，那骨杖是被他夹在腋窝下的。
妈蛋。
这种时候竟然还有装死的。
“你别以为我是装死啊，我是在给远程全覆盖火力报坐标，这里马上要被我们人族的天基重炮犁平——”
对方继续传音道：“快走！”
下一瞬。
一股浓浓的不祥之意浮现在武小德心头。
他猛然抬起头。
只见天穹深处出现了一抹刺眼的光。
见鬼——
自己“一对一”的目标会在一瞬间被那炮轰死。
四周一切敌人都会死。
没有敌人，无法发动“无限鉴定”。
——自己也会死！
武小德使出吃奶的劲，全力朝前方七百米开外的那根蓝色绳子冲去。
来得及吗？
他刚冲到一半，突然发现那光已经迅速迫近。
来不及！
武小德大脑一片空白，突然打了个激灵，记起了某件事。
——还有一线生机！
只见他突然矮身而下，使出一记滑铲。
潮霜秀明铠顿时被激活。
——滑铲时移动速度增加三倍！
只见武小德如一道电射的残影，瞬间便抵达了那根蓝色绳子处。
他攥住绳子用力一扯——
整个世界被白光笼罩。
无数惨叫和嘶吼声在耳边响起，又在一瞬间迅速远去，再也不可听闻。
最后一瞬，自己逃出来了！
咚！
武小德重重摔落在地上，大口喘着气，脑子里不断闪过刚才经历的一幕幕。
——那些堆积如山的兵器甲胄，其实都是战死者留下的吧。
事实上，那处阵地已经守不住了。
最后校准位置的军官，救了自己一命。
武小德喘息着，按捺下情绪，从地上爬起来。
——这里是群星山脉吗？
他抬头望去，但却突然怔住。
前方。
天穹之上，无数星辰环绕。
大地之间，群山围绕的平原之央。
一根巨柱耸立其中，上接天穹，下连大地。
——巴比伦神柱！
难道说……
这里即将发生那件事？
丛山峻岭之上，那绵延于山巅的城墙中，满是人类的将士。
武小德前后左右都是人类士兵。
几名士兵看了看他，友善的点点头，就算打过招呼。
他们也不说话，只是神情紧张的守着城墙，不时朝神柱上望去。
只有一个小队长模样的人类上前来，按住武小德肩头，悄声道：
“神柱上还没有消息传来，所以妖魔不敢对我们发起总攻——”
“现在所有抵达这里的士兵，第一要务是警戒，明白？”
“明白。”
武小德应了一声。
对方满意的点点头，回到了他自己所守的位置。
武小德紧张的咽了咽口水。
——接下来会发生什么事，自己知道啊！
神柱崩塌。
整个盘古世界都会毁灭。
在成为争霸战冠军，得到初始序列的时候，自己就知道了这个秘密。
可是时间之主又要求自己抵达这里，亲眼见证这一幕。
难道他不知道自己知道？
……不对。
他掌握着时间。
他应该知道自己知道！
那么自己抵达这里究竟是为了什么？
武小德满心疑惑，又无法可想，只好守着自己的位置静静等待。
时间一分一秒过去。
崇山峻岭之外。
一个又一个巨大的妖魔出现了。
它们身上的气息如狂风一样吹拂不息，实力恐怖到了极致。
武小德甚至觉得奇诡生命都不一定能有这么强！
“天神在上啊。”
他身边另一名士兵战战兢兢地念叨着。
武小德凝视着那些巨型妖魔看了一阵，心头忽然跳出来一个念头。
“不……不至于……”
他喃喃道。
这些妖魔给自己一种极其熟悉的感觉。
倒不是说自己见过它们。
而是它们身上存在着某种后世也有的力量。
自己无法看见。
但是凭借着观察这些妖魔的举动，自己已经有了猜测。
——它们总是下意识的朝虚空看一眼。
所以时间之主一定要自己来，是因为这里埋藏的秘密就是——
序列。
妖魔是序列的一员。
是这样吗？
武小德默默想着，忽听见远远近近响起连成片的惊呼声。
终于来了么？
他缓缓抬起头，朝神柱上望去。
下一瞬。
一道身影从神柱之顶俯冲而下，看上去无比狼狈。
那身影尚在半空，便已高声吼叫道：
“不好了！‘升序’要来了，我们全都是祭品，是祭品啊！”
“永恒的存在创造我们，只是为了献祭我们！”
这道声音响彻整个世界。
那些巨型妖魔仿佛得到了命令一样，立刻朝着遍布人类的崇山峻岭冲了过来。
下一秒。
半空中的那道人影忽然抬起手，用力按在虚空之中。
整个世界随之停滞。
这一次，武小德终于看清了那人的面孔。
——时间之主！
通风报信的竟然是他！！！
时间之主也察觉到了他，立刻传音道：
“双旗回来了？太好了，我现在就告诉你那个真正的秘密。”
“我从圣界盗来的永恒之力藏在无始无间之中，一切存在都无法寻找到它们，除非得到我的允许。”
“你能抵达这里，一定是在后世得到了我的认可。”
“记住，那句咒语是：”
“众生皆是流沙，而我当永恒！”
话音落下。
他在半空做了个斩击的动作。
武小德顿时被击中，整个人直接被打飞出去，没入虚空，朝着遥不可知的所在落去。
最后一瞬。
他看见那个世界之中，连接天地的神柱轰然崩塌，朝大地徐徐落去。
天空深处仿佛有什么无比巨大的东西，正在朝着世界俯瞰。
时间之主被看了一眼，直接在半空化作了飞溅的血雨。
盘古世界的毁灭徐徐降临！
唰——
时间不停倒退，一切化为空白。
武小德重重的落下去，撞击在地面上，发出沉闷声响。
他呻吟了一声，晃了晃脑袋，勉强从地上爬起来。
只见这里是一处战场。
另一个自己依然站在这里，守着左右两柄旗帜。
——所以自己又回到了原点？
亡灵之书上，金犬出声道：
“友情提示一下，你不仅回到了这个地方，也回到了这个时刻。”
“所以刚才的一切都尚未发生？”武小德问。
“从时间上来讲应该是这样。”金犬道。
武小德叹了口气。
时间之主如此强大，却还是让一切延续到了后世，没有完成对盘古的拯救。
不过他似乎留下了某种希望。
“我从圣界盗来的永恒之力藏在无始无间之中，一切存在都无法寻找到它们，除非得到我的允许。”
这是他的原话。
无始无间……
不就是这两柄旗帜么？
武小德上前一步，一手握住黑旗，一手握住紫旗。
咒语是……
他回忆了一下，轻声念道：
“众生皆是流沙，而我当永恒！”
两柄旗帜忽然拼合在一起，迅速融入彼此之中，化为了一柄旗帜。
这是一柄金色的旗帜。
它看上去就像流动的黄沙一样。
“流沙？”
武小德念叨了一句，忽然浑身一震。
西王母说过的话在他心头如雷霆一般炸响：
“你必须在奇诡赢得这场战争之前，找到时间之主的另一柄旗帜。”
“那柄旗帜叫做‘流沙’。”
