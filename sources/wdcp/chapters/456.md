# 第四百五十六章 英灵加庇！

“哈哈哈哈哈！”
多洛雷斯放声大笑起来。
这一刻，他的心情无比愉悦。
主持考验的英灵亲口说了，那少年是远古人皇的传人。
英灵甚至还为他站台力挺。
现在这些灭世议会的老家伙竟然对那少年下毒手。
简直是自毁长城！
就看接下来英灵怎么说了。
他望向半空中那身躯庞大的英灵——
融。
只见融的四张脸上齐齐流露出忿怒之色。
“你们刚才在杀自己人？”
它问道。
灭世议会的议长恭恭敬敬行了一礼，回应道：
“阁下说过，我作为团队首领，可以在团队中行奖惩之事。”
“那个小子虽然赢了一局，但他背叛了我们团队——”
“所以我要杀他。”
融怪叫一声，身形伏下来，四张面孔轮流转动，恶狠狠地盯着灭世议会的议长。
它以近乎咆哮的声音说道：
“在众多的秘境之中，人皇秘境是最容易接纳你们的地方，然而——”
“你们明明已经拥有了人皇的传人，却想杀死他？”
议长呆住。
什么？
艾利安是远古人皇的传人？
融的声音已经带上了些许杀意：
“你们有没有想过，接下来是万灵降世的时代，等你们杀死人皇的传人之后，你们自己会是什么下场？”
议长忍不住后退几步。
见鬼！
为什么会这样！
难道真的无法除掉艾利安了？
议长朝四周望去。
连个鬼影都看不到，更不要说艾利安的踪迹。
他跑了？
双方已是不死不休的局面……
议长闭了闭眼。
不行，要立刻更改策略！
虽说英灵降世的时代即将来临，但毕竟还未真的来临。
要趁着这个时刻，把艾利安干掉！
不然的话，他的力量只会越来越强，到最后整个灭世议会都会被他夺走！
议长猛然睁开眼睛，开口道：
“伟大的英灵阁下，我作为灭世议会的首领，承认在这场生死战中，我们不是万灵寂静协会的对手——”
“我们认输。”
四周一静。
议长身后的各位长老们却暗暗点头。
这是最明智的决策了！
放弃这场考验，保存所有长老的有生力量，全力去对付议会的真正敌人——
艾利安！
一定要干掉他！！！
毕竟双方已经没有任何缓和的余地了。
只要杀了他，就去除了心头大患。
至于万灵降世的事，听说英灵们不止一个阵营，实在不行，到时候投靠人皇的敌人也行。
杀人皇的传人反倒是最好的投名状！
——短短数息之间，议长就把前前后后的事情都想明白了，还立即做出了决断！
多洛雷斯远远听见了这番话，不由深深的看了议长一眼。
这个老狐狸，恐怕并不是不敢上场跟自己打。
他的一切都要经过利弊上的权衡。
这是个阴毒而冷静的敌人！
“我们确实认输了，请让我们回去吧。”
议长再次朝着融行礼道。
融听了这番话，脸上的表情转为淡漠。
它高高的飞到半空，开口道：
“生死战已经分出胜负。”
“获胜者为万灵寂静协会。”
“接下来——”
“你们要面对真正的考验了。”
话说到这里，融的脸上才露出几分诡谲莫测的笑意。
一名灭世议会的长老忍不住道：
“我们明明已经认输了，为什么还不能离开？”
融摇头道：
“当然不能离开，生死战只是分出不同的考验待遇——”
它故意停住了话头。
众人心头都浮现出一股不妙的预感。
武小德躲在暗中，忽然发现亡灵之书上的几行文字开始散发出光芒。
——正是罪恶任务。
“注意！”
“罪恶任务：‘你死我不死’已经激活。”
“请尽力生存下来吧！”
原来如此。
这任务果然具有预言性，眼下才进入到任务之中。
却不知英灵的真正考验究竟是什么？
武小德屏息凝神，暗暗戒备。
只见融俯瞰着地面上的众人，嗡声说道：
“因为我们的力量有限，只能庇护你们中间的一部分，所以才让你们打一场。”
“现在——”
“万灵寂静协会的成员们，你们获得了我的加庇，在考验中可以一直保持充沛的体力。”
融随意的挥了挥手。
大片大片闪烁的光芒从它手中飞洒出去，落在万灵寂静协会众人身上。
会长多洛雷斯静静感受着这些光芒，忍不住低声道：“原来如此，我感觉到身上充满了用不完的力量……”
他回头望向身后众人，只见大家都是喜形于色。
不管接下来的考验是什么，如果有着用不完的体力，就可以一直保持高水准的战斗水平。
这确实是极大的优势。
反观灭世议会这边，众位长老都是一脸的阴沉。
已经输了。
眼下唯一能做的，就是想办法抗过整个考验，然后回到议会的世界，发起对艾利安的追杀！
只听融继续说道：
“至于灭世议会，也有一位成员在生死战中赢了一场。”
“当然，他无法获得万灵寂静协会所得到的高等级加庇，只可以得到一丢丢安慰奖励。”
融的脸色变得认真而严肃。
它双手合在一起，飞快的结出手印，四张面孔齐齐张口，念着无法被理解的晦涩咒语。
足足念了二十三息。
“啊啊啊啊啊啊啊，以我的全部力量，赐予你这一类别的加庇！”
融怒吼出声。
轰！
炽烈而恢弘的光柱从它身上冲天而起，席卷整个世界——
自然也卷到了武小德身上。
当那无比绚烂的光接触到他的身体，所有光芒顿时全部没入他的体内。
从外界看上去，这一场盛大的加庇之术仿佛在一瞬间偃旗息鼓，消失得干干净净。
异象完全消失。
融漂浮在半空不动，气喘如牛，四张面孔上都是疲惫之色。
“好了，呼，已经给了他一丢丢的加庇，呼，呼。”
它看着地面上的众人，一边擦汗，一边解释道。
众人一片默然。
混蛋。
给万灵寂静协会加庇的时候，你只是随意挥了挥手。
给那小子加庇的时候，你看上去连吃奶的劲都使出来了啊！
——你管这叫一丢丢的加庇？
我们又不是被骗大的！
多洛雷斯脸上似笑非笑，目光直朝灭世议会众人身上瞥，一副幸灾乐祸的模样。
议长的脸色宛如一口黑锅，难看已极。
“听着，诸位，我们一定要团结，争取顺利度过接下来的考验。”
议长低喝道。
“是！”众位长老齐声道。
至于武小德——
他默默站在寂静魔雾之中，心中也有些好奇，忍不住朝亡灵之书望去。
只见书页已经打开，显现出一行行冰晶小字：
“万灵降世，突获加庇怎么办？”
“不要怕，快使用龙族专属骨phone2max，帮你看清一切数据。”
“骨phone2max，就是这么酷！”
“广告观看完毕。”
“本书接收骨phone2max共享数据如下：”
“你获得了远古英灵的寿命增长大祝福加庇之术。”
“恭喜。”
“你获得了三十条命。”
“持续时间：24小时。”
三十条命！
万灵寂静协会只获得了体力，而自己获得的是三十条命！
——能给人增加命，融的实力也太恐怖了吧！
武小德不禁诧异地抬头望去。
只见融的四张面孔中，有一张面孔偷偷朝他望过来，飞快的使了个眼色。
——哥们儿够意思吧？
武小德忍不住抱拳拱拱手，开口道：“多谢了！”
他又忽然意识到对方听不见自己的话。
毕竟对方能掌握自己的位置，是凭借席卷整个世界的大祝福术，才发现了自己。
武小德索性放出数百个青铜手，在半空拼出了两个字母：
“nb！”
——这两个字母足以表达自己的情绪和感谢。
青铜手拼完便瞬间消失。
融自然是看到了。
它笑了笑，大声道：
“好了，真正的考验现在开始！”
