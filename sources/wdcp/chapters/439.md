# 第四百三十九章 还治己身！

迷雾缭绕。
武小德在雾中前行。
他穿过空旷的厂房，径直来到烟囱前。
劫魔站在烟囱顶上一动不动。
念兽们在彼此厮杀。
没有人敢靠近——
那些强者看到如此数量的念兽，早就逃命去了。
武小德站在那里，闭上眼，叹了口气。
“何故叹气？”
小骨的声音从亡灵之书中传来。
它举着骨phone2 MAX，正在拍摄半空中的那些灵系念兽。
武小德道：
“我自从觉醒魂力以来，每次战斗无不是讲究公平和公正，以堂堂正正的风格去打，力求让敌人被我打得心服口服。”
“这一次完全打不过，恐怕要用点上不得台面的招式了。”
小骨默默低下了头。
——你？
堂堂正正？
你竟然是这样看自己的啊！
那么，连你都觉得上不得台面的招式，又无耻到了什么程度？
“有信心赢吗？”
小骨低声问。
武小德也低声道：“不好说，因为不知道这家伙还有什么手段，总之，我要出手了。”
小骨就不再说话，瞪大了一双黑洞洞的眼眶，想要看看他究竟怎么战斗。
只见武小德脚尖点地，整个人飞腾而起。
迷雾跟随在他左右，遮蔽了他的身形，令那些怪物无法看见他的行藏。
锵！
长剑出手。
缭绕着黑暗烈焰的剑锋瞬间没入一只念兽的背脊。
霎时间。
念兽不见了。
——死亡烈焰之兵！
“法则技。”
“说明：召唤死亡烈焰依附在你的兵器上，被斩中的目标将被直接送至死亡世界。”
那头念兽直接被传送至了死亡世界！
武小德抬头朝劫魔望去，恰巧劫魔也发现了这里的动静。
它脸上露出了然之色，笑道：
“藏在暗中的虫子终于忍不住了？”
武小德神色平静，开口道：“滚出这个世界。”
名号技发动！
“圣刺猬。”
“名号技：群灵呵斥。”
“描述：你可以直接指定一位降临人间的灵，使它被强制遣返回它的世界。”
“当你对灵之外的目标施展，可以让其强制离开当前世界，随机抵达其他任意世界。”
劫魔似有所觉，脸色一变，开口道：
“该死，等——”
一句话没说完，它直接消失了。
趁着这时，武小德双脚在烟囱上用力一蹬，整个人飞上高空，长吟道：
“忆昔午桥桥上饮，坐中多是豪英。”
“长沟流月去无声——”
魂力界面上，八百万魂力一跳，顿时被消耗殆尽。
一道横贯天地的剑影骤然而生。
此剑影如同日月当空，又有无穷黑暗火焰缭绕，一时威势无两。
半空中。
那些正在厮杀的念兽纷纷被剑锋斩中，无一例外，全部被送至了死亡世界。
整个过程说起来慢，但从武小德出第一剑，再到送走劫魔，又出一剑送走所有念兽，一共加起来，也不过一次呼吸的时间。
这还没完——
亡灵之书上，魂力再次消耗了5万点。
暗之道力·见至！
霎时间，武小德也从原地消失，径直出现在这个世界对应的死亡世界之中。
按照他原本的计划，是想在死亡世界中与念兽们打一场。
毕竟这些念兽已经成为了灵。
灵属于死亡世界，它们来了之后，想要回到阳世，必须要达到1000级！
眼下它们并不满足这个条件。
所以最好就在死亡世界干掉它们！
武小德刚要出手，却忽然发现事情有些超出了自己的想象。
——这里是死亡之海。
在无边的死亡之海上，无穷无尽的人头齐齐朝念兽们冲了上去。
武小德放眼望去，完全没有看到人头的穷尽之处。
仿佛整个海洋里都藏着人头。
那些念兽虽然强大，但在这种量级的怨灵面前，就像掉进食人蚁窝的几块血肉。
武小德只看见无尽的死亡之海泛起了一朵朵涟漪——
念兽们就不见了。
只剩下一阵阵波涛浪涌。
“你们是……”
武小德疑惑道。
一个人头开口道：“我们是无穷世界的众生，被那劫魔杀了之后，灵魂永世在死亡之海中受苦。”
另一个人头道：“原本报仇无望，毕竟生死相隔，没人知晓我们的事。”
第三个人头道：“谁知今天有个活人来了。”
人头们陷入了寂静，一起看着他。
亡灵之书打开，愿墙上浮现出一行行冰晶小字：
“你的暗之道力‘多重’已被激活。”
“凭借‘愿墙’作为介质，你可以将多重亡灵、英灵、神灵的力量凝聚在一起，化为你的专属暗系道力为你所用。”
“怨灵是暗之道力进阶的根本力量。”
“你是否愿意接受怨灵们的委托？”
“接受。”武小德道。
一行行冰晶小字再次浮现：
“战胜劫魔：人类狩猎者。”
“如果你赢了它，那么被它杀死的无尽怨灵愿意把自己的力量赐予你。”
“你的暗之道力将立刻完成双层进化！”
意外收获。
武小德看着海面上漂浮的无尽怨灵，开口道：
“你们等我的消息。”
“你觉得自己能赢过它？”一位怨灵问。
“堂堂正正的打也许有点问题，毕竟我从觉醒那一天开始，仔细算来，才不过5天时间。”武小德道。
众怨灵一阵默然。
“不过我会想想办法。”
话音落下，武小德张目朝虚空望去。
虚空一闪。
他离开了死亡之海，再次回到了现实世界。
依然是那个烟囱前。
时间——
刚刚过去一息。
只见天空中一道流光闪过，劫魔再次出现。
它脸色阴沉地望向四周。
——自己好不容易搞出来的念兽，全都死光了。
那小子竟然有这么大的能耐？
如果他真的能一瞬间杀死所有念兽，刚才为何不出手？
“出来，跟我打一场，我们之间要分个胜负。”
劫魔道。
武小德悄然现身。
“你为什么一定要毁灭人类？放下屠刀立地成佛不好么？”他问。
劫魔隔空打出一拳。
武小德脸色一变，瞬间侧身让过。
唰——
猛烈的拳风贴着他的脸刮过去，在大地上打出一个巨大的深坑。
武小德伸手摸了摸脸颊。
一手的血。
“原来你很能打，是我一开始小看你了。”
武小德鼓掌道。
“受死。”劫魔喝道。
却见武小德身形一闪，再次消失在虚空之中。
劫魔的拳头刚刚再次提起来，忽觉眼前一花——
不知道怎么回事，自己好像看到虚空中站着几个穿白衣服的人。
一晃就不见了。
——这是什么情况？
劫魔暗自疑惑，再一转头，却发现武小德也不见了。
“滚出来！”
它怒喝道。
出来？
武小德听了，不禁眯起眼。
不。
你的实力太强了。
为了大地上的爱与正义，为了众生能够活下去——
我不出来。
他身形连闪，不断变化方位，悄然落在数十米之外的一处厂房上。
早有一行行冰晶小字浮现在他眼前：
“对方依然处于‘灵异’状态。”
“当前状态为：见鬼。”
“即将更换。”
“此外，你发动了双天使咒印。”
“对方已经击伤了你。”
“咒印激活。”
“从现在开始，对方的一切治愈之术将只对你生效。”
“此外，你可以从对方身上挑选一项能力，复制到自己身上作为医疗费和损失赔偿。”
“请选择！”
密密麻麻的技能浮现在书页上。
——这下可以了！
利用审判天使咒印，一下子就得到了对方所有的技能情报！
如果能在这里干掉它……
那当然最好不过。
武小德目光不断游寻，终于选中了一个技能。
种族邪化·谎言之躯！
——被这一招命中的存在不可以说谎，否则会变成爬虫！
“发动吧。”
他看着对面半空中暗暗警惕的劫魔，轻声说道。
