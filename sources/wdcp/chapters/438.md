# 第四百三十八章 上场！

夏蕙兰蹲在地上，用小刀从墙角刮下一块单薄的甲壳。
“这种东西也值得收集？”
武小德问。
“人类异变成念兽这种事，还是很值得研究的。”夏蕙兰道。
甲壳被她装进一个密封袋。
她重新给手枪上了子弹，回到武小德身边。
“走吧，该去跟那个家伙战斗了。”
“不用。”武小德道。
“不用？”
“对，刚才那怪物问我是不是这个世界的强者，现在么，强者们应该已经来了。”
武小德抬起头，望向亡灵之书。
圣行之墙一直开着。
刚才发生的一切，全都面向这个世界播放了出去。
接下来会发生什么？
黑色的天空深处忽然冒出一道道流光。
强大的力量波动在虚空中传递。
一个接一个人类强者出现。
武小德望着天空，轻声说道：
“这个时代有它自己的守护者们，而我们不能越庖代俎——”
“我们存在的意义是刺探情报，直到关键时刻，再出手狠狠的帮他们一把。”
夏蕙兰听完，微微颔首道：“有道理。”
轰——
工厂里传来一阵阵剧烈震动。
有人在天空中高声道：
“怪物，滚出来！”
工厂中响起一道无比怪异的声音：
“来不及了，这个世界的人类们啊，我已经完成了最原始的力量积累。”
灰暗的光芒冲上高空，呈环形朝四面八方投射而去。
那怪异的声音高高扬起：
“来吧，所有的人类，你们现在所说的谎言，将成为你们的念躯，它会让你无比强大！”
几乎是声音冒出来的瞬间，亡灵之书上也显现出一行行小字：
“对方为世界法则添加了一种专属终末：”
“种族邪化·谎言之躯！”
“描述：只需要说出一句谎言，普通人类可以将自己激活为爬虫体。”
“越有潜力的人类越可能转化成为念兽类爬虫。”
“最强的人可以直接化为灵体类爬虫。”
“所有人类都将转化！”
“——这是整个人类种族的邪化！”
武小德不禁浑身一震。
终末！
想不到这个劫魔居然会用终末！
而且自己从未听说过有这种针对一切众生的终末，只凭借说谎就可以实现邪化。
这是何等不可思议的终末！
“兰姐，夏鲵，现在不能说谎了，一说谎就会完蛋。”
他赶紧提醒道。
“嗯，小武，你喜欢我吗？”夏蕙兰忽然问。
“喜欢。”武小德毫不犹豫地说。
夏蕙兰端详着他，不动。
一息。
两息。
三息。
武小德四周没有任何异样。
“都什么时候了还在说这些！”夏蕙兰白他一眼，转过头去。
武小德一阵无语。
——这是你问的啊！
他抬头望向天空，只见那些半空中的人类强者们纷纷骚动起来。
这种时候，只要有一位强者试试说一句谎话——
试不得啊！！！
武小德正想着，忽见一道身影出现在工厂那高高的烟囱上。
正是那个劫魔。
它仿佛知道武小德在附近观察，回头朝街道方向露出微笑。
“我猜……你大概还躲在这里，并且认为人类不会做那么愚蠢的尝试。”
“但你忽略了一件事。”
“人类本就靠着谎言才可以维系社会文明。”
“好奇是你们的本性，是一切恶的导火索，而谎言就是那颗炸弹，它代表了人类的虚伪和恶念。”
“谎言将毁灭你们。”
世界化作无声。
一切仿佛不断变缓。
武小德看见天空中，有人张开嘴，尝试着说出一句什么。
不要啊！
他自己身怀双天使的终末，自然知道在终末面前，一句话说不好，就会立刻激发出无比恐怖的力量。
但是来不及了。
那位强者已经说完了一句话。
啪。
他的身体瞬间炸裂开来。
他死了。
但是一道狰狞的身影逐渐浮现，开始在半空中爆发出低沉的咆哮声。
爬虫。
——他被转化为了爬虫！
看上去，这头爬虫比那第一头爬虫不知道强了多少倍。
它的身躯已经不太像蛇，而是像一种扭曲的蜈蚣，但身周又比蜈蚣多了许多锋利的尖锐长刺。
趁着其他强者愣神的功夫，这头爬虫闪电般动了一下。
在它旁边的另一名人类强者直接被它撕裂了身躯。
所有人慌忙散开。
“注意！不要说谎，会出大问题的！”
有人喊道。
武小德突然意识到了什么，头脑中一片空白。
不……
已经来不及了。
任何人，只要说谎，立刻就会化为念兽。
可是正如这个怪物所说——
在这个辽阔的世界中，在任何人类的聚居地——
人类是无法停止说谎的，一百个人里，只要有一个人尝试——
他猛然转头望向劫魔。
只见那劫魔站在烟囱上，张开双臂，脸上露出微笑。
天空中，不少强者开始对付那头念兽。
更多的强者朝着劫魔出手。
一道道流光飞来，眼看就要打在它身上——
四面八方的虚空里却响起了一阵密集的“嗡嗡”声。
霎时间。
数不清的念兽同时出现。
它们以自己的身躯挡住了所有攻击。
从天到地，念兽的数量不可计数，排山倒海一般铺陈开来。
人类强者一哄而散。
——这完全没办法打！
劫魔也不在意，更不追赶，只是冲着那数不尽的念兽道：
“开始吃吧，灵类的念兽，你们可以吃光其他任何念兽，只要你们能吃得下。”
灵类的念兽本就是最强，这时得了命令，顿时陷入了疯狂的厮杀。
其他念兽一动也不敢动，任由它们杀戮和吞噬。
灵类念兽越吃越强。
武小德忽然叹了口气，摇头道：
“太失败了，本以为会看到一场精彩的战斗，谁知道是单方面碾压。”
夏蕙兰道：“这样的话，我们就得不到什么有用的情报了。”
“除非……”武小德没说下去。
——除非奥德薇娅借我魂力，让我跟它打一场。
可惜奥德薇娅消耗太过，正在休息。
怎么办？
没有充裕的魂力，自己就无法上场，只能看到这个程度了。
他正想着，忽见亡灵之书仿佛感受到了他的意志，自动翻开，显现至他的个人属性那一页。
“储备魂力：37850093；”
“储备愿力：135913；”
“储备功德：193。”
“注意：以上数值还在不断增长。”
武小德忍不住失声道：“怎么会有这么多！”
一行冰晶小字浮现于书页上：
“你的魅魔克里斯提娜通过‘暗影随从’使用了你的‘圣行之墙’，正在赚钱养你。”
赚钱养你。
养你……
武小德眉头跳了跳。
怎么说呢？
诚然，自己有一点窘迫、尴尬和忐忑，但是看到那个千万级别的数字，内心深处又升起了一股奇妙的感觉。
那一道穿着“求实、创新、开拓、进取”制服的少女身影再次浮现在他心头。
夏蕙兰似有所觉，问道：“怎么？”
武小德下意识的要说“没事”，但立刻意识到这是在说谎。
——说谎可是会有大麻烦的！
“之前觉得没法打，但现在我应该可以打一场试试。”
武小德道。
“你的等级不够跟它打。”夏蕙兰道。
“我有一种储备魂力的能力……可以一试！”
武小德道。
大约四千万魂力——
确实可以一战！
“小心。”夏蕙兰道。
她悄悄给了夏鲵一个眼色。
万一不行，夏鲵就算自己死，也要把他救出来。
夏鲵微不可察的点头。
武小德没注意到这一幕。
他双目一垂，低声道：“降临。”
轰！
他浑身冒出黑暗火焰，连同荒剑的剑鞘上也缭绕着一团团黑火。
——黑暗魔王降临！
书页上跳出一行小字：
“消耗100万魂力。”
一次降临就花费了100万魂力。
接下来还有道力和终末要用。
武小德以手按剑，一步一步朝着劫魔走去。
战斗要开始了！
