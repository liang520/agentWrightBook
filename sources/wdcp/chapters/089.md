# 第八十九章 看上我了

皇帝挥手道：“你们也累了，去找个馆子吃早饭吧，傍晚再来见朕。”
“是！”
三人齐声应道。
出了皇宫。
赵芷冰跟两人招呼一声，说去平行世界的皇宫要做许多准备，便匆匆走了。
剩下赵君羽和武小德。
“走，去吃早餐？”赵君羽问。
“等等，夏蕙兰怎么样了？如果还在睡，我就不打她电话。”武小德道。
“她昨晚送到皇家医院，检查了都还好，这才过去一夜，可能还要多休息一下——怎么了？”赵君羽问。
武小德也醒过神来。
是啊。
才过去了一夜。
“……她没醒吗？”
“早上醒了，吃了点东西，又睡着了。”
“听上去还好。”
武小德手机忽然响了。
“喂，老钱？”武小德道。
“小武哥，我在异世界得了一本武功秘籍，你回来吃早饭啊，顺便帮我看看这秘笈，行不行？”钱明魁的声音从话筒里传来。
“我饿了，在城里吃完就去你那边。”武小德道。
话音刚落，手机听筒里传来一声巨响。
钱明魁的声音顿时断了。
武小德脸色一变。
他忽然想起一件事。
当时那个脱狱者说过，一旦有人觉醒力量，世界就会激活某种机制，以图抹杀觉醒之人。
难道钱明魁……
“很重要的朋友？”赵君羽问道。
“是啊，我们得马上去看看情况。”武小德焦急道。
“哪里？我们现在就看。”
赵君羽拿出手机，低声道：“调卫星画面，马上开始传输影像。”
“踏雪寻梅武馆，首都郊区。”武小德道。
“锁定位置，立刻展示生命体状态。”赵君羽道。
一幅全息图像出现在两人面前。
从高空朝下俯瞰，视角逐渐拉近，可以看到一个大胖子带着小女孩正在外面的操场上养鸡。
房子里面似乎发生了爆炸。
大胖子跟小女孩都吓了一跳，大胖子的手机掉在了地上。
“没事，是手机摔坏了。”赵君羽道。
“那爆炸呢？”武小德问。
“看房屋的情况，初步分析是煤气爆炸。”赵君羽道。
武小德松了口气。
“见鬼！我就知道这胖子有祖宗庇护——”
他正要说下去，忽然想起了什么。
祖宗庇护……
而且觉醒了魂力……
老钱，你这是祖上全家发配罪狱啊！
话说回来，你们家祖上到底是干什么勾当的？
咕噜。
武小德的肚子开始抗议。
“走，别管他们，先吃饭，吃了我们再去武馆。”
“好。”
几分钟后。
两人站在一处街角的大排档前，顺着人群排队。
“这家面特别不错，牛肉、牛杂、卤蛋、黄酒和米酒也很有味道。”赵君羽夸赞道。
“说的我都饿了。”武小德揉着肚子道。
滴滴滴——
两人手机同时响了。
拿出来一看，却是赵芷冰分享了一条新闻。
今日头版头条。
一名穿着朱红色镶绣金纹长袍的皇室发言人站在画面正中。
下面是一行大字：
“皇室注资‘武德充沛’电影公司，为大型科幻电影追加投资。”
“本片的拍摄将持续下去，预计拍摄周期为一年。”
武小德一看就懂。
自己搞了两场直播，后面如果要面对各种新问题，自己还得出动。
——S级俱乐部剩下的三人都要出动。
为了掩人耳目，最好的办法就是“拍片子”。
这可以让中央智脑把异常事件归于“电影特效”一类的事情，不会被激活，更不会疯狂的学习和分析。
这样，皇帝的大脑就好受多了。
——完美。
两人饥肠辘辘的排着队，不时翻翻手机。
几分钟后。
终于轮到了他们。
赵君羽要了一碗臊子面，加卤蛋，米酒。
武小德则要了一碗牛肉面，双份牛肉，两个卤蛋，黄酒，冰镇可乐。
两人端了面，也不去排挡里面挤，就在外面搬了个小板凳，将面放在大板凳上吃了起来。
早上的风还是很凉快的。
面馆里人进人出，那带孩子上学的少妇，骑自行车的大爷，嘻嘻哈哈的中学生，背着钓鱼竿的中年大叔，穿着练功服的大妈，纷纷埋头吃面，见了熟悉的人少不了打个招呼，热络的说着家长里短。
武小德一边吃着卤蛋，一边喝着冰镇可乐，看着络绎不绝的人群，听着那些平淡无奇的言语，心中忽然就有一种满足感。
“谁说这里是罪狱？我倒觉得这样才好。”他自言自语道。
“兄弟，这些都是凡人的小快乐，也许你们这些觉醒者以前都是神仙呢！”赵君羽凑过来说道。
神仙……
武小德想起了被杀死的那个少年。
他曾经是一个星际的帝国之王。
他失败了，被关在这里，成为一个什么也不知道的少年，然后被仇家找上门，毫无反抗之力的杀死。
他所守护的帝国也随之成为别人的傀儡。
“是啊，”武小德感慨道，“谁知道呢。”
“喂，你看对面第三桌，那个穿红衣服的女生。”赵君羽小声道。
武小德目光随意一扫。
那女生长相甜美，但娇弱无力，一看就是普通人，没什么威胁。
他放松下来，继续吸着面条，含糊不清的问：
“看见了，怎么？”
“她在看我们。”赵君羽道。
武小德翻了个白眼，说道：“看你一眼你又不会掉块肉。”
“不是啊，我觉得她可能对我们有兴趣，可能看上我们中间的一个了。”赵君羽压低声音道。
说话间，那女孩果然起身朝两人走来。
她礼貌的打了个招呼，找两人要联系方式。
武小德以不方便为由拒绝了。
赵君羽风度翩翩的加了对方为好友。
女孩又礼貌的说了再见，笑嘻嘻的走了。
“喂，大庭广众之下，你要给女孩留面子，懂吗？”赵君羽语重心长的道。
武小德耸耸肩道：“你可能不像我们这样天天在街上混，有些事儿见多了，一看就闻得出味儿来。”
“你是说，她心怀不轨？”
“那倒不至于。”
“就是说嘛，我觉得她可能对我们有好感。”
“一次对两个人有好感，你猜会是什么事儿？嗯？”
“这个嘛……”
赵君羽有些卡壳。
正好他的手机震了一下。
拿起来一看，正是那个女孩。
她发来了一条消息：
“帅哥，我们城东健身馆有泳池和拳击项目，要不要了解一下？”
赵君羽默默放下手机。
“别看到个女孩就觉得人家喜欢你，好吗？做人呢，实诚一点，别老觉得自己酷炫狂拽。”武小德语重心长的道。
赵君羽低着头默默吃面。
两人慢慢吃完了早饭。
赵君羽拍拍武小德的肩膀道：
“你请客。”
“为啥？”武小德问。
“你签了合约，吃饭不要钱。”赵君羽道。
“真有这好事？”
武小德拿出手机，扫码支付。
滴！
手机上跳出一行小字：
“朕买此单，钦此。”
武小德怔了怔，喃喃道：“当今陛下仁爱亲民，实在是佩服佩服！”
皇宫里。
皇帝得意的歪嘴一笑，翘着二郎腿，端起茶杯慢慢喝了起来。
在他对面，赵芷冰已经回到大殿里来。
她正拿着手机，把这次的补给消耗单传给皇帝。
“你这次消耗的符箓不算多嘛……朕出钱给你补齐。”皇帝看着单子道。
“多谢父皇。”赵芷冰淡淡的道。
“咦？为什么‘一次性肌肤隔绝符’也用了一张？”皇帝突然问。
“当时情况危急，武小德为了救我，牵我的手拉了我一把，让我躲开了一次攻击。”赵芷冰面无表情的道。
“这小子，他肯定不知道你身上贴了十二张替身符……哈哈，看在他如此回护你的份上，朕也不跟他计较这么多。”皇帝道。
说到一半，他似乎又想起什么，追问道：“这符只能用一分钟，话说你们牵了几次手？”
“就一次——他救我的那次。”赵芷冰平视前方道。
“哦，那就好，今晚再过去的话，准备带多少符箓？”皇帝放下心来，问道。
“平日没什么危险，所以只带三百张，现在遇到的情况越来越复杂，越来越危险，今晚准备五千张符，差不多可以保命。”赵芷冰道。
皇帝脸色一苦，软声道：“冰冰啊，你看你这次去也没用多少，一百张都没用到，怎么就一下子要带五千张了？”
“父皇是想让我死在那个世界吗？今晚只会更危险，五千张是保命的最低限度。”赵芷冰冷冷的道。
“好了，朕给你准备，你去忙吧。”皇帝无奈道。
“谢父皇，父皇最好了。”赵芷冰微微一笑，拱手而去。
大殿内。
只剩下皇帝一人。
几名臣子缓缓进殿，恭敬的立在一旁。
“公主殿下也长大了啊。”一名大臣感慨道。
皇帝叹口气道：“朕这个女儿，非常爱惜性命，也是好事，嗯，好事。”
他挥了挥手。
一个全息的地下隧道结构图在半空缓缓旋转。
“好了，现在我们要开始朝地下挖了，这是刚刚得到的隧道图，各位，说一下你们的意见。”
几位大臣一起看着那结构图，神情凝重。
