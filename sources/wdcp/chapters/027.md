# 第二十七章 神话成真

夏鲵站在通道口，微微闭着双眼。
“这是她第一次参与实战，她正在进行学习，接下来会升级和优化战斗策略。”夏蕙兰解释道。
武小德点了点头，望向漂浮在半空的亡灵男子。
“还要守七分钟，通道里的人才可以全部撤离完毕。”亡灵男子道。
“刚才那个怪物是怎么回事？他明明是人，为什么长着那么多触须？还敢调戏女孩子？”武小德又问。
“他本是人类。”亡灵男子道。
“人类？”武小德奇道。
“是的，他是一个背叛者——他背叛了人类，被改造成了那个样子，专门用来寻找和屠杀我们。”亡灵男子愤恨道。
“谁改造了他？”武小德问。
亡灵男子摇摇头，没说。
这时又有一个亡灵冒出来，示意又有怪物来了。
“下面就是第五号秘密通道？”武小德低头朝悬崖下望去。
“是的。”
“其实你找我找对了。”
“怎么说？”
武小德在通道口坐下来，说道：“我是第一次进入这个异空间，身上的波动明显，怪物都会来找我——既然它们的注意力在我身上，就不会太过注意其他。”
除了愿墙之外，自己已经打开了第二堵墙。
要不要把这堵墙的功能确定下来？
……太仓促了。
自己完全没想好怎么用这堵墙。
从长远来看，这种时候胡乱设置一个辅助能力，其实是不对的。
武小德打消了这个念头。
亡灵沉默片刻，开口道：“还有六分钟。”
夏蕙兰走过来，轻轻摸了摸武小德的头，问道：“小武，你没受伤吧？”
“没有，我大约还要杀三头怪物，我们就可以回去了。”武小德道。
“三头？等等，你的魂力上限达到六点了？你竟然突破的这么快！”夏蕙兰吃惊道。
“别人很慢吗？”武小德问。
“根据统计数据，一般都需要至少好几次进入，才可以提升一点魂力。”夏蕙兰道。
我就说不对！
武小德露出释然之色。
能这么快提升魂力，其实是因为乌鸦之灵在这里帮自己的缘故。
如果大家杀几个就汲取魂力，提升上限，那岂不是要一直在这里杀下去？
——男人不能比快。
自己还带着兰姐，刚才她就很危险！
等下次自己一个人来的时候，再考虑全力升级的事。
武小德若有所思的想了一会儿，朝着肩膀上的鸦发出了心灵感应：
“接下来的战斗，不要再给我超量魂力了，补满消耗就行。”
“你怕继续扩大魂力上限，以至于要不断的杀怪物，然后回不去？”鸦道。
“是的。”武小德道。
“很好，你懂得忍住提升的欲望，这是对的，毕竟现在的情况非常诡谲危险，我已经闻到了死亡的味道。”鸦凝重的道。
死亡的味道……
武小德想了数息，冲着夏鲵招招手。
“怎么了，小武？”夏鲵微笑道。
“打硬仗一点意思都没有，趁着我是第一次进入，我们来玩点花招如何？”武小德道。
“你说怎么玩。”夏鲵道。
“我呢，有一种感应的能力，可以知道敌人从哪里来——”
他看了半空的亡灵男子一眼。
亡灵男子点点头。
这件事他们还是能保证的。
“你提前埋伏起来，一旦怪物出现，肯定被我身上的波动吸引，你就从背后偷袭。”武小德道。
“好主意。”夏鲵道。
亡灵男子忽然神情动了动。
只见另一名亡灵穿墙而来，对着他小声说了些什么。
亡灵男子立刻道：“23号通道——你对面墙壁上左数第五排第六个通道，即将出现一条双头蝮蛇。”
武小德伸出手，指着那个通道：“夏鲵，你去对面那个有青藤的洞口附近找个埋伏点。”
“好的。”夏鲵道。
她飞过去，落在23号通道上方，紧紧贴着墙壁。
不一会儿，她不见了。
“兰姐，你躲到通道后面，等我们这边杀完再出来。”
“好。”
一切安排妥当。
等待。
一，二，三，四，五。
23号通道里钻出来了一个怪物。
它长着两颗女人的头颅，却共同着一具类似于蝎子的身躯，双手如钳，长尾末端是一根毒针。
怪物出现的一瞬间就看到了武小德。
“这股波动……你是支援者？”
怪物的一颗女人头颅道。
武小德远远坐在对面的通道口，冲它招手道：“没错，是我——我是第一次来，请问你们这里是什么世界？”
怪物狞笑道：“多么青涩稚嫩啊，正好用来做献祭，很快你就会知道这是什么世界了。”
一抹轻柔的凉风拂过。
怪物忽然觉得世界开始不停旋转。
它猛然回过神，把感官放在另一颗脑袋上。
不料这颗脑袋所感受到的只有极其强烈的痛楚——
夏鲵从洞口上方悄然落下来，手中炽烈的光形长刃斩掉了怪物的一颗头颅，紧接着立刻开始削砍它的身体。
就像切香肠一样——
武小德化作乌鸦飞过来的时候，怪物已经只剩下小半个身躯了。
夏鲵抽出几根沉重的铁矛，如同钉钉子一样把怪物钉在地上。
“最后一下留给你。”
她微笑道。
“多谢了。”
武小德一拳轰塌了地面。
怪物的尸体混着山石，不断朝下滚落。
乌鸦之灵赶紧追了上去。
“这些怪物似乎并不强啊……”武小德念叨着。
“他们还残留着人类的一些特征，证明只是刚刚开始转化的堕落者，是敌人中最低等的存在。”亡灵男子努力解释道。
不强？
你有这么厉害的帮手，又是设饵，又是埋伏，然后自己上来打最后一拳。
你还有脸说人家不强？
换个其他新人来试试，分分钟都被吞了好吧！
“也就是说，它刚获得某种力量，跟我一样是新人？”武小德抱着双臂道。
“可以这么说。”亡灵男子道。
“有没有厉害点的——算了，我第一次来，不想出什么意外。”武小德道。
击杀怪物4个。
还剩2个。
武小德朝亡灵之书望去，只见上面浮现出一行冰晶小字：
“你杀死了一个双魂怪物。”
乌鸦之灵从悬崖下飞回来，落在他肩膀上，欣喜道：“好多魂力！等回去了我分给你。”
双魂怪物！
也许是两个人融合成了一个怪物？
武小德朝亡灵男子看了一眼，疑惑道：“明明是人类，为什么会变成怪物？”
这一次亡灵男子回答了。
“你小时候读过神话或童话书么？”他问。
“当然。”武小德道。
“如果有一天，神话里的事情被证明是真的，你觉得人类文明该走向何方？”亡灵男子道。
“神话……那岂不是人人都要成仙做神？”武小德道。
“你想多了，随便一个魔怪都要吃百万计的人类，甚至开始发展信徒，让人类转化为魔种。”亡灵男子道。
“那神呢？神灵不管吗？”武小德摊手道。
“神？哪有什么神，魔就是神，它们吃人为乐！”亡灵男子道。
武小德心中一颤。
如果真是这样，这个世界也太恐怖了。
“刚才我杀掉的是魔化的人？也就是说，她们是——”
亡灵男子打断他道：“还是别想太多的好，魔化之后，彻底不再属于人类这个物种，她们已经是魔种生命了，很快就会助纣为虐。”
几名亡灵从墙里钻出来，飞快的说着什么。
亡灵男子认真听完，朝武小德道：“邪术的使用者刚刚进入了地下掩体，正在朝着你的方向摸过来。”
“几个人？”武小德问。
“两个。”
“是不是有一个戴着灰犀牛面具？”
“对，还有一个手上缭绕着惨绿色的火焰。”
“惨绿色的火焰……”
武小德立刻想起刚进入这个世界时，那个所谓的鬼火团队。
那些招揽自己的文字，都是用惨绿色火焰书写的。
——怎么看都是同一个人啊。
猜魔请这样一个人做帮手，就敢来杀自己。
这个人有什么本事？
武小德闭上眼睛，静静思索起来。
夏蕙兰说过，这个世界很大，很多地方都是未知的，职业者们进入是随机传送的模式。
那么。
他们凭什么能这么快找到自己？
要知道，邪术师猜魔想杀自己，但在首都的时候，他根本找不到自己，也追不上自己！
那个绿色火焰……
火焰构成的小字能直接出现在自己面前。
他们又非常迅速的找到了自己的所在地，并且赶了过来。
也就是说——
猜魔的帮手，一定有着某种穿过空间找目标的能力，以及空间传送能力！
当时萧白红招募自己加入S级俱乐部，画面上形形色色的人之中，除了影子、戴灰犀牛面具的邪术师，剩下的还有什么人。
……想起来了。
好像有一个家伙，始终手捧着一根蜡烛，蜡烛的火焰非常微弱，有些看不清楚颜色。
也许就是这个人？
一个能控制尸体的邪术师，再加上一个能空间移动和寻找目标的陌生能力者。
他们要杀过来了。
——两个都是10点魂力以上的高手，很难办啊！
武小德扭头问道：“兰姐，我这里可能会有点问题，你有什么选项吗？”
“问题大吗？”夏蕙兰和夏鲵齐声问。
“估计挺大的。”武小德道。
“用核武怎么样？”两女齐声问。
武小德无语道：“核武的话，我们也会完蛋吧……”
“用微型的，以定时的方式控制，设置好我们就飞走。”夏蕙兰道。
武小德看了亡灵一眼。
那帮亡灵拼命摇头。
“——辐射太大，这四周还藏的有人呢。”武小德摊手道。
“那就温压弹吧，核武之下最恐怖，适合这个满是地下通道的地方。”夏蕙兰拍板道。
武小德微微点头。
只要第五号秘密通道里的人全都走了，丢一个温压弹也是可以的。
“喂，你也干点活儿吧。”武小德朝亡灵道。
“需要我做什么？”亡灵男子问。
“你带着手下去各个通道看看，如果他们来了，提前打个招呼。”
“可是那很危险。”
“你已经死了，还怕危险？”
“也是，那我去了。”
