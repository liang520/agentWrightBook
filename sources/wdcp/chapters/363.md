# 第三百六十三章 入道！

“注意：”
“你已入道。”
“你成为人族中特殊的存在：修道者。”
“由于你本身具备死亡之力，当前觉醒逆五行之力：暗。”
“暗的根本是死与终结。”
“你具备这样的才能。”
“开始觉醒。”
“死亡归于暗之力，因此你觉醒的根本能力为‘多重’与‘见至’。”
“‘多重’。”
“凭借‘愿墙’作为介质，你可以将多重亡灵、英灵、神灵的力量凝聚在一起，化为你的专属暗系道力为你所用。”
“‘见至’。”
“你的双眼可以洞穿一切世界屏障，带你前往并看见一切幽冥所属。”
所有小字一闪而去。
武小德浑身蛟龙血发出“滋滋”声响，迅速缩短、消弭不见。
他猛然睁开眼睛，朝四周望去。
——他的眼神空洞而幽深，仿佛并没有看到当前暗室中的一切。
黑暗在他身周缭绕，几乎要显现出另一个世界。
赵从南嘴角微微绷紧，轻轻朝后一退，跃出暗室，朝马三和孙钱打了个手势。
“将军？”两人迟疑道。
“道的力量是无穷的，已在他身上觉醒了——不要挡路，我们让一让。”赵从南低声道。
两人浑身寒毛都竖了起来，立刻跟着赵从南一起退到了墙角。
一息。
两息。
武小德从暗室中走出来。
他站定，忽然转头朝三人望去。
“将军，我是暗。”他开口道。
“金木水火土常有，风雷光暗音不常有，在下惭愧，对暗一无所知，还得靠你自己摸索。”赵从南道。
武小德抬起手，轻轻扶住长剑，轻声道：
“不碍事，我现在很清楚怎么用这样的能力。”
“很好，想做什么就去吧——你们的事，是我们这些凡人不能理解的，不过我希望你记住一句话。”赵从南道。
武小德看着他。
赵从南紧紧咬了咬牙，目光中有什么在燃烧，平静说道：
“人皇纪三七六一年，边城为妖魔所破，十万百姓尸横遍野。”
武小德点了点头，走出烧饼铺子。
世界在他眼前变得孑然不同。
街道上，伏尸遍地，骸骨血水长流。
武小德目光动了动，朝前踏出一步。
冰晶小字一跳：
“你发动了‘见至’。”
霎时间，尸横遍野的长街化作了奈何之山，山间长路上满是木然行走的人们。
“原来你们在这里。”武小德道。
四周人群望着他。
他想了一瞬，抽出长剑，高高举起道：“无恨之人自去投胎，想报仇的人们——你们的力量借给我吧。”
“多重”发动！
霎时间，武小德背后浮现出一堵虚幻的墙壁。
——愿墙！
四周人群默默看着他。
忽然，一名穿着士兵服的男人走上前来，以满是血迹的双手按在墙上。
“都……给你，为我报仇。”
他沙哑着嗓子说道。
一股无形的力量从他手上传入愿墙之中。
这股力量离开他身躯之后，他的身形变得愈发单薄，仿佛随时都会散去。
“太多了，你应该给自己留一点。”武小德道。
“我唯一的愿望就是要那条蛟龙死——如果不是它，我们本可以胜的。”男人沉声道。
“我知道了。”武小德道。
他朝四周人群望去。
只见人们纷纷走上前来，将手按在愿墙上。
他们做完这件事，也不出声，只是望着武小德。
武小德继续前行。
愿墙随着他的脚步逐渐朝山川两边延伸，令一切亡者都可以看到它的存在。
武小德不说话。
他只是高举着荒剑，一步一步朝前。
每当他走过一名亡灵，无论男女老少，孩童与孕妇，士兵或屠夫，文人和走卒，纷纷把手按在愿墙上。
一行冰晶小字浮现在武小德眼前：
“‘多重’与‘见至’持续发挥作用，令你收集一切被屠杀人类的残余力量。”
“你的专属暗系道力即将诞生。”
武小德一眼扫过，高举长剑，迈开脚步，沿着山岭上的小路一直朝前走去。
不知过了多久。
终于——
他走到了路的尽头。
戴着面具的女子站在灰暗瀑布前，静静的看着他，以及他身后那无数啼哭的亡灵们。
“你收走了他们的力量，这样的话，你也就接过了他们最后的愿望与诅咒。”
女子说道。
“这是我愿意做的事，不过我有一件困惑之事——”
武小德摩挲着下巴继续说道：
“人族是不是经常死这么多人，然后一起来此投胎啊。”
在他背后的愿墙上，显现出一团充满了灰暗气息的暗火。
一行行冰晶小字浮现：
“诅咒与愿望之火。”
“你收集了十万亡灵的力量，也承受了它们的愿望与诅咒，如果你无法完成它们的愿望，这股力量便会化为对你的诅咒。”
戴着面具的女子听完，开口答道：
“人类的城池被攻破是常事——毕竟妖魔以人类为食，更多高等级的存在更是把人类踩在脚下——”
“这是人类永恒的宿命。”
武小德想了数息，将荒剑贴在愿墙上。
那股灰暗火焰顿时没入荒剑，渐渐消隐不见。
“最后一个问题——蛟龙你要不要。”
他头也不回的问。
“你觉醒了道力？不过人族里有些与你一样的存在，他们虽然有这样的力量，战斗却并不在行。”戴着恶鬼面具的女子摇头道。
“我领导夸我善战。”武小德认真地说。
“你确实有把握？”女子问。
“说实话，没有什么把握，但我会努力。”武小德道。
“——除非它已经陷入濒死，否则我不会出手。”戴着恶鬼面具的女子道。
“濒死么，这个条件不错，那就说定了？”武小德道。
女子想了想，点头道：“可以。”
“怎么找你？”
“我的真名为：流灵，如果你有本事把我召唤到人间，而蛟龙已经濒死，我便出手。”
“好。”
武小德将荒剑收入鞘中，回头望向那山岭间无数亡灵。
十万亡灵。
它们全都无声的看着武小德。
武小德咧嘴一笑：
“我们走。”
他踏出一步，解除了“见至”，霎时间重回人间。
所有寂静魔雾散开。
无边的诅咒与怨恨力量从他身上散发出去。
这股力量立刻让四周的一切开始变得灰败残破。
无形的力量波动散发开来，远空顿时有什么东西受此感应飞窜而来。
“哈哈哈！你终于敢出现跟我一战了！”
蛟龙在天空中疯狂大笑道。
它朝着武小德俯冲下去。
武小德以手按住荒剑，双目微垂。
两个冰晶小字跳出来：
“降临。”
下一瞬。
蛟龙突然停住俯冲，隔空喷吐出一团青火。
“去死吧，被我烧为灰烬的痛苦会一直伴随着你的灵魂！”
它在半空狂笑道。
武小德略一蹙眉，有些犹豫。
——这只是一团试探用的青火，蛟龙并没有用全力，所以接下来的战斗中，它随时可以再用这样的火！
那么策略改变——
同一时刻，武小德身上燃起黑暗之火，凭空凝聚成形，在他背后化作魔王冠冕。
黑暗魔王降临！
蛟龙的青火瞬息而至，将武小德彻底吞没。
“免疫！”
“免疫！”
“免疫！”
“免疫！”
“免疫！”
“一切法则之下的法术无法对黑暗魔王造成伤害！”
“注意，如果对方加大威力，恐怕会让此火超越法则的界限，从而可以伤害到你。”
武小德神情不变。
——正因为是对方的试探，所以自己才选择以魔王之躯硬抗。
这会给对方留下一个印象——
试探是没用的！
要么用全力，要么被这个人类再斩一剑！
青色的火海之中，忽然有一道残影冲天而起。
蛟龙见机极快，一口朝那残影咬去。
咔擦！
那残影顿时被咬得粉碎。
——却是一个假的青铜人偶！
“其实我在你背后。”
蛟龙背后传来一道声音。
“你以为我不知道？”蛟龙怒吼一声，身形以超乎想象的速度闪了闪，立刻首尾倒置，改为龙头对准那个发声者。
它再次喷吐出更加强烈的青色火焰。
“啊啊啊——”
那人发出一声惨叫，身形被焚烧殆尽。
高空的寂静魔雾之中。
武小德站在一只青铜手上不动，静静的看着这一幕。
——这种规模的青火，已经是蛟龙全力施为。
所以它的这个技能进入了冷却时间。
三分钟！
看着那青铜人被青火喷烧殆尽，他才抽出荒剑，轻声道：
“说出你们的愿望。”
荒剑上响起十万亡灵此起彼伏的声音：
“……复仇。”
“复仇！”
“复仇！！！”
充满痛苦与绝望的亡灵哭泣之声如排山倒海一般轰鸣不休，令长剑几乎握不住。
武小德将一只手轻轻按在剑锋上，令血水浸透剑身，轻声道：
“好。”
长剑立刻不震了。
这一刻，一切化为寂静。
武小德朝下一跳，越过长空，朝蛟龙直坠而去。
风在耳畔呼啸不休。
他握紧了剑。
那蛟龙刚全力喷吐青火，不防耳边忽然传来一道轻笑：
“别动哦。”
——魅魔！
蛟龙浑身一震，暂时凝在半空。
但它也早有准备，浑身鳞片上浮现出重重神异无比的天然符文。
这时武小德的剑到了。
一行冰晶小字跳出来——
“即将释放专属暗系道力。”
长剑上闪过灰暗之光，从天至地，斩出一个充满死亡气息的世界。
“没用的！我有龙神道符护体——”
蛟龙怒吼道。
它虽然看不见武小德，但已经感受到了对方的攻击。
所幸这种刚刚觉醒的修道者力量还不够。
幸亏如此。
对方伤不了自己！
——这一战，是自己赢了！
“最喜欢你们这种诚实的敌人。”
虚空中响起一道声音。
武小德突然收了剑。
那刚刚出现的世界随之全然收敛，回到荒剑之中。
却见武小德凌空踩在一只青铜手上，身形奋力一闪——
百里幻身步！
他整个人如疾风一般穿过长空，径直撞入蛟龙满是獠牙的大口之中。
继续！
武小德埋头继续朝里冲。
不等蛟龙有任何反应，他已经冲过了对方深深的喉咙，进入了它的肚子里。
长剑重新斩出去。
异象从长剑上纷呈而生！
霎时间，一行冰晶小字重新跳出来：
“你释放了专属暗系道力：”
“道·幽冥洞天。”
