# 第七百六十五章 吃一口可得长生！

猩猩的名字是流猩。
荷兰猪自称大壮。
马是白王子。
“久仰大名，见过各位。”
武小德一一拱手。
这些都是骸原本的力量，经过“诡命灾行”的汲取，化为有灵性的存在。
“大佬明，你真的曾听过我的名头？”流猩忍不住问。
“当然听过，”武小德点头道，“阁下在天神宫闯下那么大的名头，平天之矢，啧啧，谁人不知，谁人不晓？”
——诡命灾行里发生的一切，武小德自然知之。
毕竟这是他的术。
至于“三途薨地”——
自己已经脱离了那个地方，跟王灵秀来到了这根柱子里，自然不再受其影响。
多亏了不夜城。
话说不夜城到底在哪里？
猩猩的大笑声拉回了武小德的思绪。
“哈哈，好！大佬明，你果然是个不错的人，俺流猩越看你越顺眼！”
武小德笑笑，端起酒杯，跟猩猩喝了一个。
这里虽然是荒郊野岭，但猩猩摸出一根狼牙棒，左打打右打打，将一众邪化的存在全部杀了个干净。
世界清净了。
——托它的福，武小德跟王灵秀也就不必担心安全问题。
流猩喝了酒，忽然皱眉道：
“大壮探个路怎么去了那么久？白王子，你也去看看。”
“是，大师兄。”白马道。
轰——
它化作一道闪电，瞬间冲飞出去，在山林间几次起落就不见踪影了。
叶荒云从篝火堆前站起来，举着一张符，朝另一边密林走去。
“哎，叶道友，你去哪儿？”
武小德喊。
“哈哈，没事，我回个消息，马上再来跟你喝。”叶荒云道。
他钻进了草丛。
武小德看看猩猩。
猩猩耸肩道：“师父每天晚上都要跟他媳妇用符法视频，习惯了就好了。”
武小德端杯跟它一碰。
也不知道骸被“诡命灾行”分成了这么多份之后，会不会诞生一点真正的人性。
走一步看一步吧。
他下意识的回头看了一眼王灵秀。
只见王灵秀盘膝坐在一边，双手握诀，正在修行。
猩猩也看在眼里，感慨道：
“修行是个苦差事，俺也是下了一番狠功夫，才走到今日。”
“流猩兄修的是什么法？”武小德感兴趣地问。
“难！难！难！真法全在心，欲辨已忘言。”猩猩道。
武小德只是随口一问，倒也不认为对方会如此轻易就把邪性力量的秘密告诉自己。
忽然一阵狂风吹过。
猩猩放下酒杯，大叫道：“好妖怪，莫要跑，俺与你战一场！”
它冲上天空，追着那阵风走了。
篝火前安静下来。
武小德夹了一筷子菜，不停扒饭。
他一会儿夹菜，一会儿喝酒，吃的不亦乐乎。
王灵秀突然睁开眼，传音道：
“有敌人。”
“过来。”武小德道。
“嗯。”王灵秀起身，悄无声息的躲在武小德背后。
一只黑暗巨手将两人护住。
整个世界仿佛陷入死寂。
一息。
两息。
三息。
那股恐怖的悸动悄然消失。
“敌人怎么走了？”
王灵秀不解地说。
武小德也有些意外，忽然想起什么，站起身朝不远处的草丛望去。
——草丛里什么也没有。
疾风的呼啸声再次传来。
猩猩从天而落，不屑地说道：
“哼，这过路的妖魔跑的还挺快。”
武小德看着它，沉声道：
“大师兄，不好了，师父被妖怪抓走啦！”
“什么！”
猩猩大惊失色，径直跑到那草丛中看了一圈。
叶荒云果然不见了。
“我苦命的师父啊！你打电话也要注意安全撒！”
流猩一跺脚，捏了个术法，冲天而起，再次消失不见。
原地再次剩下武小德和王灵秀。
“要去帮忙么？”
王灵秀问。
“没事，你先修行，我来看看情况。”武小德道。
他在篝火前坐下来，双目微闭，仿佛入定去了一般。
另一边。
大山深处。
一处洞府之内。
十几个邪化的妖兽一起抬头，看着那吊在半空的叶荒云。
“大哥，”一头妖兽不确定地说，“这应该是某位大人吧。”
妖兽们全部望向坐在上首的那头邪熊。
邪熊看着叶荒云，点头道：
“没错，这一定是哪位大人，它被一种极其厉害的术法困在这人类身躯之中，不得脱身。”
众妖纷纷道：
“这可怎么办？”
“我们要不要帮帮这位大人？”
“可是用什么术呢？”
“直接把这位大人化为人形可是很难的。”
“对啊。”
邪熊大声道：“诸位，都想吧，看看用什么术法能破掉大人身上的术！”
众妖盯着叶荒云，纷纷陷入沉思。
然而它们议论纷纷的时候，虚空中有无数的呓语声在叶荒云耳边响起。
“它们在说什么？”
叶荒云只觉得头脑昏昏沉沉，强打精神才能听清一二：
“吃了这个家伙。”
“他身怀人类功法，又是修行许多世的功德圣人，吃了他可以长生！”
“没错，大王，下令吧。”
“好想吃他一块肉啊！”
——原来这些家伙想吃我。
叶荒云暗自寻思。
不过它们为什么这么厉害？
我连反抗的机会都没有，就被抓住了！
没办法，现在只能看流猩它们能不能来救我。
突然。
那一群邪化的妖兽中，一头恶蛟开口道：“我想到了一个术，兴许可以破开这位大人身上的力量，让他想起自己的一切。”
数百里外。
武小德猛然睁眼，低喝道：“去。”
一只黑暗巨手从他身侧的虚空冒出来，瞬间冲上天空，朝山林中飞去。
短短一息。
黑暗巨手从天而降，“轰”的一声打在那座洞府的门上，直接把门打了个稀烂。
邪熊大怒道：“竟然敢打上门来，左右，随我去看看究竟是什么人！”
山间的这一声巨响，立刻吸引了正在寻找敌人的流猩、大壮和白王子。
流猩先到。
它冲进洞窟之中，举起狼牙棒肆意锤击，直打得血流成河。
“慢！”
邪熊喝道。
“你抓我师父，还有什么遗言？”猩猩恨声道。
邪熊道：“听我说，你们本是——”
轰！！！
猩猩的狼牙棒照着它脸狠狠砸下去，直接把邪熊砸成肉饼。
那句话也就没有说出口。
——甚至说出口也没有用，毕竟“诡命灾行”依然在运转不休。
邪熊一死，身上忽而冒起一团魂光。
这魂光飞至半空，围绕叶荒云，急切喊叫道：
“大人，快醒过来！”
“什么？你都死了，还想着吃我？”叶荒云瞪眼道。
狼牙棒再起。
啪。
魂光被打散。
直到这一刻，虚空中，所有絮絮叨叨的呓语声才缓缓消失。
叶荒云只觉得脑子一清，立刻看清了当前局面，大喜过望道：
“徒儿，我在这里！快救救我！”
“师父莫怕，我这就来。”猩猩道。
不一会儿。
荷兰猪大壮和白王子也赶到。
三位弟子一起把叶荒云救出了洞府。
只见武小德和王灵秀已经站在洞府外相迎。
流猩上前一步，行礼道：
“多谢大佬明刚才那一击，我这才发现师父被藏在这处洞府。”
叶荒云听见它这么说，才明白过来，刚才那一声巨响乃是武小德出手引起的。
“多谢明兄。”他也感激地行礼道。
武小德淡淡一笑，说道：“都是去西天灵山问长生的同道之人，不必客气。”
叶荒云慨然道：
“正是如此，今后我与明兄当彼此扶持，争取早日抵达灵山。”
