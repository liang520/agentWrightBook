# 第六百六十章 相亲相爱钓鱼佬！

十一万三千五百年前。
某个世界。
一名身上系着树皮裙的野人坐在湖边，将一根树藤的末端系上小虫子，抛入水中。
他耐心的等待着鱼儿上钩。
忽然。
在他背后，出现了另一名野人。
“你这是在干什么？”
那个野人感兴趣地问道。
野人们之间的沟通原本只有“啊、咦、哈、哇、呼”之内的词，配合着语气和肢体语言，才可以让对方懂得自己的意思。
但是。
没有任何障碍，钓鱼的野人直接明白了他的表述。
“啊啊——咦——啊哇哈——”
野人放下钓竿，手舞足蹈地表达着自己。
询问的野人听着，看着，渐渐理解了他的意思。
“你说——”
“水里是一个未知的世界，里面是人类无法得到的、从未见过的怪物，你想看看它们？”
“有趣……这样的事情非常符合无知且愚昧的物种。”
野人失笑摇头，冲对方摆摆手，转身朝山里走去。
一万年后。
圣界。
一个类人的存在坐在青铜巨柱之顶，百无聊赖之下，想起了当年的事。
“钓鱼……也许可以打发事情呢。”
它索性随手捏出一根鱼竿，将长长的鱼钩抛出去，任由它穿透圣界，随机落在虚空乱流之中。
过了一会儿。
它觉得没意思，正要放弃之际，钓竿忽然传来一阵手感。
用力一扯。
一个从未见过的生物被扯进了圣界。
这是一头在虚空中飞舞的星灵鱼，是刚诞生的物种，浑身透明，变化出七彩之色。
就连圣界也没有见过这种奇特的鱼类。
有趣！
类人的存在继续钓起鱼来。
它不知疲倦的钓起自己从未见过的、绚丽多彩的物种，然后吃掉这些生命，只为了感受其中的乐趣。
某一段时间。
诸天万界陷入了无止境的杀戮和战斗之中。
这让整个钓鱼环境急剧恶化。
生命不再继续进化和演变，虚空中也不再安宁静谧。
钓鱼这件事受到了极大影响。
类人的存在有些恼火。
自己好不容易才找到一个合适的娱乐。
有没有什么办法让“鱼塘”恢复原本的样子？
想起来了。
有一些永恒的圣者们，为了下界的安危和物种演化，专门制作了干预机制。
那么。
自己也做一个。
类人的存在释放出永恒的伟大力量，具现成了一种独特的秩序——
“垂钓序列！”
这个序列的一切，都是为了帮助自己更好的完成钓鱼这件事。
钓鱼才是真正的乐趣所在。
时间缓缓流逝。
十万三千五百年后——
类人的存在坐在青铜柱顶，一动不动，依然在钓鱼。
忽然。
一行树藤般的小字浮现在它眼前：
“序列‘混乱’申请与您的序列‘垂钓’联合，开始在‘盘古’苏醒的时代活动。”
“本次奉上礼物：神器级钓竿‘老六王’。”
“担保人：顾青山。”
“申请者：‘混乱’的最后持有者。”
“申请理由：”
“从未出现在历史中的奇诡生命已经来了，这是全新的‘鱼塘’，如果您感兴趣的话，请允许‘混乱’与您的序列联合。”
它静静的看着这些文字，直到最后一行才略有动容。
“永恒的升序被翻转了，会有很多稀奇古怪的猎物……”
“确实有点意思。”
伴随着这道话音落下——
另一边。
诸天万界融合后的‘盘古’世界。
撒热镇。
酒吧。
二楼某个房间。
武小德眼前猛然浮现出一行行灰影小字：
“已链接‘垂钓’序列。”
“特别说明：”
“世间一切真实序列，皆是圣界永恒力量所化，各有其目的，各自运行。”
“‘垂钓’序列是其中的超级序列，身份尊贵，强大无匹。”
“在今后的战斗中，如果你的敌人被‘垂钓’序列看上，你将获得该序列下其他老六们前来助阵，你将获得丰厚的报酬！”
“其他老六也会邀请你参加钓鱼活动！”
“你们是相亲相爱的钓鱼佬！”
所有小字闪了闪，消失不见。
武小德咂摸了一阵。
——也就是说，如果敌人被看上，自己就可以摇人是吧？
那如果敌人没被看上呢？
还得靠自己。
总之是一个后手，但不一定奏效。
所以……
武小德心头忽有所感，站起身，朝两女道：
“你们不要惊慌，就在这里等着，我去办点事，马上回来。”
话音刚落，只听“唰”的一声，他从两女眼前消失了。
……
一座漂浮在天空中的岛屿上。
武小德显出身形，立刻洞悉了当前的情况。
他在心中默念道：
“使用叹息之墙，以我所学的‘天崩’献祭给‘不夜城’。”
叹息墙壁上，不夜城的图标顿时开始变化。
一行冰晶小字浮现：
“这是超乎想象的战斗神技，确定要将它献祭给‘不夜城’？”
武小德毫不犹豫道：“献祭，并且用叹息墙壁进行个人风格化。”
冰晶小字就不再出现。
——从谢道灵手中得到的技能都可以献祭！
这简直是一个天大的助力，武小德原本打算多学几门拳法用的，可是眼下的情形却不允许——
不远处。
一道人类的身躯站在那里。
但是他身上散发出奇诡的涌现波动。
很明显，这是一个被强制化为人类的奇诡生命。
至于武小德为什么会出现在这里——
代表“混乱”的灰影小字已经不断浮现在半空：
“第一次联合钓鱼执法开始。”
“你是新加入者。”
“你将代表‘混乱’迎战此怪物，为其他老六们争取时间。”
“一旦你争取到30秒时间，本次任务便可完成。”
30秒！
自己要争取三十秒的时间！
加入这个组织，第一次出场，岂能把事情搞砸？
武小德正默默地想着，却见对面那高大人影看了他一眼，淡淡地说：
“难怪我感觉有人在窥探，你这样一个人类，竟然敢出现在我的面前？”
武小德心头顿时感觉不妙。
电光火石之间——
他摇身一变，忽然化作一条光暗黑龙，在虚空中洄游不息。
化龙二阶！
在这一阶段，他可以化为光暗黑龙，具备“无存”的属性，但是不能攻击！
无穷的、恐怖的、伟大而不可思议的力量穿过了光暗黑龙的身躯，瞬间不知所踪。
光暗黑龙瞥了一眼虚空。
“神技‘天崩’已献祭完成，五秒后‘不夜城’进阶成功！”
无论是自己，还是整个任务，都需要时间！
光暗黑龙忽而高声道：
“你想要摆脱人类的身躯，重新回归奇诡的本相么？”
“你想知道圣界的秘密，以及这一切的起因么？”
“——想要真正的活着么？”
“其实我们可以谈谈！”
大地上，那高大的人类神情淡漠，开口道：
“聒噪，你以为说这些事就可以拖延时间？”
“不，可悲的众生啊，多活一秒，你的命运也不会改变。”
“等我杀了你，你的一切我自然知晓。”
他浑身散发出排山倒海的力量。
光暗黑龙装不下去了。
这特么太没面子了——自己第一次出任务啊！
他突然冲了上去。
“妈蛋的，你打我也是掉一格血，我打你也是掉一格血，你拽个毛线拽！”
它吼了一声，径直变回武小德，握着拳头就朝对方脸上打去！
一行冰晶小字早已跳出来：
“‘不夜城’进阶成功，它完成了专属风格化。”
“由于‘天崩’的超高等级，此死亡技已经化为如下技能：”
“地裂。”
