# 第八百三十二章 选定的秘密！

木桌后。
一名戴着金丝眼镜的鳄鱼人缓缓抬起头，朝“柳平”望过来。
“客人，请稍等，住店要登记您的信息。”
它彬彬有礼的打了个响指。
桌上竖立不动的眼镜蛇偏过头，朝“柳平”看了一眼，飞快说道：
“今早收到一条全系统通知。”
“真名为‘柳平’、‘顾青山’、‘武小德’的客人，务必先收现金，方可入住。”
“阁下既然是‘柳平’，请先支付住宿费用，然后再办理入住手续。”
“柳平”怔了怔，忍不住道：“为什么别人可以先住店？这不公平！”
鳄鱼人和眼镜蛇齐声道：
“注意你的言行，绝对公正的慈父在天上看着你。”
“不过，”鳄鱼人从怀里摸出手机看了一眼，“慈父给你准备了一张五折的优惠券，这可是史无前例的折扣，从来没有人享受过，你还不快快付账？”
“五折劵很少么？”“柳平”不禁道。
“当然，一般都是九点九折，这是历史上唯一的一张五折券。”鳄鱼人和眼镜蛇齐声道。
“此话当真？”武小德问。
“当真！这是绝无仅有的一张五折劵！”鳄鱼人加重语气道。
“柳平”略一犹豫便做了决定。
啪。
一声轻响，“柳平”消失，武小德出现。
五折券可以！
而且慈父给了这么高的折扣，想必是希望自己住在这里。
——这位存在全知全能，做事必然有他的用意。
他绝不会毫无理由的给一张五折券！
一定有重大理由！
“请给我办理入住吧。”武小德道。
“您选择什么房型？”鳄鱼人问。
“最便宜的那个——修炼小屋。”武小德道。
“请支付黄金，原本是要你八成的黄金，现在对折，只要四成。”鳄鱼人道。
武小德默默付钱。
“客人，这是钥匙，请去地下室左手第五个房间。”鳄鱼人指着一个方向道。
武小德接过钥匙，正要朝地下室走去，忽见客栈门外又走进来一名客人。
那是一名浑身长满金色鳞甲片的女子。
她出现的瞬间，鳄鱼人突然跳出柜台，直接挡在武小德面前。
“快去！”
鳄鱼人低喝道。
与此同时，大堂里忽然出现了一缕圣芒，直接化为头顶悬浮着光环的天使。
天使乍一出现，立刻高声道：“慈父的目光注视着这里！”
武小德还来不及多看几眼形势，便被两名穿着白袍的女子拽着，迅速走下台阶，进入地下室。
“请跟我来。”
一名女子道。
她们把武小德带到门前。
咔嗒。
门自动打开了。
“您可以享有安全的24小时。”一名女子笑道。
武小德道：“多谢，不过刚才那个女人是——”
“进去休息吧，晚点您会知道的。”另一名女子打断他道。
“……好吧。”武小德道。
他能感觉到，对方其实是在保护自己。
所以自己也不太好让对方为难。
那就先休息吧。
他迈步走入门口。
嘭！
门立即死死关上。
站在门前的两名女子浑身爆发出璀璨的圣芒，一起长出翅膀。
——却是两名天使！
“他进去了。”一位女天使低声道。
“立即开始转换时空，混淆所有房间。”另一位女天使道。
“嗯。”
两名天使齐齐释放出一道术法，打在门上。
门后传来沉重的移动声。
——房间离开了门，缩入地下，不知去向。
两位女天使忽然一起朝走廊望去。
不知何时，那名浑身覆盖着金色鳞片的女子已经出现在走廊尽头。
她注视着两名天使，将手摊开。
只见她手心里有着一颗跳动的心脏。
“天使……你们不想陷入永恒的折磨，就说出他的房间号。”
女子开口道。
两名天使脸色变得苍白。
但是下一瞬，忽然有一道人影浮现在她们身旁。
柳平！
“刚才多谢你们的掩护，现在快离开吧，由我替你们抵挡片刻。”
柳平笑着说道。
那金甲女人似乎想要做什么，却忽然浑身一颤。
一柄长剑穿透了她的胸口。
顾青山站在她身后，手持长剑，扭头朝外面的大堂看了一眼。
“你杀了客栈的人……恕我孤陋寡闻，难道这里可以随意杀人？”
金甲女人尖叫一声，直接扭转身子，一把抓住了顾青山的脖子。
“不，你不是末日，他的实力不会这么弱。”
女子道。
“但你依然没有避开他的剑招。”顾青山笑道。
他忽然化作虚幻的影子，消散不见。
——武小德的三神器！
原来在顾青山和柳平回归世界之前，他就以三神器复制了两人。
虽然复制品的实力大打折扣，但毕竟能用！
金甲女子一把扯出贯穿身体的长剑，扔在地上，再次回头朝走廊望去。
走廊上哪里还有什么天使！
“该死……”
金甲女子发出短促的尖叫，大步走出去，沿着满地的血水和尸体一路走出客栈。
只见客栈外的平原已经化作了血肉蠕动之地，各种稀奇古怪的器官内嵌在血肉里，缓缓的生长着。
数不尽的邪物遍布天地。
——这里已经被彻底封锁。
可惜——
金甲女子开口道：“他跑了。”
另一边。
武小德一步踏入房间，忽见四周皆是熟悉的场景。
街道旁的早餐摊上。
大蕉正吃着早点。
“老大，你回来了？”
大蕉远远的就看到他，直接打招呼道。
武小德呆住。
大蕉站起来，解释道：“暴牙守着沈小姐，我吃了早点顺便给他带一点回去——老板，再下一碗牛肉面，米酒，鸡蛋——老大来坐，先吃早饭。”
武小德缓步走上前，在大蕉对面坐下。
四周一切毫无虚假可言。
自己修习“万维垂相”之后，对于各种维度已经有了感应，能感觉到这里就是真实的世界。
这么说，自己又回到了昆仑山里世界。
一道声音忽然在心头响起：
“吃面吧，你也有一段时间没吃饭了，我慢慢跟你讲。”
“这声音……你是撒旦？”武小德立刻问。
“对头，我来其实是为了告诉你一件事。”
“什么事？”
武小德一边问，一边在大蕉对面坐下，等着牛肉面和米酒端上来，自己动手拿了筷子。
“你觉得邪魔好对付吗？”撒旦问。
“……顾青山似乎打的很轻松啊。”武小德道。
“刚才你抵达战场之后，一共来了三个找麻烦的，U级邪魔只说了一句话，另外两个就退走了，现在请重新回答这个问题。”撒旦道。
武小德想了想，耸肩道：
“你想说它们在战场上很有势力？”
撒旦的声音再次响起：
“萨琳娜进入圣界之前，由于极度的不甘心，所以才会寻遍一切世界和宇宙，最终在偏僻的‘虚空乱流’之中发现了一片枯叶。”
“你们的世界就藏在枯叶之中。”
“邪魔也藏在那里，正在全力攀爬圣柱。”
武小德突然意识到了什么。
萨琳娜已经要成圣了，还觉得不甘心。
为什么？
而且——
凭什么邪魔能躲起来，自顾自的攀爬圣柱？
天国的布娃娃指点自己获得了一条圣柱。
这完全是慈父开后门啊。
可惜，却被邪魔置换成它自己的。
若不是最后时刻，自己将两根圣柱毁灭……
撒旦的声音再次响起：
“萨琳娜想尽了办法也没能阻止邪魔的计划，幸而在她离开之际，因为你的一系列作为，让邪魔失去了那根圣柱。”
“她终于觉得有希望了，所以才把那个参战资格留给你。”
武小德不解道：“我们明明有顾青山和柳平，为什么不直接把参战资格给他们？”
“柳平本是最有希望成圣的参战者，但他被一百多名邪化的参战者围攻，最后一刻宁可让出位置，也没有投靠邪魔！”
“让出？”武小德追问。
“对，他用尽手段，把位置让给了萨琳娜。”
“——他可是个记仇的家伙，放弃参战之后，立刻就抵达了你们的世界，一路寻找邪魔的弱点。”
“如此一来，虽然邪魔没能获得前往圣界的资格，但柳平也同样失去了参战资格，只能作为帮手出场。”
“好吧，他只能作为帮手，”武小德摊手道：“那顾青山呢？”
撒旦加重语气，肃然道：
“圣界即是多元宇宙的‘道’，一切存在，都为了得‘道’而存在于战场上。”
“邪魔一直在邪化所有参战者，这样一来，无论谁获得升往圣界的资格，都等同于邪魔获得了资格。”
“它们要成为一切的真理，成为所有存在仰望的‘道’！”
“战场的裁判和规则只能勉强阻拦邪魔，堪堪维持住局面。”
“局面如此危急，如果再让一个强大到离谱的末日作为参战者抵达战场——”
“如果他投靠邪魔，一切都完了。”
“我们计算过无数次，假如末日投靠邪魔，那么圣界真的要成为邪魔的乐园。”
“邪魔将主宰一切世界，一切时空，一切维度。”
“所以不能让顾青山作为参战者，他顶多只能当一名帮手。”
“你们不信任顾青山？”武小德问。
撒旦加重语气道：
“这跟信任无关——假如末日和邪魔联合的话，诸天万界翻盘的概率是0，所以我们必须设下一道保险，你懂吗？”
武小德若有所思的点点头。
其实站在对方的角度也可以理解。
毕竟那个后果太可怕了。
“所以经过萨琳娜的选定，我就代表世界参战了。”武小德道。
“对，我们一直在观察你。”
“首先你已经获得圣界的失落之刃认可，自身不会邪化——这是最重要的一点，然后才是萨琳娜的指定。”撒旦道。
“我大致明白局面了，”武小德夹起一筷子面条，“还有什么要说的吗？”
“你只有一天时间能在这个世界中休息，好好抓紧这个机会。”
“邪魔已经在围杀你了。”
“一定要抓紧这一天的时间去做些什么！”
撒旦的声音徐徐消失。
