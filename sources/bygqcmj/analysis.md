# 原作分析：白月光的千层马甲

## 一、基本信息
- 题材类型：玄幻言情/马甲文/穿书/女主视角
- 总章数：28章（含前言）
- 总字数：约211369字
- 核心卖点：穿书女配借尸还魂后不断换马甲（叶玫→妖九→狂刀→女王），在男主魔尊面前千层伪装，所爱所恨之人皆为她一人

## 二、故事骨架

### 开局（第1-2章）
叶玫在穿书世界做了99年病弱女配，偷走女主复活项链，以身祭阵壮烈牺牲后借尸还魂为妖族公主妖九，却发现穿成了自己最痛恨的种族。

### 发展线
- **阶段一：妖城危机**（第3-5章）：妖九身份站稳→武器库收服妖刀→救老妖王免遭女主分身暗杀→魔尊裴羡黑化来袭→变身狂刀引走魔尊→妖城被毁但族人通过密道逃生
- **阶段二：华城考核**（第5-12章）：赚钱报名→天赋石全黑→与龙战李燕组队→考核赛中以药师身份carry→对战君海天→柳萌萌（女主分身）抢牌→凭隐藏牌拿第一→选择道宗
- **阶段三：荒山决战**（第12-14章）：诱杀柳萌萌→佛子神识阻拦→裴羡赶到灭佛子神识→钉死柳萌萌→各势力围攻→裴羡屠杀来敌→林行救场→叶玫装死脱身
- **阶段四：神宗生活**（第14-19章）：入驻道宗→巫管事欺压→背诵门规反制→夜探第五峰盗墓被裴羡撞→年亚澜现身→梅花桩显露实力→骗君海卫搜刮宝库+获合欢功法→筑基突破→年亚澜揭穿身份
- **阶段五：身份交织**（第19-22章）：小兰复活化形→年亚澜结契→裴羡vs年亚澜→挖坟成功→裴羡认叶剑为岳丈→道观建成→凝丹突破→老宗主渡劫（实为被雷劈死）→柳梦梦掘墓→裴羡入魔屠正道→叶玫以叶玫身份现身安抚魔尊
- **阶段六：妖族复兴**（第24-27章）：带资回妖族→老妖王已死→师父林行用镇魂铃杀妖王→身份暴露师徒决裂→新王登基→借无昼城之力建城→裴羡带魔族拆城→无锋剑认主掉马→与裴羡摊牌

### 高潮（第25-28章）
秘境探险→阴阳双令合一无效→年亚澜融身开门→天道真相揭露（造化玉牒=原书剧本）→天道企图吞噬叶玫→裴羡及时赶到→滴血认主造化玉牒→天道意志消散→开辟上层位面

### 结局（第28章尾）
叶玫成为新天道维系者，开辟真正的飞升空间，与裴羡大婚，妖魔人三族和平。
