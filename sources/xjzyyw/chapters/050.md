# 第50章

维里斯与黎诺对视了眼，想要抬手去摸黎诺的脸。
“小心，你手上正在输液。”黎诺将握着的手紧了紧，“有没有哪里不舒服？”
维里斯感觉有些困倦，眯着眼摇了摇头，嘴巴却说道：“腹部不太舒服……”
黎诺望着维里斯有些朦胧的双眼，将想要说的话咽进肚子，他清楚维里斯到底有多期待能够怀上宝宝，低声安抚道：“困就睡吧，没事了。”
“还没醒？”圣戴乔走进来在输液器中又添加了一些液体，这才扭头看向正一脸低落的黎诺。
“醒了一下，又睡着了。”
“加了两个营养成分。”圣戴乔习惯性地跟病人解释道，扭头见到黎诺一脸纠结地挠着头，忍不住问道：“怎么了？维里斯没事的。”
“不是，我在想怎么跟他说孩子的事。”黎诺轻声用气音说道。
“他知道。刚刚手术完醒了一下。”圣戴乔添加完药剂后做在黎诺旁边开了瓶水咕嘟咕嘟灌了几口。
“这样……”黎诺视线慢慢移到维里斯放到肚子上的右手，轻轻叹了口气。
“最近注意别让他受凉，适当走动但别做力气活儿，不过最多的还是多休息调理，保持一个好心情，我这边会给他配一些营养补充的输液。”圣戴乔坐在黎诺身旁，耐心说着一些注意事项。
黎诺认真地点了点头，“记住了，你放心。对了，给士兵试用解毒剂的场地准备好了吗？到时候你也过去吧，士兵们的具体情况还是你了解多一些。”
“当然，我会在的。场地估计下午才会安排好，上午我估计会在隔壁闵铖那边，你有事儿可以直接过去找我。”
“行，我今天刚想到调整下药剂添加一个成分，明天场地安排好可以先模拟一下。”黎诺点了点头。
两个人又沟通了一下关于解毒剂与士兵实际情况的一些猜测后，圣戴乔起身拔掉了输液的针头，将病房另外一张床推到了维里斯床旁，“完事儿了，好好休息。”
圣戴乔贴心的将病房里的防窥板打开，顺便还关了房间的灯，将病房留给了黎诺和维里斯两人。
第二天。
“醒了？”黎诺见维里斯醒来，伸手将他略长的头发捋到一侧，“安妮叔叔送来了塔叔一早炖的汤，起来喝点？”
“嗯。”，维里斯应了一声，缓缓坐起身来，没有多说话。
“肚子还疼吗？”黎诺一边问着，一边轻轻将手掌揉了两下维里斯的小腹。
昨晚黎诺没睡多久，就发现麻药过后的维里斯腹部难受翻来覆去，便爬起来揉了一晚上的小腹。
说是揉，其实不过是极轻柔的安抚。
维里斯低声道：“不怎么疼，雌子没那么柔弱。”
两个人都沉默了一会儿，安静地吃着塔叔做的早饭。
“再多喝一点儿。”，黎诺见维里斯还剩大半碗汤，忍不住说道，“你现在要把身体养好一点，不然根本抗不了我配置的解毒剂。”
“吃不下。”维里斯将眼前的汤往黎诺的方向推了推。
“再喝一点。”黎诺舀了一勺递了一下，见维里斯真的没有胃口转了个弯儿塞进了自己的嘴巴中。
黎诺快速又吃了几口饭，将东西全部收好，这才在维里斯身旁坐下，轻轻亲了亲维里斯的额头，“维里斯，你比任何人都重要。”
维里斯抬头看向黎诺，黎诺的眼神透露出坚定，仿佛十分有力的告诉着他自己的重要性。
“我知道，我只是现在肠胃不好消化，吃不了太多。”维里斯伸手环住黎诺，头靠在黎诺的身上，认真地解释道。
就在这时，圣戴乔和弗宾洛推门走了进来。
“材料和场地都准备好了，走！”圣戴乔丝毫不理会两个人此时如胶似漆的模样，说完后扭头走出了房间。
场地被加班加点准备就绪，整个一层中除了维里斯和闵铖的病房外，其他房间全部改装：模拟室，观察室，配药台，以及士兵们的排队等候区，休息区，全部都划分清晰。
透过控制室的玻璃，弗宾洛敲了敲窗户隔空指着配药台上面的茶青素原液，“哝，你需要的所有配药成分，包括茶青素原液也准备好了。”
弗宾洛和圣戴乔一同望向黎诺，眼神中透露着期待。
“谢了，麻烦圣戴乔医生和我一起进去，将模型调整成更贴近士兵身体情况的数据，我们再模拟一轮。”黎诺先倒了杯温水递给坐好的维里斯，转身走进了控制室。
圣戴乔对所有士兵的数据了如指掌，很快便调整好了数据，扭头看向正在配药的黎诺。
黎诺手上的动作也在不断地加快，他这次需要配置大量的药剂，所有的成分比例都不能出错，因此格外的专注。
一行人安静的望着认真配药的黎诺，都没有开口说话。
由于毒素以及士兵情况的不同，黎诺一共准备了四种药剂。除了之前配置的两种之外，还准备了另外对应的两种药剂的稀释液进行实战调配。
“开始配了啊。”玥恩教授声音不大，说着和理清教授走了进来，坐到了维里斯的身旁。
“教授，你们来了。”弗宾洛挥了挥手，拽着陪两位教授过来的斯威特坐到自己的一侧。
玥恩教授点了点头，随即扭头望向维里斯，说道：“身体感觉怎么样？”
“挺好的。”维里斯说道。
“那就好，好好养身体，等你体内毒素清除，让我爱人给你配几副备孕时候吃的药剂，备孕时候的身体状况很关键的，不然宝宝营养达不到很容易出生有缺陷的。”
玥恩教授耐心地跟维里斯讲雌子备孕时的注意事项，借此来向维里斯解释为什么没有留下宝宝。
维里斯没有说话，垂着眸子仔细地记下了玥恩教授说的一些注意事项。
玥恩教授和理清教授两人都是过来人，对这些事情了解的多，而且身为贵族，本身就对子嗣养育方面更加严谨一些，每一句对于维里斯来说都十分受用。
慢慢的，两位教授的科普引得弗宾洛和斯威特两人也凑上前，一时间控制室外变成了一个备孕小课堂。
大概过了一个多小时，模拟室内的光屏亮起，“模拟装置开启，数据加载成功。”
圣戴乔和黎诺的准备工作都已经完成了！
