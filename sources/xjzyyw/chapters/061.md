# 第61章

名单随着调查不断的变化，三军部率先开始在边境与偷盗者们进行阻击。
闵铖和黎诺等人受到的袭击早就让维里斯和拜里等人内心充满了怒火，再没有以往对待偷盗者的柔和，一旦发现他们在边境的临时驻地，军用信号屏蔽器包围，直接攻破，绝不让任何人逃走。
偷盗者在边境停留的地点本身就不确定，加上未知领域本就危险重重死亡率极高，一个个团无声无息消失，等到察觉时，在边境的探索队人员已经少了大半。
而这时，两位教授早就被三军部秘密送到了驻地，因为维里斯等人收缴的货物全部暂存在三军部驻地的仓库里。
与他们同去的，还有之前进行元素分析的专家和军医院派来的几位专业医生。
维里斯躺在驻地的病房中，皱着眉等着身体的检查结果，他带来的解毒剂都已经喝完，医生给他带来了新的药剂。
“恢复的不错，这个疗程的药剂再喝完应该就没大问题了。”医生看完检查报告，把新药剂放到了床头。
一旁的理清教授忍不住笑了一下，“挺好挺好。”
维里斯翻身坐起身，将药剂收进了自己的口袋，扭头礼貌的对两位教授说：“我陪你们去仓库。”
两人对黎诺的照顾早就让维里斯对他们心存感激，只是平时都是黎诺与两位教授相处，维里斯一时间还对两个教授的热情有些无措。
他们这次到来带了大包小包的研究器材外，还带了舅舅舅妈给他准备的各种补品，同为雌性的两个教授对他更是没了黎诺面前老师的模样，总是疼爱的眼神看着他。
“你再休息一会儿，圣戴乔医生说你要多休息的，我们有人带着去的。”玥恩教授阻止了维里斯下床，和理清教授大步离开了病房，“走了，别下来。”
虽是这么说，维里斯还是跟在他们一同去了仓库。
军部仓库特地开辟了一个区域放着缴获的物资，在教授等人来之前又特意开辟了一个空间给他们做研究室，还准备了防护服。
两位教授到研究室的时候，设备已经都装载完毕，元素分析的研究员已经进了仓库寻找他们的目标。
交货的物资大大小小有几千种，不过士兵早在他们来之前就做了许多简易的架子将东西全部分类摆放整齐。
维里斯站在仓库门口，看着两个教授穿着防护服进仓库和研究员们汇合，转身离开了那里，他还要准备参加连线会议，了解军部下一步的活动。
进了会议室，克丝亚正站在会议屏幕前拿着厚厚的一摞资料讲着，他擅长的不是作战，更擅长的人心，因此现在活捉的人全部有他带兵看守和询问。
这么多天的调查陷入了瓶颈，最核心的几个贵族名单经过调查都被排除，现在名单上的人都是一些可以砍掉的旁支。
“这些名单里的人已经可以确定都是内部成员，他们有不同的职责，除去现在已经确定负责边境物资搜集……”克丝亚见到维里斯走了进来，将手上一份资料递给他，没有停顿继续说着。
维里斯翻开文件，就看到了克丝亚目前整理的最新消息，这么多天，克丝亚已经将负责边境物资这方面的关系网络整理十分清晰，同时也罗列了其他已知消息。
翻到一半，维里斯视线停到了一页。
“维里斯？怎么了？”拜里中将率先透过屏幕看到了维里斯的停顿。
“这个人我见过。”维里斯拿起手上的纸，将他看的那一面对准投屏确保大家知道他说的是谁。
纸上印着的是几张照片，这些照片的主人是个‘关系户’，因此说了很多有用的消息，其中一张里与他合影的年轻雌性则正是维里斯认识的人。
“他就住在我家附近，我回家的时候会碰见他在园子里玩耍。”维里斯脑海中飞速闪过之前看到的所有名单和照片，“他的家人应该也是一名贵族，但没有在名单上面。”
维里斯说完这个人的住房地址，沉默了许久。
克丝亚和萨力贝也没有说话，只是克丝亚的手在听到维里斯说完忍不住抖动着，他承受的煎熬要比维里斯等人承受的更多。
闵铖作为最早中毒的人，已经开始出现病情加重的情况，随时可能突发意外。
有了地址军部定位的十分迅速，很快就查到了这个人的资料。
照片的人是维里斯所说的贵族家中私生子，没有上贵族户籍，也没有出现过任何公众场合，若不是维里斯提到，还真的没有人能查到他的消息。
简单的交流了后续边境这边的安排，一区那边匆匆挂断了通讯，开始认真调查起维里斯的邻居--兹尔家。
“终于有点进展了。”克丝亚轻声舒了口气，微垂着头露出疲惫的神色，接连几天的拷问调查让他眼下有了黑眼圈。
“还有更好的消息，他们在仓库找到毒素来源了。”维里斯关上手上的光脑，站起身向门口走去。
三人走到研究室的时候，门外已经围满了士兵在安静的透着玻璃关注着研究员们的进展。
见到维里斯等人，士兵们自然地给他们让了点位置，一起看着里面。
只见屋内的台子上放着一块不大的黑褐色硬块，一名穿着防护服的士兵正在里面用小锉刀一点一点在边缘磨粉末到一个小碗中。
研究员和教授等人则坐在仪器前讨论着什么。
“进展怎么样？”克丝亚抓住身旁的士兵问道。
“刚刚教授喊了人去询问那些偷盗者这是什么，他们没人知道，说管这个的人已经死了。”士兵抿了下唇，“具体还得等教授再出来的时候问了。”
似乎是注意到了维里斯等人过来，理清教授起身走了出来，递给克丝亚一张照片，“这个东西应该是存放了很久的，而且只有一小块，还得麻烦你再去盘问下那些人。”
克丝亚拿过照片，一边点头一边匆匆去了牢房。
不一会儿，克丝亚便询问回来了，这个物体是一个成员从去世的老成员房间拿到的，具体的情况他也不了解，但听过那个成员说这个东西似乎是个很厉害的东西。
理清教授听完点了点头，讲了他们的检查结果。
这个硬块应该是一种树木的躯干，里面含有的毒素正是导致黎诺等人中毒的原材料。
通过这个木头的其他成分分析，这个树木应该是生长在较为阴冷潮湿的地方，目前最好的办法就是找到他的生长地点和伴生兽。
萨力贝少将没有半点犹豫：“没有线索的话，很难找到。”
边境之外的神秘领域不是没有人去过，包括这些偷盗者和以前国家派出去的探索的士兵，能够活着回来的概率很小，在这种没有方向下，更是艰难。
“我们现在先把这个木块切一部分提前毒素，用来找解毒办法，另一方面就还得你们尽快找到知道这个东西的‘偷盗者’了，我们需要尽可能知道更多地信息。”理清教授看了眼屋里。
“我们懂得，辛苦你们了。”萨力贝少将点了点头，拉着维里斯和克丝亚向后退了一步，“我先带他们两个休息，明天还要继续出任务。”
萨力贝少将说完拽着两人向宿舍走去，这两个人自从来了边境就跟疯了一样忙碌，只能他亲自压着两人休息才行。
理清教授这才注意到外面已经黑了，走到维里斯身边拍了拍他的肩膀，“放心，很快就能找到解决办法的。”
送少将和士兵们离开，理清教授进了实验室望向还在低头认真帮忙磨粉末的小士兵，“小伙子，你明天是不是出任务？不然你去休息吧，我们自己来。”
士兵摇了摇头，“我后勤的，不出去行动的，我们小队长说全程协助你们。”
大概怕教授们不好意思，他又补充道：“缺人或者是我累了，可以喊其他后勤士兵过来接班的，教授你们有事吩咐就行。”
理清教授点了点头，“我们今天要通宵的，你要是累了到时候别硬撑。”
士兵嘴唇动了动，没有再多说什么，现在时间紧迫，他没有办法开口劝几位教授休息，只能轻轻点头继续认真工作起来。
明明是很清爽的气候，但对于整个三军部来说似乎是来了一场寒冬，所有的小进展都是漫长而又枯燥，头上悬着的倒计时又在开始倒计时。
小士兵休息的时候舔了舔唇，在他知道自己和军部大家都中毒的时候，他和大家都没有任何的畏惧，唯一担心的就是比较严重的少将。
从做了士兵，到了边境，看了太多太多的生死离别，他把自己的命看的很淡很淡。
可是现在知道闵铖中尉和黎诺药师中毒时候，他和大家一样，所有人的心都悬了起来。
说句真心话，在他的眼里，他认为自己，这样一个小小的士兵的性命和一个制药师的性命重量是不同的。
因为他把自己看的很渺小，而且他是一名士兵，早就做好了奋战最后一刻的准备。
确切说，如果真的有死神拿着镰刀站在他面前，让他选择自己死还是黎诺药师死，他更希望自己死。
他觉得每个士兵都是这么想的。
黎诺救了所有中毒的三军部士兵，他提供的药剂救了所有患病的士兵，解决了三军部长期以来药剂不足问题。
黎诺不能死。
派去保护黎诺的弟兄们也不能死。
他们都得，好好活着。
