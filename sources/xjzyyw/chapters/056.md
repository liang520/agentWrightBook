# 第56章

和舅妈回到军医院分开后，黎诺依旧一副若有所思的模样。
维里斯从闵铖病房回来，便看到了黎诺思考的模样，忍不住上前问道：“怎么了？”
“没事，今天跟着去药厂跟着舅妈学到很多，如果不是现在情况特殊，我真想回和你一起去雌父以前住的地方生活一段时间。”黎诺伸手环住维里斯的腰。
黎诺靠在维里斯怀里，一时间不知道该如何跟维里斯表达自己的想法。
舅妈炖汤的方法，让他不禁想起家中那本誊抄古书，他的直觉告诉他舅妈的做法很像古方中的煎药。
比起更加先进发达的一区，舅舅家在的地方像是世外桃源。他不禁想，是不是带着维里斯到舅舅家休养一段时间更利于维里斯身体？
维里斯其实也听过舅妈提起他们那个星球上的生活，忍不住摸了摸黎诺的头。
这时，克丝亚从闵铖病房中走了出来，见到黎诺如同一只挂件一样依靠在维里斯身上，忍不住翻了个白眼侧身走过。
“呵呵，感情真好。”拜里中将和其他几位士兵在克丝亚身后从闵铖病房走出，经过两人。
黎诺没想到突然出现这么多人，强装镇定的从维里斯怀里出来，“他们都在啊。”
“嗯，闵铖今天状态好了很多，大家都过来看一下。”维里斯望着黎诺红透的耳朵，忍不住伸手捏了捏黎诺的耳垂。
“闵铖的情况好很多吗？什么时候能醒过来？”黎诺听到很高兴，忍不住开口问道。
“还不能醒过来。外伤已经都恢复差不多了，但整个人还是没有意识。”维里斯垂了一下眼眸，“圣戴乔借了你们实验室的设备，估计明天一早设备到了会进行一些检查。”
“我们实验室？什么器材？”黎诺好奇地问道。
“检测毒素的器材。”维里斯轻声说道，他望了眼黎诺，黑色的眸子映出了黎诺略带惊讶的表情。
“是毒吗？”黎诺问道。
“圣戴乔怀疑，也是受了你这次试剂的灵感。”
黎诺点了点头，袭击闵铖很可能跟掉包军部药剂的是同一批人干的，他们能够想到毒药剂，很有可能已经对毒有一段时间的研究。
但是……
黎诺忍不住皱起了眉头，灵植、异兽、伴生兽体内大多都有含有毒素的，可以说即使前段时间黎诺和两位教授整理了一本厚厚的毒品集，却只是冰山一角……
边境之外更加神秘无法探寻的地方，还隐藏着众多无法探索的神秘物种……
“教授他们下午来吗？理清教授前阵子帮忙整理的资料更多些，有些仪器无法完全判断的毒素可能需要他帮忙判断。”黎诺问道。
“来的，你也要去。”
黎诺连连点头，“当然，我肯定在的。”
第二天一早，黎诺就看到士兵们将实验室的仪器一台台拖进闵铖的病房。
“教授们呢？”黎诺忍不住望着最后一个士兵的身后。
“玥恩教授开房车走特殊通道，估计刚接到理清教授。”克丝亚站在黎诺身旁说道，“等他们的时候你接一下仪器。”
克丝亚说完，越过黎诺进了病房，站在了闵铖的床前。
这么多天，克丝亚都忍住没有要求进病房打扰闵铖，只是跟其他人一样在观察室里远远望着。
黎诺进门见到克丝亚拉着闵铖的手十指相扣，没有出声，快步走到仪器面前。
这些检测和实验仪器都是为了解毒剂定制的，由于赶工匆忙，这些设备体积庞大，线路复杂，没用过的人真的没办法完整复原。
“这个帮我接那个蓝色接口。”黎诺拽着一条线跟身旁的士兵说道。
士兵一脸茫然的望着埋头连接线路的黎诺，一时无从下手。
“我来吧。”维里斯从士兵手中接过线的另一头，走向黎诺说的蓝色接口。
没有看着仪器的黎诺完全不知道身后的士兵换了人，只觉得配合他的动作越来越熟练。
等到所有的线路接完，黎诺才擦着薄汗回头看到站在自己身后的维里斯，“怎么是你？有没有累到？”
“没事儿，我就接了几个，其他都是他们帮的接的。”维里斯指着踩在设备顶部的几个下属。
维里斯总会最快找到黎诺说的线和接口，士兵们看到维里斯找到对应的地方便接手负责爬上爬下连接接口。
“那就好。”黎诺看着几个士兵麻利地从机器上跳下来，一惊，忙说道，“小心些，太高了别摔到。”
砰砰几声，士兵们顺利跳下，身手灵活。
其中一个士兵突然发现有接线工具落在机器顶部，一个跳起，抓住机器一角麻溜儿地又爬了上去。
见此，黎诺想起刚刚自己让士兵扶着梯子爬上爬下的场景，忍不住闭上眼不愿回忆。
“还是黎诺厉害，我们根本都没看懂这些线路咋连的。”士兵说道，他们接触的设备都是最顶端的粒子技术，平时很少要跟这种旧式接线设备打交道。
黎诺仰起头，望着硕大的整个大型仪器，话语中流露出了不易：“这些都是前段时间翻旧资料模仿古文明建的，确实有些复杂。”
他一点都不想承认哪怕是实验室的新设备也都达不到士兵们的粒子技术的战机和武器。
两位教授很快到了，黎诺和教授等人站在设备旁，一切准备就绪。
圣戴乔带着几位医生到了闵铖床边，拉上了窗帘开始进行样本采集。
很快，第一份样本从帘子缝中递了出来。
护士通过医院的仪器对样本中的细胞及液体进行一些处理后，陆陆续续开始向检测毒素的设备仓中放标本。
设备在第一份样本时便开始响起。
黎诺忍不住盯着设备屏幕，皱起了眉头。
现在哪怕是普通人体内都会多多少少含有毒素，人体缓慢排出的没什么危险，最怕的是在还没有排出体内的时候与一些外界因素发生反应。
密密麻麻的解析单列了出来，解析出来的毒素也开始逐步放进一个个标好标签的标本针中。
“这几种都是常见微量毒素，不用惊慌。”理清教授说道，有意的安抚围观的人。
“呼……”一个士兵轻声呼了口气，他看到黎诺皱眉的时候立刻屏住了呼吸，好在没什么大事。
“怎么？”维里斯轻声凑近黎诺的耳边问道。
黎诺扭头，嘴唇贴近维里斯的耳边，解释自己皱眉头的原因，“这些毒素应该是很快排出体内的，闵铖住院这么多天，按道理不应该还存留在闵铖的体内。”
维里斯听完，回忆了一会儿，说道：“他身体代谢减少很多，像我的话那些外伤应该几天就开始愈合了。”
黎诺对这些不太懂，没有说什么，继续盯着仪器工作。
闵铖的各处检测标本采集越来越多，自始至终都是常见微量毒素，就在大家内心开始打算排除中毒可能的时候。
“停！”黎诺和两位教授一起大喊道。
“这是哪里取得样本？”玥恩教授问道。
“脖子。”圣戴乔的声音从帘子后面传出，“有东西？”
“对，你稍等。”黎诺和两个教授默契十足的开始进行新毒素的封装，记录保存等等。
三两下搞定后，理清教授打开自己的光脑，“继续。”
过了半个小时后，圣戴乔结束采集拉开帘子，望着外面的护士和黎诺等人画的分布图，脸直接变黑。
从闵铖的脖颈开始到脑部，全部都能采集到一种无名毒素，且通过分析表明，当时毒素应该就是从脖颈处向上蔓延。
大脑的位置，怎么解毒？如果不及时解毒，后果又会怎样？
