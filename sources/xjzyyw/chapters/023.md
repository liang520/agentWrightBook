# 第23章

看到汴扬和汉苏要结婚的消息，黎诺微微一顿，慢慢翻看着记录，心里开始回忆上辈子的事情。
从他开始在庆典对少将求婚到现在，事情的发展轨道已经与前世越来越不同，但又好像会得到与上辈子相似的结果。
虽然他与汉苏分手让汴扬没了药方和金钱，但是汴扬依旧凭借着公开与自己pk，赢得了众人的关注，岗位升职加薪。
就像维里斯也是。
上辈子，接连斩获功勋升级少将的维里斯并没有发生战场的失误被停职，而是继续在边境获得累累功勋，赢得了众多人的崇拜。
但这辈子，就算维里斯被停职，但在黎诺的担心中，所有的严重后果都没有发生，小打小闹的罚了点钱就过去了，根本都没有轮到维里斯动手解决。
要知道，维里斯少将在后世最出名的就是能武力解决的从不会多讲道理。
黎诺对汉苏和汴扬的婚礼一点都不感兴趣，他在意的是“未来”是否会兜兜转转又发生呢？
黎诺心里想着，一时间有些晃神。
他并没有注意到滑动光脑的手指停留在了毕丽斯发出来的婚纱照上，更没有注意到身后维里斯正注视着他的举动。
战场上以一敌百，杀异兽如同碾蚂蚁的维里斯少将怎么可能先累得睡着呢？
维里斯的眼神变得冰冷，若是他手下的士兵一定一眼分辨出维里斯在生气。过了一会儿，维里斯翻身靠向了另一侧。
黎诺听到声音回神，将聊天记录翻到了最新，回复道：“收到了，不打算去，有那个时间不如多接几单药剂来做。”
毕丽斯很快就回了消息：“你终于出现了，你和维里斯少将的婚礼什么时候举行啊。”
“估计要晚一些，我需要把户口分出来然后去将维里斯少将的身份加入族谱。”黎诺回道，随后他皱了皱眉，他的雄父又跑去别的区玩去了，也不知道什么时候回来，发的消息也不回复。
“对哦，还有这个手续，弄完再举行婚礼吧。”弗宾洛也冒泡出来了。
“你和斯威特两个呢，什么时候举办婚礼？”话题转回到弗宾洛和斯威特身上，黎诺将光脑放回床头柜，给维里斯塞了下被角，相拥沉沉睡去。
第二天一早，圣戴乔敲响了黎诺家的大门。
维里斯望着与总军医院同型号的治疗舱被扛进来，一时间陷入沉默，这种治疗舱是目前市面上顶级的型号，没想到黎诺直接买了一台回来。
“我昨天跟黎诺一提说买了你可以在家里进行调理，他立刻就说给你买，完全都不带犹豫的，对你多好。”圣戴乔推了推维里斯，一点都没有被维里斯的臭脸影响到。
维里斯望着还在里屋陪工人调试机器的黎诺，冷着张脸心里想着：小鬼对钱根本就不在意，一点都不心疼，这就算好？
前面对他温柔如水，转头盯着前任照片目不转睛。心都捂不热。
维里斯哼了一声，淡淡说道：“没我对他好。”
圣戴乔被维里斯的话一噎，想起维里斯对军部众人的态度，一时间觉得说得对。
“也不能这么说，你看你还配合拜里搞什么军部停职检查，平白被总部扣了那么多钱，还不如直接跟小黎诺说想要合作。”圣戴乔小声嘀咕道。
“你知道这事儿？”维里斯问道。他当时知道拜里上报违纪是有什么战术，但还真的不知道是冲着黎诺的合作来的。
“当然是……我和萨力贝讨论的。”圣戴乔在维里斯的注视下只能收起装逼的心，“你当时忙军务不清楚，我全身心盯着网店就有猜测咯，拜里让你冲锋我和萨力贝后来一起讨论就猜到了。”
维里斯深深看了眼圣戴乔，“嘴巴牢一点。”
“我嘴巴牢的很，昨天是因为，你又不在意那些谣言，我就以为没什么啊。”圣戴乔瘪嘴道，他想起维里斯在光脑给他发消息骂他蠢猪，斜眼看着维里斯，“我看你俩反而更亲近了，你应该谢谢我。”
“呵。”维里斯轻轻一声，引得圣戴乔老老实实闭上了嘴。
过了一会儿，黎诺从里屋走了出来，他们还要赶去见拜里联系的老士兵们。
拜里在三军部开了一个办公室出来专门给几个老士兵进行药剂融合，黎诺三人到的时候，几个老兵早就在里面等待着了。
门一推开，维里斯走进去，几个老兵唰的一声站了起来，“维里斯少将好！”
声音嘹亮，眼里还闪着光。
“他们大部分人的命都是维里斯救回来的。”圣戴乔凑近黎诺的耳边，小声解释道，“而且维里斯都会自费买医药食物帮助他们。”
黎诺望着屋子里八九个雌子，没有说话。前两天他和维里斯去看望的福港并没有在里面。
正想着，维里斯的亲卫官闵铖走了进来，“少将，就差福港了，他估计要出院才能来，到时候让其他的兄弟教他。”
维里斯点了点头，让闵铖将准备好的药剂放到黎诺跟前。
其实整个教学过程十分简单，老兵们都是操作精神力作战的好手，只需要了解融合药剂时的如何用精神力控制住药植中的灵力就可以，其次，就是要了解不同的药剂融合的方式。
几个士兵在看着自己的精神力引导下逐渐融合形成的一级药剂，显得格外开心。
“哎呦，早知道这么简单，我们军部哪会缺药啊。”角落里一个叫纳布斯的老兵很快就上手，摇晃着手中的药剂说道。
“少来了，这才一级药剂，而且最重要的是药方，又不是药剂融合，你去药厂干这个你看看你才能拿几分钱。”坐在他一侧的士兵罗纳说道。
罗纳右手的无名字和小拇指连同半个手掌已经都失去了，他三个手指仔细的输送着精神力引导药剂融合。
等他弄完，黎诺走上前拿起他融合的试剂，淡淡地说道：“可以，第一个合格的人。”
原本高兴的纳布斯一愣，一脸茫然地看着黎诺。
“少将的对象，为啥我的不行啊？”
