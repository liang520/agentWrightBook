# 前言

《我在星际做药王[重生]》
作者：连城雨
文案：
黎诺身为贵族雄性，为了爱情甘做舔狗，娶平民宠上天，然而，他死在医院时才知道，他的头上长得是绿绿的草啊。
幸运的是，他重生在了给平民申请正妻之位的宴会上，他望着眼前的国王和身后看好戏的众人，灵机一动话锋一转，“还请国王能允许我娶维里斯少将为正妻。”
维里斯少将，出身贫民窟却做了帝国最年轻少将，是大家心中的战神、白月光。
黎诺微微一笑，上辈子维里斯少将一直单身，这次也一定会拒绝他的，想必今日可以顺利渡过此劫。
没想到这一举动让他娶战神回家的同时，拿回了上辈子被渣雌偷走的药方。
黎诺望着前世让小三名扬天下的药方册子，又哭又笑。
这一世，他才是药王！
