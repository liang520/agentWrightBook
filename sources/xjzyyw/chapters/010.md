# 第10章

第二天，黎诺准时到了研究院，刚进门便被制药部的玥恩教授拦下。
“黎诺，你怎么要转到灵植系，你看你学习成绩好，在这边实习的时候也做的很好，我相信你很快就会研制出自己的药剂的。你是不是有什么困难，我们来聊聊？”玥恩教授一脸的和蔼。
“教授，我……”黎诺正在想如何委婉地结束对话，毕丽斯的雌父理清冲了过来。
“黎诺虽然制药成绩好，但灵植系也很适合他啊。”理清教授抓着黎诺的手，扭头拽着就走。
玥恩教授跟在身后一把抓住了即将合上的电梯门，闪身挤进电梯。
“没想到玥恩教授一把年纪了身手还如此矫健。”理清教授说道。
“闪边去，”玥恩教授摆手说道，“黎诺呀，你再好好想想…”
理清教授拍了拍玥恩的肩膀，拽着他进了办公室。
“玥恩啊，我来跟你聊，你说现在制药部还是我们刚进研究所那会儿吗？就拿着我们当年找到的原液随意搭配看看有没有戏出新药方，也不研究药植，也不研究其他制造方法。”
“那是他们自己把路走窄了。”玥恩教授叹了口气，“但黎诺有天赋成为很好的制药师，你们灵植部这不埋没了嘛？”
“怎么会埋没，自己一个人在家都能研制出新的药用原液，这不是我们灵植系探索未知最需要的孩子吗？”理清喜滋滋的看着黎诺，眼神里透露出了满意。
“新原液？谁？黎诺发现的？”玥恩一脸惊讶，“那更不行了，黎诺跟我回制药部…”
两个人各自拽着黎诺的一只胳膊不松手，开始互相嘲讽了起来。
玥恩：“你一个研究种草的怎么教学生药理，还得我来？”
理清：“放屁，我今年就开药植课题，到时候和黎诺一起研发新原液，当年我做原液可比你快，再说你们部门都没人了，谁教黎诺？”
玥恩：“你还知道我们部门没人了你还挖人，我亲自教！你照照镜子，你哪里像个埋头钻研学术的。”
理清教授一顿：“咳咳，我知道我比你好看，你别嫉妒。””
玥恩教授气急，“我嫉妒个屁！”
理清教授个子比黎诺要矮一些，偏瘦的身材配上一张娃娃脸，一身休闲装，一点都看不出孩子已经18岁了，反而自己像个学生。
再一看玥恩，披着实验室里的白大褂，里面白色衬衣加西装裤，微秃的头顶，看上去更像理清教授的老师。
“我们听听黎诺的想法吧。黎诺是不是愿意跟我研究药植？”理清教授笑眯眯的望着黎诺问道。
“黎诺以后是不是打算继续制药？你来制药部，我绝对把毕生所学都教你。你认真思考下以后打算往哪个方向发展？”玥恩教授说道。
“玥恩教授，我知道你很希望我去制药部，但目前理清教授打算开药植的灵植研究更加吸引我。”黎诺语气带了几分肯定，轻声说道。
在理清教授鼓励的眼神下，黎诺继续说道：“比起立刻拿起原液去钻研药方，我更希望能够更多地去了解灵植，找到他们的特性，摸清它们以及它们与其他灵植产生的影响。”
“对嘛，你看，你们制药部不适合黎诺。”理清教授说道。
“我昨天无意中看到了一个古植的介绍，它可以做药方中的一味药材，但同时他是杂草类植物。因此我认为当我足够了解灵植以后，很多灵植自然而然就会成为药植，药剂也随之研制出来。”
“所以你发现的原液？”理清教授忍不住想问道。
“是观赏树熟透的金果。”黎诺说道。
“这样啊……”理清教授摸了摸下巴，“你回去吧，我和玥恩教授再聊聊。”
黎诺点头，把打算带给毕丽斯的原液留下，转身离开。
理清教授望了眼陷入沉思的玥恩，忍不住说道：“怎么啦，是不是感觉被年轻人教育了？”
“你真……别烦……”玥恩拍开理清伸过来的手。
“早就跟你说了那老驴头带不了制药部，现在他们都跑了剩你一个人接他们的锅，有意思嘛？”理清强硬的摸了摸玥恩的头发。
“老子早就不想接锅了，要不是黎诺这孩子有能力我能过来堵门？”玥恩憋了憋嘴，扭头出了办公室。
“干嘛去？”
“转部门！”
--
出了研究院的大楼，黎诺坐在楼下的便利店里的想了很久也没有回忆起上辈子玥恩教授提前退休的原因。
但很快他的注意力便被吸引到了别的地方。
有人一口气将他店铺里的龙舌补血液全部买走了。
往常，一级止血药剂大概10金币一瓶，原材料就需要用三分之一的龙舌草。黎诺仅仅用了一株龙舌草就制作了40瓶，也就是说，这一株就赚了200金币。
黎诺为这个数字感到震惊。若是所有的药植都可以达到这个效果，那么药剂的价格就会大大下降，平民甚至贫民区的人都可以轻易的用上药品。
最重要的是，这个药剂不需要他的精神力，完全可以用机器大量生产。
黎诺思索着是否要大规模生产的时候，一条评论映入眼中。
“那个买家是要倒卖，已经挂到他的店铺里面了，真黑心，竟然卖20个金币一份。@卖药的来了”
黎诺一愣，点开了评论里面的链接，上面的商品图与黎诺之前的一致，ID也与购买人一致，黎诺顿时有些无奈，在自己的店铺商品评论区留言说自己以后会记得设置限购。
回到家，黎诺忍不住拿出了雌父给他留的药方，他做的龙牙补血液和原液都是增益辅助类，还没有尝试过真正立刻见效的药剂，他决定尝试制作一种，让“失传”药剂重出市场。
考虑到现在刚是初春，很多人尤其是没有精神力的人会感冒，黎诺选择了发散风寒药作为下一次店铺上新的药品。
他列出了几种药植清单发给了弗宾洛，没过十分钟，药植已经送到了手中。
黎诺将苏芥枝握在手中仔细端详了一会儿。苏芥枝是二级药植，根茎粗长，叶子又圆又小，味苦，辛辣，根据雌父留下的药方，黎诺将粗粗的灵植拿剪刀细细剪碎，泡进了诱导液中。
紧接着，黎诺拿出了另一株药植——未青叶。
未青叶与苏芥枝完全相反，叶子硕大，举起来一片就可以遮住头顶的太阳，黎诺将未青叶放在一个大的培养皿中铺开，随后在里面放了糖晶，加了水一片一片摞起来。
两份药植一同隔水加热，随着诱导液逐渐变成土黄色，培养皿中也慢慢渗透出绿色的析出液。
一个小时后，两份液体呈现在了书桌上，黎诺慢慢释放精神力，让两份液体慢慢汇入到同一个器皿中。
在精神力的引导下，两份液体互相融合，并且在吸收了精神力后发出了微微的亮光，黎诺心中稳了下来，耐心的等着所有的液体都流出融合。
融合液的颜色逐渐变浅的同时，液体中间逐渐出现了一些亮黄色的小晶体，并且随着时间推移越来越多，等到黎诺全部弄完，整个液体变成了淡黄色，器皿的最底下，沉落了薄薄一层亮黄色。
黎诺照例吸取了一滴进行检测，3级驱寒剂，含毒量16%，已经是很低的毒量了。
黎诺快速将药剂倒进锁鲜机中进行封装，他需要趁着原液拍卖结束前上架，这样就可以让众多的买家知道他的店里有其他的中高级药剂产品，可以关注收藏！
很久没有批量上架产品的黎诺已经有些不熟悉界面，他翻找了半天，才找到了交易小助手，这个工具可以通过药剂的等级、功效、含毒量等与市场上的药剂进行比对，提供一个合理的价格区间。
很快，小助手便提供了一个市场价格：100金币一支。
不算很贵，可以接受。黎诺从锁鲜机中取出已经封装好的药剂，一口气上架了300支并且设置了限购。
锁鲜机还在工作着，两个药植分开出了后融合在一起就变成了预期的双倍的量，足足有上千只试剂，一时半会还真的要封装一段时间。
设置好后，店铺的首页在黎诺的设置下弹出了提示横幅，关注的新原液的近百万人都看到了横幅。
一大半的人看到消息便点进了驱寒剂界面，有钱的已经毫不犹豫的下了单。
原液还没拍出去，300支驱寒剂已经抢购一空。
下面的评论区瞬间热闹了起来。
“我也不是制药师，就是上来看看新原液抢购凑个热闹，没想到抢到了支这么便宜的驱寒剂，寒冬不涨价，良心店家啊。”
“这是药师自己开的店铺吧？这么刚，原液药剂都卖，6666”
“没抢到啊，我后悔了，店主还有吗？”
……
还有三分钟，原液竞拍就要结束了。
整理好从锁鲜机取出来的药剂，黎诺点开店铺准备和大家一起目睹竞拍的最终价格，他点击了一次刷新，店铺空空如也。
星网界面卡住了？黎诺疑惑的点了点刷新按钮。
界面突然弹出消息，“原液已经拍出，两百万四十一万三千星币！”
黎诺有些疑惑，咦？没卡吗？
