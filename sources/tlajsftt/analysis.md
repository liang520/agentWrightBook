# 原作结构分析：谈恋爱就是犯天条

## 基本信息
- 作者：鱿不右
- 平台：晋江（2023-05-14完结）
- 总章数：54章
- 总字数：206,097字（统一口径）
- 平均每章：3,816字
- 题材：仙侠（实际为女频无CP仙侠）
- 标签：前世今生、天之骄子、仙侠修真、打脸

## 核心设定
- **主角**：上古神女青厌，三万年沉睡后苏醒
- **核心能力**：万千化身（分身术），可化为男女老少动植物
- **核心矛盾**：天条被篡改、执著之物（凌崖）暗中操控三界
- **叙事特色**：多线并行+伪群像（所有化身都是同一人）

## 三条叙事线
1. **天界线**（青厌/万舟）：潜伏调查→发现阴谋→推翻天帝→登天位改革
2. **人间线**（晴烟/青岩等化身）：寻找玄薇转世→惩治堕落仙人→对抗执著之物
3. **支线群**（花蕖线/财神线/挽柳线/长夜茉娘线）：展现人间众生百态

## 主线弧线（起承转合）
1. **【起】第1-5章**：神女苏醒→发现天条被改→得知三界浩劫真相→分身入世
2. **【承】第6-22章**：元黎山线（萱薇之死→魔鳐追踪→长生城事件→银镜入梦）
3. **【转】第23-33章**：执著之物正面交锋→推翻天帝→天界改革→清尘道心转变
4. **【承二】第34-42章**：人间救世→海神堕魔→花蕖线→执著之物反击
5. **【合】第43-54章**：凌崖身份揭露→上古真相→魔族度化→终局大梦
