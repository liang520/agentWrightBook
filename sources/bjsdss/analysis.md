# 原作分析：霸剑十大杀手

## 基本信息
- 作者：刘定坚
- 题材：武侠
- 总章数：38章
- 总字数：92077字
- 结构特征：十大杀手各有独立故事弧（2-6章），由雄霸、君子旗两个贯穿角色串联

## 逐章标注

### 故事弧一：拳王·狗发（第001-002章，约11100字）

第001章：【开局引入】贫贱少年狗发为妹争"狗口饭"，以超人意志淹死野狗
- 章节功能: 人物塑造
- 核心事件: 狗发在"全香居"以咬舌激发痛觉突破极限，淹死野狗赢得狗饭喂养妹子，揭露母亲已死、曾遭绿脸马贼凌辱致死
- 可压缩度: 不可压缩
- 依赖关系: 被第002章依赖（狗发身世、仇人绿脸马贼、妹子狗俏、夫子机缘）
- 关键角色: 狗发、狗俏、臭黑皮、焦大嘴、饭天赐、胡不来（夫子）、绿脸马贼
- 伏笔: 埋设（夫子关注狗发；绿脸马贼身份；母亲之死）

第002章：【真相揭露】胡不来即绿脸马贼，狗发吞七神参杀敌，妹子惨死
- 章节功能: 主线推进
- 核心事件: 胡不来暴露绿脸马贼真身欲侵犯狗俏，狗发吞七神参爆发杀敌，妹子惨死，惨绿丑脸融合己身；狗发习皇拳成为杀手
- 可压缩度: 不可压缩
- 依赖关系: 无（本弧完结）
- 关键角色: 狗发、狗俏、胡不来/绿脸马贼、八万、古惑、苦海、老疯、老云
- 伏笔: 回收（绿脸马贼身份）；埋设（皇拳·御剑·圣之刀体系；十大杀手排名）
