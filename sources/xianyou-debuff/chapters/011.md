# 第十一章 “载体”

清沂环顾四周，数了数别墅的多少。
一栋别墅里有四间房子。阁楼一间，二层两间，一层一间。一共有五栋，那么就有二十间房子。因为是之前的时间，除了开头献祭的勇士还没有人死亡，所以按照34人来算。
也就是说，按两人一间来算，有人住的一共是17间，三间空房。
但因为人数是偶数，而他单独住一间，所以必定还有一个人落单。也就是说，实际有人住的房间共有18间，两间空房。
按理说，这栋别墅里除清沂外还有一人，但清沂昨天回来的时候并没有看到别墅里有其他住户。
清沂揉了揉太阳穴，坐在床沿上开始复盘。
首先，昨晚的一切绝对不是梦境。痛感、伤疤都是真实的。但为什么这里的一切都被复原？
清沂想到了昨晚抱着他的那个人。
自己左肩上的伤口莫名被治愈，恐怕也是他的手笔。
昨晚被他抱着的时候，清沂的意识几近昏迷，但他记住了那枚十字架胸针上刻着的字。
【S级系统，现任北区执行官。】
系统……如果他没猜错，这位系统，恐怕就是江黎歌的上司。
至于王富贵，恐怕是那位小姐给自己上司取的外号。
按理说，系统应该与玩家处于对立面，但为什么自己遇到的这些系统，一个个脑子都有坑，来保护一个只想躺平的咸鱼。
这个人身上疑点太多，最大的谜团就是对方保护自己的动机。
总之，非常可疑。
但如果有机会见面，不管对方出于什么动机，清沂还是得当面道个谢。
抛开这点不谈，清沂决定重点梳理一下昨晚与黑影周旋时看到的回忆。回忆的主人基本可以确定，就是塞斯兄弟两。只不过……清沂没想到，塞西瑞浮居然跟他是同志。
清沂，标准理工IT男，一条兢兢业业摆烂二十四年的咸鱼，在今天，一个恐怖游戏副本里，找到了同志。
不过今后再见到那位神父，清沂就有了一张底牌。
这里的游戏副本，不得不说，真的很奇怪。
回归正题。窗花玻璃上的背影是塞西芙丝，那么后来玻璃上芙丝头部碎裂，也代表了她惨遭抹杀。
在西方宗教里，最常见的神明形象就是一颗插着白羽翅膀的眼球。如果花窗外那颗散发着恶臭的眼球就是所谓的“神明”，那么很明显，这一切的一切都是一场邪教与贵族联手，用来压迫百姓的巨大骗局罢了。
而塞西芙丝，作为这个邪教里最有名的修女，看似被所有民众尊敬，但只要有一个人察觉到了不对，他们最先攻击的对象就是这个修女。
她自以为做着光荣正义的事，但事实上，可怜的塞西芙丝只是资本压迫的牺牲品，灾祸降临时最好用的挡箭牌罢了。
结合之前走廊外的花窗玻璃所歌颂的故事，不难猜出，塞西瑞浮、塞西瑞仪就是被修女塞西芙丝所领养的双生子。
而这对双生子就是底层清醒人民的代表。弟弟一心想的就是破除邪教，拯救百姓。但哥哥却日复一日地认真祷告，仿佛真的是一个虔诚地信徒。
弟弟塞西瑞浮有很多先天性疾病，因此一直身体不好。结合最后一段回忆，清沂合理猜测，哥哥的身体里也藏着和弟弟一样的反骨。他之所以一直祷告，只是为了从“神明”那里求来一个机会。
一个让他的弟弟重获新生的机会。
然而弟弟不理解哥哥，只是觉得哥哥在一味地逃避。
但这件事仍然有一个疑点。
既然“神明”不是真的，那塞西瑞仪的阵法、塞西芙丝的恋戒，都是谁给的？
还是说，那颗眼球只是一个“载体”，“神明”来到人世的载体。
又或者，它只是一个神明的复制品。而塞西一家见到的神明，是正版。
双生子自以为是地认为，百姓所有的灾祸都是因为修女，也就是他们的养母塞西芙丝引起。
因此，他们手刃了自己的养母，亲手杀了那个最爱他们的人。
作为挡箭牌的塞西芙丝，被榨干了她作为修女的最后一丝作用。
从回忆可以看出，双生子最终认识到了自己的错误，但一切已无法挽回。二人为了赎罪，接替了塞西芙丝的职位，成为“神父”。
因为塞西瑞仪失去了与弟弟有关的记忆，所以也不再记得自己做祷告的初衷，才会有之前在教堂里他一心聆听民众忏悔的那一幕。
而“神父”塞西瑞浮所要忏悔的事，就是自己杀死了养母。
现在的疑点可以分为三大块。
第一，神明到底是真是假。
第二，为什么会出现这么多不同的自己，并且每一个自己都身处不同的时空节点。
第三，那位无聊的系统先生，到底想干什么。
清沂换上衬衫走到楼下，瞥了一眼其他房间。果然，另外三间都没有人居住的痕迹。一楼客厅有一张圆形餐桌，上面摆着牛奶面包等西式早餐，看起来很是丰盛。
清沂微微蹙眉，盯着桌上被管家放下的两副碗筷。
果然，这里有两人没错。但为什么另一人至今没有出来？是没醒……还是他昨天就没有回来。
清沂不确定另一位落单的人会不会经历和自己一样的遭遇。
以及，在自己的时空中，第一晚就莫名死亡的两人……
按理说，除去死掉的怨种大哥，以及滞留在教堂里的自己和江黎歌，剩下的人数也是偶数才对，不应该有问题。
清沂随意拿起一片面包叼在嘴里，没有人在，他就又恢复了那副懒洋洋地样子，走到一楼的房间前，敲了敲门。
没人回应。
“打扰了。”清沂推门而入。
虽然他已经做好了心理准备，但看到眼前的一幕，胃里还是如同翻山倒海一般泛起阵阵恶心。
扑面而来的，是一股浓厚的恶臭味。
一个已经被砍得看不出性别的人倒在血泊里。头部和四肢都被剁下，横七竖八地倒在房间的各个角落。躯干躺在毛绒地毯的正中央，肚子里白花花的肠子早已流了出来，各种内脏碎片、肉沫糊在房间里的每一角落。
清沂扭头，忍住自己想要呕吐的欲望，默默把嘴里叼着的面包放回盘里。
那人的胸膛微微隆起，十分不自然，就好像里面还有什么东西在不停地蠕动。清沂随手拿起一副筷子，把口鼻埋在衬衫的领子里，走进了房间。
他绕开血泊，缓缓地蹲下来，用筷子从肚皮的缺口处插进去，掀起胸膛上方薄薄的一层皮肤。霎时间，密密麻麻的蛆虫从胸腔里爬出，迅速隐秘到房间的各个角落。
蛆虫爬出后，那人的胸膛肉眼可见地瘪了下来。清沂向内部瞥了一眼，几乎所有的血肉、血管都被那些蛆虫啃食得干干净净。
在肋骨的地方还有几个没有长成的软体蛆虫，以及许多形似虫蛹的黄白色小点。肋骨上缠绕着黄色的凝胶物。
眼前的一幕让清沂响起了系统的提示。
【上看，下看，她看见地上躺着一个死人】
【从他的鼻子到下巴，蛆们爬出，蛆们蠕进】
这就是歌里说的“死人”吗？
与此同时，大门突然被粗暴地推开，门板撞在墙上发出巨大的声响。
不好，有人来了。
要是被人看见他和尸体共处一室，那麻烦可就大了。
清沂环顾四周，周围没什么可以躲的地方，房间虽然精致但设施很少，只有一个衣柜和一张床铺。
值得一提的是，这座房间里也有一扇一模一样的花窗玻璃。而修女头部那块的玻璃莫名消失了。
果然。如果昨晚清沂没有挺过去，那么就会和这位哥们一个下场。
清沂定了定神，钻进了床底。
出乎意料的是，床底部分的地毯异常干净，完全没有被血液侵蚀。而且蛆虫们似乎都在惧怕这一块地方，没有虫子往床底跑。
很快，清沂就知道了原因。
床底也躺着一具尸体。
一具，跟他长得一模一样的尸体。
这已经是清沂见过的第三个“自己”。所以这次，他没有过多的不适感。清沂借着微弱的灯光看向尸体。他穿着洁白的教服，左耳带着金色太阳图腾耳骨夹，手中捧着一束百合。身上没有明显的伤疤，只有脖颈处有一道触目惊心的青紫色勒痕。
“这里没人住吗？奇怪，那个落单的小子呢？”门口传来一个熟悉的声音，是钟于。
床底光线很暗，清沂轻手轻脚地在自己的尸体上摸索，终于在尸体的左脚脚踝处摸到了一根绳被牢牢绑在床头的绳子。
“先不管这些，我们快去收尸吧，完成任务才是要紧事。”隔着门，清沂隐隐约约听到一个中年妇女的声音。
之前在墓地里，他们说过，只有在休息场地完成任务才能获得积分。而钟于这一组的任务，是收尸？
收谁的尸？
“餐具怎么被人动过？”外面的人好像在翻找着什么。
清沂来不及想这些，从床底爬出来快速把自己尸体上绑着的麻绳解开。尸体被清沂从床底拖了出来，胳膊碰到床腿发出碰撞声。
“谁？”
钟于猛地转身，推开了一楼房间的门。
“呕……”钟于往里看了一眼，胃里瞬间一阵翻江倒海，趴在桌沿上止不住地呕吐起来。
房间里空空荡荡，只剩下一具四分五裂的尸体。
