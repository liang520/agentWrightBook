# 第八章 为什么这个系统里还有消音啊

“我对你的兴趣又加深了。”“塞西瑞仪”又在说些莫名其妙的话，“这才**了五次，你似乎就已经意识到了什么。”
“塞西瑞仪”的话里有一个单词被系统用“哔——”的电磁声取代了。
“本来我以为他跟我打赌，他毫无胜算呢。”
“现在看来，是我必输无疑了。”“塞西瑞仪”的语句里看不出一丝失望，反而满满的都是期待。
“唔，我想想，这会不会是因为你的意识体格外强大呢。”
“你到底想说什么？”清沂冷着脸，沉声说道。
“对了，你可以叫我塞西瑞浮。如果不是为了让那帮人昏迷，我可不愿意用哥哥的名字。”
“我只想看看，作为**的你，会有些什么令人兴奋的表现呢。”塞西瑞浮慢条斯理地戴上手套，对着玻璃做了一个奇怪的手势。
又被消音了。
转瞬之间，花窗玻璃被震碎，大大小小的玻璃碎片四散开来，原本被块块玻璃割的斑驳陆离的阳光瞬间消逝。清沂下意识地侧身，正好躲开一块直直向他刺去的玻璃碎片。
如果他没有提前躲开，那这块玻璃就会直接从他的胸口穿透。
塞西瑞浮懒洋洋地趴在十字架上，似乎对他接下来的行动很感兴趣。
清沂定了定神，走到花窗玻璃外的世界。
那是一片熟悉的坟地。
裸露的尸骨随意地堆成一堆，每一堆近乎于皎白的骨架旁，都有一捧用蓝色丝绸系起来的百合花。
远处起了一阵微风，不知从哪吹来一朵又一朵红色的花瓣，落在这片洁白的大地上，仿佛雪中红梅。
尸骨的最中间，是一具由昂贵玉石雕刻而成的棺材。
清沂走进了些，推开沉重的棺材盖，瞬间愣在了原地。
棺材里躺着的，是他自己。
青年面容清冷，五官深邃立体。他的双眼被一条三指宽的白色蕾丝布条蒙住，顺着黑发散落在一边。左耳带着一枚金色太阳图腾耳骨夹衬得青年皮肤更加苍白。
一头黑色的长发不再被红色丝带系着，而是随意地散落在肩头。‘清沂’身上穿的不是衬衫，而是换上了一身白色的教服。一双骨节分明的手交叠在胸口，左手带着看起来十分昂贵的恋戒。
他睡在碎花落叶之中。
虽然清沂承认，自己长得确实很好看，但看到自己睡在棺材里的样子着实诡异。
【清沂，不要看】
一个类似于系统的声音莫名其妙地出现在他的脑海里。
那声音带着成年男性特有的磁性，语调低沉，掺杂着一丝人工合成的电子杂音。
但不知道为什么，清沂早就无法移开视线了。
微风轻拂，一朵被鲜血染红的白百合在空中摇晃，最后，恰好落到青年的眉间。
转瞬之间，清沂的大脑就像被入侵一般，一片空白，许许多多零碎的记忆涌入他的脑海。
清沂什么都想起来了。
他不知道为什么重新经历了一次过去的时间，但，排行榜上的变化着实匪夷所思。
【三号复制体，模拟实验开始】
当他再次清醒过来的时候，自己已经不知道被谁关到了一个狭小黑暗的空间里。清沂尝试弓起腿，刚好抵到了空间的顶部。
现在，清沂有理由怀疑自己穿到了棺材里那个‘清沂’的身体里。
清沂摸了摸左耳，一片空荡荡。随即，他揉了揉自己衣服的布料，令他意外的是，他身上穿的是廉价的白衬衫，而不是教服。
周围的空气很稀薄，如果不采取措施，过不了多久，他就会窒息而亡。
清沂放缓自己的呼吸，微微拱起背，伸手向自己的背后探去。他扒开底层铺着的散发着清香的百合花，摸到了一道长约10厘米的裂缝。
【清沂，往右靠】
那个声音又出现了。
清沂向来是个极度叛逆的人，别人叫他干什么，他偏要反过来。但不得不承认的是，自己居然很心甘情愿地听脑海里，那个陌生人的话。
更何况他本来就打算这么做。
清沂缓缓侧身，背靠在棺材的右侧。
几乎是在他转身的下一秒，裂缝处突然照进一抹阳光。紧接着，一把锋利的匕首带着一股浓厚的血腥味猛地从裂缝里猛地刺出来，又缩了回去，就这样反反复复刺了十几次。
清沂借着光，眯了眯眼。
那把匕首的刀柄由金子铸成，上面镶嵌着一颗玻璃制成的眼球，以及由水晶拼成的棱角分明的白百合。刀刃上有了些许铁锈，刀尖带着残留的肉末，以及让人匪夷所思的各种内脏碎片。
清沂：“……”
卧槽……
自己要是还躺在原位，就彻底凉透了。
“你是谁？”他尝试与脑海里的声音沟通。
然而并没有人回答他。
匕首还在不停地补刀。清沂就在旁边看着它，百无聊赖地数着次数。在匕首整整刺了12次后，它完全穿了进来，在空中停滞了一会。
此时不反杀更待何时。
清沂握住刀柄，刀尖朝下往缝隙里刺去。裂缝中顿时喷射出猩红的液体，染红了满地的百合。他用力握住匕首往下移动，在棺材下的不明物体被划开划开一道口子，像一个球体似的渐渐干瘪下去，光芒也逐渐消逝。
忽然间，清沂手中的匕首就像是活了一般，玻璃眼球四处转动，挣脱他的舒束缚，将自身横了过来，刚好与裂缝的形状完全契合。
他尝试拿起匕首，却发现匕首好像被黏在了底部，牢牢地卡在缝隙里。
“你怎么不解释解释？”棺材外传来一个少女的声音，那声音莫名很熟悉，可能就是清沂要找的人。
但下一秒，又一个熟悉的声音，让清沂浑身发冷。
“他们爱怎么想怎么想，我不在乎。”
青年的声音清冷，语气平淡，尾音上挑。
这是他自己的声音。
如果自己和棺材里的‘清沂’灵魂互换，那么棺材外面的那个是谁？
清沂突然发现，自己进入了一个思维盲区。
有没有一种可能，这并不是灵魂互换，而是时空互换。
而现在，如果清沂的记忆没有问题，棺材外的自己，现在要开棺了。
果然，上方传来一阵石料被推动的声音，一丝光亮从清沂的上方照了进来。
【0911号初始副本，时间暂停】
脑海里那个神秘的声音再次出现，清沂瞬间浑身僵硬。
上方，石料摩擦的声音停止，光亮也停止了扩张。
棺材外，原本喧嚣的人群都像被定住一般，万籁无声。
【玩家清沂，权限开放】
话音刚落，身体的控制权重新回到清沂手中，属于清沂的时间开始流动。棺材的背后传来机械运动的声音，紧接着，石料裂成两块，向地底坠落。
满棺材的百合跟着清沂一起，坠入未知的深渊。
“清沂？清沂！”江黎歌捧着从白骨堆里找到的百合花，担心地看着清沂。
清沂皱了皱眉，只感觉自己的脑袋如被绞肉机绞碎一般，疼如钻心。
眼前模糊的景象渐渐清晰，一个穿着黑粉JK的少女站在他的面前。
少女长着江黎歌的脸庞，却又有一双蓝宝石般透亮的双眸。
“清沂，你怎么了？不会失忆了吧。”“江黎歌”忧心忡忡地看着清沂，伸手试了一下他的额头，“也没发烧啊。”
“我们认识吗？”清沂后退一步，躲开少女的手，警惕地看着她。
“我是江黎歌啊。”少女双手叉腰，看起来有些生气，“大哥，别装了，一点都不好玩。”
清沂依旧没有放松警惕，他摸了摸左耳，耳骨夹不在了。果然，他身上穿着的不是教士服，而是自己被修女的血染红的白衬衫。
自称“江黎歌”的少女手中捧着一朵百合，百合原本洁白无瑕的花瓣上，沾着鲜红的血液。
“你不觉得这朵花，比其他的都要好看吗。”“江黎歌”注意到清沂定在百合上的眼神，笑了笑。
清沂忽然觉得墓地似乎过于安静了一些。他环顾四周，但周围只有他和“江黎歌”两人。
清沂的兴趣久违地被勾起来了。
“当然。”清沂笑了笑，卸下了防备。
“那我们一起把它贡献给神明吧，相信神明大人一定会喜欢的。”江黎歌也笑了，她虔诚地把那朵百合举过头顶，虔诚地闭上双眼。
【清沂，趁现在】
“啰嗦。”清沂一记手刀砍向“江黎歌”的后颈，少女双腿一软，瞬间晕了过去。
眼前这人大概率不是江黎歌，而是塞西芙丝。
首先，塞西芙丝有着一双湛蓝的眼眸。其次，江黎歌的尾音上扬，而清沂在棺材里看到的幻想中，塞西芙丝的语调是很低沉的。
塞西芙丝的语气简直就和眼前的“江黎歌”一模一样。
他打横抱起少女，向着远处并没有消失的教堂跑去。
【你不能背她吗，非要抱着？】
清沂气笑了，“因为方便。”
这系统怎么回事，为什么茶里茶气。
“妄悠，你怎么犯规啊。”
塞西瑞浮站在圣台后，居高临下地看着站在教堂玻璃窗外的清沂。
妄悠……是谁？
【犯规也是一种实力】
这次，系统男声直接在大厅里响起，而不是在他的脑海里。
他就叫妄悠？
【更何况，没有我，他照样能赢】
“霍，你很信任他嘛。”塞西瑞浮慢悠悠地下了台，他的身影逐渐虚化，然后消失在教堂里。
【当然】
教堂的正中央摆着一具空棺材，棺材里铺满了被鲜血染红后的百合花。
清沂抱着“江黎歌”走到棺材前，把她轻轻放在花堆里。
似乎还少了某样东西。
清沂四下寻找着，终于在圣台上看到了一串恋戒。他将恋戒戴在“江黎歌”的左手，把她手里的百合放在她的眉心。
转瞬之间，少女身上的衣服虚化成了黑白的修女服，一头黑发由上到下见见变成金黄色，体型也有所增长。待到完全恢复后，塞西芙丝的身体又开始迅速溃烂，最终只剩下一具纯白的骨架。
