# 第42章 回宗，取藕

“道友，请自重，搂搂抱抱成何体统！”
赵无极尴尬至极，他努力挣扎着，想推开楚荣，哪知楚荣手如铁箍，他竟是不能推动分毫。
他叹了口气后就认命了，这是渡劫巅峰的大修士，我一个中期修为自然无法推开的。
楚荣熊抱了一会，才松开双手。
他看着周围天剑宗几千号人近乎呆滞的眼神，他笑 道：
“对不起哈，换了一副躯体，你们不认得我也属正常。”
“我是楚荣呀，在下界蓝星借体复活过来，又杀上了灵星。”
赵无极瞪大双眼：
“你就是小徒楚荣？不可能吧？他命牌都碎掉了。”
他一把抓住他的手，仔细端详起来。
他鼻子发酸，竟有些想哭。
周围的人也好一阵缓过来，
楚荣笑呵呵道：
“师傅，徒儿这次大难不死，必有后福。您看，这不走的时候一个人，来得时候几十万大军给您拉回来了。”
“以后，我们天剑宗就是灵星最大的宗门，徒儿可没给您掉链子呀。”
赵无极悄悄传音给楚荣，问了只有他俩知道的隐秘。
楚荣一一答上，赵无极这才完全相信。
楚荣又向三位太上长老行了礼，又让身后各大宗门扛把们过来认识了一番。
当然，核心的风玉儿也怯生生过来见过赵无极。
火凤则大咧咧过来叫了赵无极一声“前辈”后，走到楚荣身后，一脸傲娇。
三位太上长老脸上一直笑容不断，天剑宗这下可捡着宝了。
其他宗内修士更是喜出望外，有了这位强横的天剑宗弟子，自己在宗内的前景自然是一片光明。
同时，那位当初保护不力的小师叔也是心头一松，这小子总算没事，不然一生的心魔都不会解开。
楚荣吩咐其他黑舟上的那些大佬：
“你们先在外面找个地方玩儿吧，我先进去和师祖和师傅说点事。”
其他人恭声答是，这把赵无极惊得眼珠子直转。
“徒儿，这里面可是有不少大修士呢，这样弄在外面不会滋生不满 吧？”
楚荣不以为然，摆手道：
“ 不会，他们可乖着呢，谁要造次，我自然有办法说服他。”
赵无极内心苦笑，这小子，是靠实力说服人家吧。
师徒二人走向宗门深处，三位太上长老则各回洞府。
楚荣来到赵无极修炼的洞府中，设下重重禁制后，这才言简意赅把自己不慎到蓝星，又在蓝星借体凝魂，喜获小塔获得九天仙法的事说了出来。
同时，蓝星上的重要之事也都透露了许多。
赵无极边听边称奇，这小子还真是走了狗屎运。
关键是渡过天劫，成为预备仙人，实在让他也是羡慕得很。
楚荣这才提起重塑肉身的事情，毕竟老是用张贤的身体也很别扭。
赵无极沉思一会，突然眼睛一亮：
“据我所知，东域的超级大宗玉女崖有株万年莲藕，只要一小段，就能重塑肉身，而且重塑过的肉身灵根极佳，毫无杂质。”
“只是此物价值连城，不知玉女崖是否愿意拿出？”
楚荣眉头一挑：
“现在灵盟势大，已成摧枯拉朽之势，他们应该不会拒绝吧。”
赵无极摇摇头：
“只怕没这么容易，玉女门底蕴深厚，传说有老祖中飞升仙界，留下不少传承。”
“如果硬碰硬，胜负还真不好说。”
楚荣嗯嗯的点着头，一边琢磨着得赶快去一趟玉女崖。
如今，只有东域还没平定下来。
那些宗门他都已送了信，不听话也要一一拔除，以绝后患。
师傅小聚一会儿，楚荣就立即调出一艘黑舟，直奔几个不老实的宗门而去。
同时，留下一艘黑舟守护天剑宗。
很快，那几个不识好歹的宗门被灭。
不多时，楚荣出现在玉女崖，轻松破开阵法，溜到崖中核心区，一个叫迎仙崖的地方。
据说，那崖上一处池水中，就养育着神奇的莲藕。
这玉女崖一片平静，也已公开承认灵盟的地位。
因此，楚荣对此宗门还是很有好感的。
迎仙崖一处清泓中，鱼虾嬉戏，鸟雀纷飞，十余朵莲花开得正艳。
楚荣来到池边，看着这些莲藕不由双眼一亮。
莲花下应该就是那珍贵无比的莲耦了，四处没人，要不我顺手拿走算了。
他正意动，未曾想池中泛起一阵光阵，将那莲花护得严严实实。
楚荣运足灵力一击，光阵塌下大片，但却又恢复如初。
他咦了一声，这阵法一时半刻还解不开。
再说，蛮力破开，那莲藕岂不是毁了？他一阵踌躇。
他击阵的声音，自然传得老远。
不久，空中飞来几条倩影，落下后一阵怒吼：
“何方歹人，竟敢盗取珍宝？”
来者是一中年美妇，两个美得让人不敢直视的少女。
少女长得一模一样，应该是双胞胎姐妹。
她们三人正欲攻击，楚荣不想节外生枝，拱手笑道：
“三位，先别动手，我是灵盟的楚荣！”
一听灵盟，再看这青年深不可测的修为。
三个女人一惊，为首的美妇拱手道：
“妾身是玉女门门主南宫泉，不知盟主大人前来，有何贵干？若有所差谴，尽管吩咐就是。”
楚荣看到南宫泉如此恭谨，真心有些想不到，他也客气道：
“是这样，南宫崖主，我有事想求，想获这莲藕一节，不知方便否？”
“当然，我不是强取豪夺，你尽可以提条件，我也尽量满足。”
南宫艳道：
“这莲藕确实珍贵无比，您是大修士，我们也不好提什么条件，只求盟主一件事就行。”
楚荣一听，爽朗道：
“崖主，你尽管说好了。”
南宫艳沉吟几秒，手指了两位绝色少女，道：
“妾身想让盟主带走她们，成为道侣，也给她们找个好前程。”
“她们灵根极佳，有您的指导，实力未来不下于我的。”
楚荣一听，牙痛道：
“南宫崖主，这恐怕不好吧？”
他从未想过找道侣之事，况且那风玉儿对他情意浓浓，如果领对双胞胎回去，风姑娘还不得气结。
南宫艳却一脸坚定，咬牙道：
“盟主，您要不同意，我宁愿炸碎这莲藕，也不能给你分毫。”
说完，她发动禁制，池中竟悬挂了一片小剑，对准那几片莲花。
意思很明白，楚荣不答应，她就要毁掉。
楚荣一听，有些傻眼，如此刚烈，如果不收下她们，还真是不行了。
唉，为了重塑肉体，只好暂且答应。
“好，我先答应你，以后我也会多多照拂你们宗门。”
南宫艳大喜，立马拉着姐妹俩给楚荣行礼。
随后，她以下禁制，朝那池中注入灵力。
不久，一截粉嫩的莲藕就出现在她掌心。
