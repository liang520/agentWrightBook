# 第29章 神秘的女子

众人大骇，此子果然不用出手，仅凭一股酒箭就重伤陈护法和十大金刚。
一时间，众人惊惧不已。
陈护法等人的兵刃被酒箭打穿成筛子，真气护体也被撕开，一身衣衫几乎震碎，肌肤留有密密麻麻的血坑。
他们虽侥幸保住性命，但也负伤极重，只怕要休养很久才能康复。
“都是人族，我已经留手了，不然你们已经归天了。”
楚荣漫不经心的话，让众人心生畏惧。
陈护法将嘴角流出的鲜血用衣袖揩去，沉声道：
“阁下好功夫，我们愿赌服输，接受三个条件。”
“只要不是太过分，我都可以代总盟主答应下来。”
陈护法多年老江湖，在实力为尊的当下，说再多也没用，还不如痛快认输，免得招惹这尊煞星，带来更可怕的后果。
楚荣点点头，轻笑道：
“算你这个牛鼻子识相，好吧，回到屋里继续喝着吃着，我再说说三个条件。”
“这三个条件不苛刻，能与我谈条件已经是你们的福缘了。”
他这话不假，世俗修武与修仙搭上联系，还真是抱上了粗腿一般。
楚荣背着手，踱着方步前往餐厅。
后面，左盟主等人急忙和其他武者将陈护法受伤者搀扶起来，及时服下金创药，以及处理外伤。
十大金刚冷清着脸，将那十口巨剑扔到一旁。
陈护法也将透风的软剑扔掉，一脸痛惜。
武盟多少年没遭遇这样的惨败了，哪怕是蓝星武道竞争对手的欧国武盟、俄国武盟、樱国武盟等，虽然与他们长期明争暗斗，各有胜负，但也从未如此大的溃败。
这，完全是单方的碾压，完全没有对抗之力。
关键更郁闷的是，对方仿佛未尽全力。
只怕总盟主对上这位，也不敢说全身而退呀。
陈护法叹着气，在众人的搀扶下，走向餐厅。
“但愿他提的条件温和一些吧，不然大炎武盟彻底没面了。”
众人一声不吭，心中百味杂陈。
心中都在想，这人到底师承于谁？竟有如此手段。
楚荣正在餐桌边坐着，专心享受着美食美酒。
他看到众人如残兵败将走进来，有些忍俊不禁：
“都来了，闲话少说，我说三个条件吧。”
“第一，把秦氏集团无偿转让给张家，让我妹妹张爽任董事长，至于如何合法操作是你们的事情了。同时，要保证张家的安全，出啥事要拿你们是问。”
“当然，秦家明智退让的话，可保他们一族一生安全。”
“第二，大炎国武盟以后听命于我，平常不会干预你们任何事，但我有指令必须完成。”
“第三，也是最重要的，帮我收集蓝星各种修炼者怪异情况，不是武者，而是修仙者。”
楚荣提出这三个条件，也是深思熟虑的。
家产转到张家，保这几代人一生富贵平安，算是他借用张贤身体的补偿。
武盟听命于他，是想将这个大组织掌握在手中，方便在蓝星上办些事情。
第三个条件也是他最看重的，想在飞升仙界之前将蓝星上的修仙者情况打听清楚。
虽说灵气几乎全无，但不排除有传承，不排除有上界修士下凡。
更让他惦记的是，那枚怪异鱼鳞的来历，属于妖修还是魔族？
必须要搞清楚，不然他心神难安。
听到楚荣说出的这几个条件，众人脸色一变再变，这哪是条件，完全是上位者的直接命令。
陈护法有些苦涩，苦笑道：
“大师，这些条件我还得向总盟主汇报一下，在下可不敢拍板呀。”
楚荣淡淡道：
“那给你们十分钟时间商量，希望结果令我满意。”
陈护法心中一凛，连忙拱手言是。
他带着十大金刚到偏厅去请示去了。
京都，郊外，一处环形山脉中，在风景如画的谷中，有几间江南 风格的住宅。
住宅中一部卫星电话响起，一位身着青色汉服的绝美女子正在梳妆，如瀑的长发随轻风飘着，犹如女仙一般。
她伸出纤纤玉手，按下电话。
“陈护法，事情办得如何？”
电话那头传来焦急的声音：
“总盟主，对不起，属下没把事情办好，我甘愿受罚。”
她心里一动，但面上平静，道：
“竟然你们出马都搞不定，看来此人还真是硬茬。”
她沉吟一下，道：
“不着急，且把事情说来我听。
陈护法见总盟主语气还好，心下稍安，连忙把事情经过详细讲了一遍。
女子面色微变，随后归于正常。
她手握电话，转身朝小院外走去。
来到门前小溪旁，将一对玉足浸泡其中。
用不容置疑的语气对着陈护法道：
“陈护法，全都答应他吧。”
陈护法在那头整个人都惊了，什么，总盟主竟然服软了，说答应就答应了？
总盟主从修炼伊始到竞争总盟主，那种杀伐果断他们可是见识过的。
现在，居然轻易答应对方，实在让他有些憋屈。
“总盟主，您老人家不再考虑考虑？”
女子嫣然一笑：
“我考虑好了。对了，再提醒你们一次，不要喊我老人家，我还年轻。”
陈护法一听，吓得一激灵，道：
“好的，总盟主。”
卫星电话挂断后，女子喃喃道：
“呵呵，居然又是一个修仙者下凡，不知道什么目的，有空得把人家请来聊聊人生谈谈理想。”
她叹了口气，手中一翻，多了一片怪异鳞片，竟与楚荣那枚一模一样。
她冷笑道：
“看来蓝星也要起风起浪了。”
她玉手一抬，一股强烈的灵力喷薄而出，小溪的水被汇聚成一股白练朝那山崖直击而去。
轰地一声，峭壁上被轰成一个巨洞。
要是修仙者在此旁观，自然明白，此女子，大炎国武盟总盟主也是一位修仙者。
再说西渝省那头，陈护法已经将总盟主同意的指令说与楚荣听了。
楚荣一听，呵呵道：
“嗯，算你们总盟主识相 ，好吧，你们抓紧办吧，我有事先走了。”
他身形一动，像流星一样投射而出，转眼间没了人影。
众人眼前这种骇人身法，不由大惊失色。
“此人不是神仙，只怕也离神仙差不多了。”
