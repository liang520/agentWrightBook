# 第25章 秦家寻仇

“爸爸，这个坏人不会找你报仇吧？”
小虎、小翠进了屋，有些担心的问道。
楚荣摸了摸他们的头，呵呵笑道：
“你们放心，他们不敢找上门的。”
这样的小虾米，在渡劫巅峰、即将飞升的他来说，实在不值一提。
哪怕所谓生哥的背后有人，又能厉害到哪去？
唯一让他担心的是那片怪异鳞片，那可怕的气息竟有渡劫期的样子。
这也是他不愿立即飞升，留在蓝星查明原因的重要原因。
一定要把这背后的阴批揪出来。
楚荣一边指导小虎、小翠修炼，一边暗下决心。
另一边，在山阳县人民医院抢救室，肥肚男生哥正痛嚎着。
“生哥，你忍一下痛，马上进手术室给你做手术。”
“秦爷请了最好的省医院创伤专家赶来，手术肯定没问题的 。”
女人不停安慰着他。
肥肚男听到秦爷二字后，面色稍稍平缓一些，眼神透出阴寒。
“我不把这个家庭碎尸万段，我就不姓秦！”
女人一旁连连点头：
“对，凭生哥的手段，还有你的背景，灭他们分分钟！”
想到自己的爱犬被楚荣踢成肉泥，她心里仍然惊怒。
不一会儿，一个五旬左右、穿着唐装、手里搓着核桃的地中海男人带着一帮气势汹汹的走了进来。
“秦海，省城专家马上到了，你放心，我马上安排为你报仇。”
肥肚男看到地中海男人后，犹如见了救星。
这个地中海男人不是别人，正是被女人称为秦爷的人。
“叔叔，你一定要为我做主呀。”
“对了，那两个小杂种的爸是个练家子，估计得武圣以上实力，您可一定要小心。”
秦爷点点头，随即吩咐一众平头开始行动。
有的通过关系查清楚荣的底细，有的联系肥肚男师承鹰爪门，有的去请省城的好手赶来山河县。
不久，肥肚男被推进手术室准备手术。
秦爷面色阴沉，想起肥肚男的惨状咬牙不已。
秦爷是省城一家上市大公司老板，全国人脉极广，财力雄厚。
肥肚男是他和嫂子私通所生，表面上喊叔侄关系，其实是亲生子。
之前，他老婆只生了四个女儿，肥肚男是他唯一一个儿子，极其宠爱。
由于他男人那方面出了问题，40岁后就再无生育能力。
如今，看到儿子在小小山河县被人伤成这样，心里着实怒火烧天。
不多时，有人将楚荣的情况报告了他。
秦爷陷入沉默，一个普通家庭竟然突然崛起，让他非常不解。
竟然在高档小区买了三套别墅，这还不算，一家子都是练家了。
连两个小孩都在武圣以上实力，这让他有些百思不得其解。
管它呢，先报仇再说。
楚家再强，在强大家庭面前还是渣渣一个。
为稳妥起见，他决定搬出另一座大山。
他弟弟秦汉间，西渝省武盟的副盟主，位高权重，据说半只脚已经踏入武帝境界。
武盟是管理运行全国武者的机构，大炎国武道极盛，人才辈出，武盟就是人人向往的武者殿堂。
这些年，秦家企业在商场上如鱼得水，有大半功劳来源于秦家有位高阶武者撑腰。
“二弟，我有事想请你办！”
电话那头传来爽朗笑声：
“大哥，你咋还对我这么客气，有事直说，只要能办的，还不是你一句话。”
“是这样的。。。。。。。”
秦爷把前因后果说了，目的很明确，加上二弟这个靠山，对付楚荣就稳当多了。
这位副盟主一听，沉声道：
“居然有不知死活的东西惹到我们秦家，必须要给他点颜色看看。”
“大哥，要死的还是要活的？”
秦爷想了想，道：
“砍掉他一只手，一只腿就行，我也不想把事情闹得太大。”
毕竟是法治社会，虽说上面一般不管江湖武者的争斗，但还是要顾全大局。
“好，我亲自带人过来！”
“让江湖上的人都知道，惹到我们秦家是多么愚蠢的事情！”
副盟主立马表态，语气十分自信。
秦爷挂下电话，焦急的在手术室前踱步。
虽说他是出了名的无情和奸滑，但对这个私生子还真是格外上心。
这天周六，楚荣带着张贤父母、妹妹张爽，小虎、小翠到山河县出名的凌云峰游玩。
张贤老婆王香这次赖着跟在一起，楚荣为了孩子也只能强忍鄙夷，让她跟着。
不过这王香也是明白人，尽量不招惹楚荣，只是一旁殷勤的跑跑颠颠，尽力表现。
看到她这个样子，张贤父母和妹妹倒是有些感叹，平日性格古怪乖张、强势的王香竟然有些转性。
“爸爸，还有好久到山上？”
看着蜿蜒曲长的山路，两个孩子有些焦急。
楚荣将神识放出，道：、
“大约还走两公里左右，就可到山顶了，到时我请你们吃酸辣粉和烤串。”
山上有几家小吃店，做得味道还算地道，生意还算不错。
一听山上有吃的，两个孩子兴奋起来。
一家人刚走到一处竹林处，突然楚荣示意大家停下脚步。
他感知到一里外有上百不速之客围攻过来，大多是武圣、武魂实力。
另外一人竟有半步武帝的境界，这在全国武道中属于一流水准了。
半步武帝已能实现真气外放化形的水平了，是属于绝大多数人仰望的存在。
“你们退到一边，让我来会会这些不速之客。”
张爽凑过来急道：
“大概来了多少人？”
“一百来人吧。”
这么多？
大家神色有些惊慌，楚荣再厉害，也不可能打过这么多人吧？
小虎、小翠一听，急忙道：
“爸爸，你一个人打不过，我们也要帮你。”
楚荣呵呵乐了：
“放心吧，这些人还不够你爸塞牙缝的呢。”
旋即，一股盛大莫沛的真气刮得竹林沙沙作响。
一个身穿中山装的中年男子背着双手走了出来，后面上百人手持各种兵刃围了上来。
中年男子不是别人，正是前来寻仇的副盟主秦汉间。
他一副倨傲表情看向楚荣一家，冷声道：
“你就是楚荣？是你伤了我侄子的手臂？”
楚荣闻知，马上明白，这一定是住在同一小区那个肥肚男寻来的帮手了。
他淡淡道：
“对呀，就是我，那个欠揍的，我见一次打一次。”
秦汉间怒极反笑：
“哈哈，小子如此狂妄，你可知我是谁吗？”
楚荣冷哼一声：
“我特么管你是谁？你们再不给我滚开，我就胖揍你们了。”
秦汉间大怒：
“来呀，把这奸人的手和脚给我各卸一只！”
说完，上百高手齐齐发出吼声，凶神恶煞冲了上来。
