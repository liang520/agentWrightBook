# 第2章 融合

“张贤，遇见就是缘，我一定让你新生，顺便帮你拍死变态老婆，替你报仇雪恨。”
楚荣意念一动，天剑宗的偷梁大法瞬间启动，一丝丝蓝色氤氲从他魂体爆射而出。
继而透过冰箱，覆盖在张贤的人体组织上。
偷梁大法属于夺舍其他人体的法术，可以暂时让神魂温养其中。
目前楚荣神魂急需肉身寄居，在找不到合适肉身的时候，张贤的碎体可以勉强考虑的。
很快，人体组织开始自动组合，首先融合的是一只右手臂。
一会儿，一只完整的右手臂从冰箱冷冻层飘到楚荣跟前。
“还不错，虽然颜值差了点，聊胜于无吧。”
楚荣面露几许喜色，心里暗道。
目光所及处，手臂组合坚实完好，这个王贤长得还是蛮壮实的。
唯一不足的是灵力太弱，导致组合处还有可怖的疤痕，犹如蛛网一般包裹着手臂。
目前因地制宜，实用为主。
他意念一动，将手臂变小，旋即装在右边肩膀上。
他挥舞了一下，感觉还比较趁手。
楚荣本想继续施法，探知王香还有哪些隐秘不为人知。
但他感觉非常乏力，灵力所剩无几，必须要休整一下。
既来之则安之，我也不急。
现在神魂的力量比较虚弱，无法走出冰箱。
必须尽快恢复实力，万一蓝星有强大的修士，或者灵星的人杀上门，我就有可能送人头。
必须把命运牢牢掌握在自己手里，决不能再犯灵星上被偷袭而杀的蠢事了。
作为修仙者，有实力才能为所欲为呀。
没实力，你就是一坨翔，苍蝇都嫌臭。
趁此机会苟在冰箱加强修炼，一边修炼神魂，一边融合张贤的肢体。
“玛德，我现在境界跌落至筑基境后期，一朝回到解放前了。”
楚荣仔细感应境界，自己现在从元婴境直接掉到筑基境，心中大恨。
作为修士，他虽然天赋异禀，但每提升一个大境界甚至一个小层次，都要历尽千辛万苦才能有机会突破。
特别是他这样重点培养的弟子，耗费宗门的资源和力量，更是不计其数。
这下倒好，一个大境界没有了，他心里直滴血。
他已在心底给七大宗门的长老画了一个叉叉，回去后这笔账一定要好好清算。
让他唯感欣慰的是，本命剑惊虹尚在紫府空间待着。
储物功能的一个酒葫芦也还在，其中装的中品和下品灵石灵草等修炼物资也都还没丢失。
七尺剑身，锋利无比，一条条道韵隐隐显现，掩盖不住的冷凛杀意，预示着它的不凡。
这是他师傅赵无极，以珍稀的上古陨石炼制七七四十九天所得。
在灵星，这些中下口灵石灵草的修炼资源不是特别珍贵。
但在两眼一摸黑的蓝星，这些灵石灵草的存在还是让楚荣有了一些底气。
他估摸着，靠着这些修复魂体和结合张贤肉身，届时突破到金丹后期大有希望。
等实力变强，扩大神识范围，仔细感应外面的世界是什么情况。
楚荣必须尽快把情况获悉，才有安全感。
他盘膝坐下，将储物袋中的灵石拿出一百枚，灵草一百株，任其飘浮空中。
然后，他开始运转天剑宗的九天化道诀，这是宗门中从炼气境到大乘境都可以运用的，功法虽不特别高明，但胜在稳定可靠。
只要你活得够长，资源够多，理论上运用这个功法，是可以摸到大乘境门坎的。
呼呼呼，只见灵石灵草犹如自动寻主一样，灵气被抽离而出，形成一道道磅礡的蓝雾。
纷纷涌入他的神魂，温养滋生他孱弱的魂体。
被吸光灵气的灵石灵草，失去光泽后，化为粉末。
一股莫名的舒爽，让楚荣忍不住“嗯，啊”了一声。
伴随着舒爽，是魂体坚固度和力量明显增强，楚荣明显感觉神识现在能扫描二十米以上的距离。
咦，他有了新发现。
这个女人的邻居，一个猥琐的大叔正在手机上兴致勃勃地看着女主播们热舞直播。
他奸笑着，挑逗着，不停地说些露骨的话刺激那些的扭腰送胯，极尽妖娆的女人。
“老婆不在家，放飞自我了。”
他说着，又打开岛国的动作片欣赏。
这个死变态坏得很，骗我看主播看片片。
楚荣有些无语，感觉污了眼睛。
他随即收回神识，继续专注修炼。
只见灵气不断温养重塑下，他的魂体从虚无变得凝实许多。
特别是那只右手臂与魂体整合出奇的适配，竟有些自己原肢的感觉。
他感觉精气神愈发旺盛，心念一动，将冷冻层的左手臂也开始重组。
如法炮制，冷冻层的人体组织在他术法下，慢慢恢复原状。
四肢、头颅、骨骼、脏器等等。
几个时辰后，一个长约30厘米长短的新张贤缓缓站起身。
冰箱太窄，他自然不能原尺寸打造，只能盘个缩水版。
只是他有些纳闷，天剑守的缩身术他以前顶多只能将自己缩成三分之二大小。
不知咋的，现在竟有些随心所欲，感觉自己可以变得更小。
可能是蓝星的规则，还有可能是重生后的新变休。
楚荣没多想。
楚荣满意地打量和检查这具躯体，除了组合处仍有蛛网疤痕，长得像蜘蛛人外，其他功能健全如初。
借助冰箱里的镜面，楚荣看到张贤的这张脸，五官端正，浓眉大眼。
竟有些与原来的自己五分想你，这让他有些惊讶。
不应该吧，应该是巧合吧。
“这张贤长得不差且有男人气质，可惜惨遭毒妇分尸，可悲可叹。”
楚荣有点惋惜，不过肉身借己一用，也算是张贤另一种方式新生。
楚荣一看到冰箱里还有些食物，就一股脑儿吃了个干净。
里面有个大西瓜、一盒红嘟嘟的草莓、还有几瓶凉茶和酸奶。
也被楚荣毫不客气的张嘴吞噬殆尽。
虽然，修士随着实力增强，对食物的依靠越来越少，吃喝只是为了尝鲜解馋而已。
平时，都是靠天材地宝维持灵力。
只要灵力在，很少有饿的感觉。
此时，楚荣看到吃的就在眼前，自然不客气，多少能摄取一些能量。
至于，王香是否会发现，他觉得不重要。
他就没打算让她走出这个房门。
尽管自己处于恢复期，但要想掌控这个女人的生死，比捏死一只蚂蚁还容易。
快至晚饭时，脚步声逐渐靠近，王香果然来开冰箱了。
楚荣将身体变小成豆，躲在一颗卷心菜中。
冰箱门被打开，王香将冰箱上层一瓶辣椒酱拿了出去，然后关上门。
听得女人自言自语道：“我觉得冰箱里少了些啥，到底少了啥呢？”
她又拉开冰箱看了看，却看不出异样，只好重重把门关上。
“我这记性越来越不好了，这个天杀的张贤，都是你害的！”
“你以为我宰了你，我就心里好过吗？我每天都做恶梦！”
“哪天要是老娘不想活了，我带着你的儿子女儿一起死！”
她像突然被刺激到一样，猛地朝冰箱冷冻室猛踹了几脚。
她在发泄着心里的怨气和不满。
楚荣又一次起了杀心，毒妇不是一般的变态。
人都被你杀了，而且还是分尸，竟然觉得自己还很委屈。
你死就算了，关键还要拖着孩子死，心肠不是一般的歹毒。
简直毫无人性。
王香恨恨到：“等老娘有空了，把你丢到河里喂鱼，免得占了冰箱。”
她扭动着肥臀，端着刚包好的饺子，走向厨房。
楚荣实在忍不住了，弹出一道剑气，直奔女人。
剑气快如闪电，哧的一声，女人右耳被整齐割掉。
顿时，血流如注。
“唉呀，我的耳朵。”
王香捂住伤处，杀猪般嚎叫起来。
“是哪个王八蛋在害我！滚出来！”
王香回头四处张望，紧张得直冒汗。
四周寂静，没有任何异常。
冰箱的压缩机突然嗡的启动，王香被吓了一跳。
她看到冰箱的冷冻层，突然汗毛倒竖。
是他？
