# 第33章 退敌

只听得惊天动地的声音后，天灵塔化为一片废墟。
半个国象寺被沦为平地，不少僧众惨被埋在倒塌的房屋下。
这样大的动静自然惊动了国府的安保力量和军警系统，四大家庭掌权者迅速出动搜救和了解情况。
再看天灵塔中心点，一片鳞片浮在空中，对面悬浮着一脸冷峻的楚荣。
住持空灵大师颤抖着看着二人斗法，看样子，他们平分秋色。
鳞片发出啧啧的声音：
“哈哈，想不到，竟能接住我神念五分之力，要是我有尊化身下界，一下就能将你拍得灰飞烟灭。”
楚荣面无表情，但内心已然波澜迭起。
他也使出七分劲，没想到还不能击溃这片鳞片。
看样子，这枚鳞片凝聚的攻击力远胜血衣殿那枚。
他哈哈一笑，道：
“痛快，好久没这样打一架了。”
“魔头，你少打嘴炮，有多么力气尽快使出来，小爷陪着你玩就是。”
鳞片呵呵冷笑，道：
“想死，我成全你！”
它低吼一声，一个血红牢笼凝聚形成，飞快朝着楚荣罩来。
楚荣冷哼一声，运足灵力全力一剑刺出。
但诡异的是，那牢笼竟然如透明空气一样，透过剑锋实体，严严实实将楚荣围在其中。
楚荣顿时感到一股万度炙热的火苗烤来，还伴有腥臭的味儿。
“困！”
那灵尊一声令下，那血笼慢慢缩小，几乎把空气挤压得哧哧作响。
隐约间，可听得楚荣的骨骼发出咯咯的声音，似乎再一使劲就会分崩离析。
楚荣自感不妙，立即运转九天仙法，灵力立刻奔涌骨骼、脏腑、筋脉之中。
那种负重和炙烤感，顿时减轻不少。
楚荣心下一松，立即挥动惊虹剑狠斩血笼。
同时，他打了一道法诀，用一道防护罩隔离旁观者视线。
既为遮挡，又为安全，免得祸及整个都城。
他每一剑都足以毁天灭地，在这种持续攻击，血笼大有崩溃之势。
那鳞片哪肯罢休，冷哼一声，不断加力，血笼又立刻凝实起来。
楚荣暗暗叫苦，照这样下去，灵力总有耗完一刻，到时就麻烦了。
这个魔修的手段堪比真仙，仅凭一道神念投射到鳞片上，就有如此大的威力。
要是他灵力耗尽，自然就离死亡不远了。
都要升到仙界的人了，被不明魔物所杀，实在有些郁闷。
他咬牙坚持，正在奋力斩笼时，腰间的葫芦却溜了出来。
“你干吗，不要乱跑，这个时候不要添乱了。”
楚荣又急又气。
哪知葫芦调皮地围着他转了一圈，一副亲昵的样子，在炙热和重力环境中竟然屁事没有。
楚荣不由大感意外，看来当初师傅给他的这只葫芦还真大有来头。
魔修看到葫芦乱入，不由呆了呆，随即狂笑：
“小子，死到临头，真是什么招都可以使得出来呀，一个臭葫芦也来凑热闹。”
他正在嘲讽之时，突然葫芦的神操作让他大跌眼镜。
只见，葫芦对着血笼一吸，牢不可破的血笼竟然化成条条面条一样，纷纷往葫芦肚中钻去。
“什么？你给我停下，本灵尊生气了！”
魔修大怒，可见这份牢笼是他精血所化，被葫芦吸走，让他元气大损。
但葫芦浑然不理，继续操作，逗得楚荣哈哈直乐。
牢笼瞬间被破，楚荣身上重压消失殆尽，毫不客气朝着那空中鳞片就是一剑。
嗖，那鳞片再也支撑不住，被一剑击碎，那魔修的声音也戛然而止。
看样子，那缕神念已经溃散。
楚荣这才松了口气，将吃掉血笼的葫芦重新系好。
再看那空灵大师，已然吓昏过去。
楚荣一手按在他头上，开始搜魂。
随着搜魂进行，他的脸色一变再变。
“魔界天鱼灵尊，还曾在灵星培植势力，不少势力还是天剑宗的死对头，说不定当初害我身死道消的与他们脱不了干系。”
楚荣冷峻的眼神愈加凌厉起来，这些仇他一刻也未曾忘记。
在上仙界之前，他一定想办法杀到灵界。
咦，还真有到灵界的通道的线索。
楚荣心头一喜，灵尊给空灵大师透露过，蓝星有几处隐秘的空间裂缝，如果实力足够强横，是有希望通往最近的灵界的。
有线索就好办了，接下来仔细搜寻就应该有收获。
同时，他也获知了灵尊这个魔修通过神念控制蓝星武者血祭的基本原因。
灵尊在魔界一次遇到多名仇家围攻，身受重伤，逃到一极远之地。
他偶然发现在另一界面的蓝星，并耗费法力控制了血光、空灵大师等人，通过谋杀武者取得精血来恢复实力。
让灵尊大为高兴的是，蓝星武者虽没有灵气加持，但精血却出奇的精纯，很快让他伤势大为好转。
为此，他有了修炼化身，或者本体来到蓝星，彻底奴役蓝星的打算。
另外，是空灵大师一些零碎的记忆，毕竟灵尊为保密只透露一些梗概。
楚荣结束搜魂，稍加回味这些珍贵线索，不由纳闷起来。
没想到武者的精血竟有如此大的功效，要是外界魔妖修炼者知道，还不得争相而来。
罢了，既然来到蓝星，我就再帮一把，把所有血祭点找出一并摧毁。
至于上界魔妖来到蓝星，一时半刻也无可能，本体或化身会受到界面压制，他们不会冒着风险而来。
楚荣打算，解决血祭点之后，他就通过空间裂缝到达灵星。
他提着空灵大师肥胖的身躯，收了防护法阵。
看到楚荣拎着空灵大师走出来，外面的人如临大敌。
一会儿功夫，就将国象寺和天灵截毁去，这样的敌手只怕总盟主也无法匹敌。
四大家族在众多好手、军警保护下盯着楚荣，后面还有十余架武装直升机、坦克、装甲车严阵以待。
这也是大炎国能拿得出手的东西了，虽然有可能对楚荣没有作用。
楚荣看着这些掌权者，将空灵大师扔给他们，淡淡一笑：
“审问一下，什么都明白了。”
“真没想到，堂堂国府旁边，竟能容下空灵这样的歹人作崇。”
空灵大师立即被几名武帝级的高手控制住，押向后方。
黄小蓉见楚荣安然无恙，有些欣喜的向黄国尊介绍道：
“父亲，这就是救我的恩公。这次他怀疑国象寺藏有古怪，这才赶来查寻，果然发现这空灵是罪魁祸首。”
黄小蓉牙尖嘴利，瞬间就给空灵定了性。
黄国尊极力掩饰内心震惊，当即向楚荣施礼：
“多谢张大师搭救小女，也要感谢揪出空灵，为大炎国除却心腹大患。”
其他三大家庭的话事人，也过来见礼，齐声赞叹。
虽然还没提审空灵，但凭楚荣和黄小蓉的定性结论，他们也隐约知道了些真相。
况且，楚荣这样逆天的实力，谁也不敢惹其不快的。
楚荣见状，挥挥手道：
“此后，你们好好管理好国家，善待国民，说不定以后还可照拂你们一二。”
几大掌权者连连称谢。
楚荣摸出几张符纸，递给黄小蓉。
“拿着吧，以后有啥紧急情况，可以烧掉联系我，我自会帮你们一次。”
说完，他冲天而起，消失在大炎上空。
黄小蓉拿到符纸后，难掩喜色。
黄国尊见状，更是喜形于色。
有了楚荣这等高手的承诺，黄家在大炎国地位更加稳固。
