# 第31章 第 31 章 “沈霁昀，你想不想做？……

江栎万万没想到, 他精心给沈父沈母准备的烛光晚餐，最后落到了他和沈霁昀头上。
　　他撑着头还在思考要怎么进行下一步计划，今天沈父和沈母对彼此的态度都还比较柔和, 也没有吵起来, 这应该算是一种好的预兆。
　　沈霁昀也没说话，默默地把面前盘子里的牛排切好块后跟江栎面前的调换了一下。
　　“谢谢。”江栎礼貌地说了句，用叉子戳了一块放进嘴里, 味道还是不错的，只是江栎心不在此。
　　“栎栎, 你很希望我爸妈和好吗？”沈霁昀认真问道, 自从江栎知道了一些内情之后, 好像对这件事格外关注, 并积极地想要推动, 为什么呢？
　　江栎思忖片刻, 但最终没把原因说出口，“不告诉你！”
　　他省去了后半句话：“免得你骄傲自满。”
　　他希望沈霁昀的家庭幸福美满, 也想给自己一些信心。在他看来, 沈父和沈母之间有争吵，也因为一时冲动选择了分开, 但如果最终还是因为对彼此的爱意走向彼此, 那他会给自己和沈霁昀的未来多一些信心。
　　不是每份爱意都会在生活中消磨殆尽的。
　　沈霁昀没有追问, 江栎很有自己的心思和想法, 他不该多加干预, 更何况他也希望父母把话说开，只是不知道该怎么做，在这件事上，江栎真的是帮了大忙。
　　他看了江栎一眼, 转移话题道：“今天准备的这些，对你来说有用吗？”
　　“啊？”江栎愣了一下，话题转换的可真快，他笑了一笑，顺着沈霁昀的话题继续说下去，“有啊，但你都说出来了，我就觉得没惊喜了。”
　　沈霁昀这个傻子，追人都不会追，哪有直接问他的？
　　沈霁昀也笑了一下，两个人有一搭没一搭地闲聊两句，吃完了晚饭。家里阿姨来收拾桌子，江栎就和沈霁昀坐到了沙发上去看电视。
　　电视里正在播一部家庭伦理剧，剧情十分狗血，但江栎还挺喜欢看的。沈霁昀坐在旁边跟着看了一会儿，没觉出来有什么乐趣，就上楼去拿了笔记本电脑下来，浏览了一下节后要进行的项目。
　　江栎转过头，凑过去看了看：“你干嘛呢？”
　　结果一看是沈氏的一些项目书，这种东西，他可不看，省得知道了一些不可外传的秘密！
　　“年后我大概会出差。”沈霁昀道，他看了一眼项目书，招标会定在了M市，他必须得亲自去。
　　江栎闻言，有些意外道：“就你一个人去？”
　　“还有公司的几个部门部长。”沈霁昀道，“我在想，你要不然就在这边住，早晚让家里的司机接送，你在市中心那边，一个人住我不放心。”
　　江栎犹豫了一下，点了下头，反正沈母对他就跟对亲儿子似的，住就住吧：“好。”
　　他半躺在沙发上，听着电视里叽叽喳喳的声音，看着沈霁昀认真处理工作的侧颜，忍不住脱口而出道：“沈霁昀，你想不想做？”
　　沈霁昀不可置信地转过头，很难想象这话是从江栎口中说出来的。
　　江栎给自己找了理由，他从来就不是一个太过重.欲的人，但也不是一点都没有，毕竟一开始能跟沈霁昀搞到床上去，多少有那张脸的原因。满打满算，他都快六个月没碰过男人了，面对着沈霁昀起些心思应该挺正常的。
　　更何况，沈霁昀今天只穿了件修身的毛衣，他工作的时候又总喜欢把袖子往上捞起露出半截手臂，修长的手指在电脑键盘上敲敲打打，江栎看着真的很难不动歪心思。
　　他想了一下，沈霁昀没道理拒绝才对。
　　沈霁昀果然合上电脑，把电脑放在茶几上后走向江栎，把人从沙发上抱起来。
　　江栎顺势双手搂住沈霁昀的脖子，明知道自己不可能会掉下去却还是不忘提醒一句：“小心别把你女儿摔着了。”
　　然后手上搂的更紧了些，把头靠在沈霁昀的肩上，任由这人把他抱上楼。
　　沈霁昀抱着江栎走到房间门口，腾不出手来开门，便由江栎代劳按下门把手后，他用脚踢开房门，稳步走进去后又踢上房门，随即把江栎放下来抵在门板上。
　　然后有些急不可耐地吻上去。
　　江栎背靠着门，一边仰起头接吻一边用手攥紧沈霁昀腰侧的毛衣，然后悄悄摸摸地把手从衣服下摆伸进去，碰到沈霁昀滚烫的身体。
　　“栎栎。”沈霁昀道，“你别勾我。”
　　他不说这话倒还好，一说江栎就有些忍不住，抓着沈霁昀的毛衣就想帮他脱掉，但被沈霁昀极力制止。
　　他有些心烦地抬起头，瞪着那双泛起情.欲的眼睛，眼尾还微微泛红，看上去毫无杀伤力：“你不做你抱我上来干嘛！”
　　沈霁昀哑口无言，他也不是不想，只是考虑到江栎现在怀着孕，做这种事怕控制不住自己。垂眸看着江栎隆起的孕肚，他叹了口气：“你要是想，我用别的办法帮你。”
　　江栎：“……”
　　想骂人的话已经在嘴边了，却硬生生被江栎自己憋了回去，难道这个时候，他还能夸一句沈霁昀真是一个正人君子吗？！
　　气死了！
　　江栎推开沈霁昀，怒道：“滚，谁要你帮！”
　　他在心里暗自骂了一句沈霁昀，然后走进厕所，锁上门后对着镜子打量自己。
　　沈霁昀不是喜欢他吗？怎么能这么坐怀不乱，难道是他没什么吸引力了？不应该啊，怀孕以后除了长胖了一点，也没什么变化啊！
　　江栎不信邪，愤愤地往厕所门那看了一眼，依稀能看见沈霁昀就站在外面，但一声不吭。
　　江栎打开了淋浴头，装作自己在洗澡，然后看着门外的身影离开，没一会儿又走回来，敲门道：“你睡衣，我放外面的凳子上了。”
　　江栎本来不想理他，但越想越气，怎么想都不想吃这个哑巴亏，于是叫住沈霁昀：“你给我递进来。”
　　沈霁昀愣了一下，江栎却已经自顾自地开门裂了条缝，就等着沈霁昀把睡衣给他递过去。他明知道这是江栎故意在引他过去，甚至把江栎的行为猜的明明白白，但还是不忍心拒绝。
　　所以被江栎拉进去的时候，他一点都不意外。
　　江栎把沈霁昀逼到角落里，双手叉腰，哼道：“沈霁昀，你要是对我没兴趣了就直说吧，我立马打包回家！”
　　这种锅从天上来的情况，沈霁昀也是有些日子没体验过了，他有点无奈道：“怎么可能。”
　　江栎一听，瞪着沈霁昀道：“那你怎么对我一点反应都没有。”
　　沈霁昀：“……”
　　江栎拖着下巴道：“距离上一次都快半年了，短短半年的时间，你不会不行了吧！”
　　沈霁昀沉了一口气，江栎真的是知道要怎么激他，偏他还吃这套。一时间，两个小人出现在他脑中互相打起架来，让他忍不住皱了下眉。随后，他托起江栎把人平稳地放在洗手台上。
　　江栎穿的宽松的裤子，他自己蹬一蹬就滑落到脚腕处了，但上身的卫衣衣摆过长，还有点挡道。
　　他见沈霁昀一副要动不动的模样，心想总不能让他自己脱吧，那多不好意思，于是晃了两下腿，无声地催促。
　　沈霁昀见状，撩开江栎的卫衣，露出光滑白皙的孕肚，手掌覆上去，感受到里面跳动的生命力，沉声道：“不舒服跟我说。”
　　江栎“哦”了一声，做了充足的心理准备。
　　但事实证明，激将法什么的，真的不能轻易用，尤其是面对沈霁昀这种狗男人！！
　　江栎有点后悔，到后来已经没什么力气了。
　　沈霁昀帮他洗澡时，盯着某处看了很久，手上的动作也有些停顿。江栎本来在浴缸里都快睡着了，又清醒过来看沈霁昀，然后道：“我想睡觉了。”
　　沈霁昀回过神来，迅速帮江栎洗完澡，把人从浴缸里捞出来，套上睡衣，吹干头发之后把人塞进被窝。
　　江栎身体上的疲惫感很重，感觉怀孕之后体力直线下降。半梦半醒的时候，他感觉沈霁昀躺上了床，搂住他后，叹了口气。
　　“你老叹气，我会觉得你没尽兴。”江栎小声道。
　　沈霁昀哑然失笑，随后摇了下头：“不是，只是刚刚……看见了你腰间的纹身，痕迹好像比以前淡了。”
　　江栎：“废话，洗了一次能不淡嘛！”
　　要不是因为后来怀孕了，他早把这傻逼纹身洗干净了。
　　突然提到这个，江栎顿时回想起来第一次的时候，沈霁昀就逮着这个地方咬，跟狗一样。
　　哦，总不会是因为知道这个纹身是他为了李然纹的，才一直惦记到现在吧。
　　江栎睁开眼睛，鄙夷地看了一眼沈霁昀：“警告你，别瞎吃飞醋。”
　　“没……”沈霁昀还想狡辩，但转念一想，有什么可狡辩的，他就是吃醋了。再怎么说，江栎跟李然在一起的那七年，在江栎的生活里都占据了很多时间，而他却只能当一个旁观者。
　　“不过沈霁昀，我得到了一个教训。”江栎道，“千万不能因为一时对感情上头就轻易在身上留下些感情的印证，早晚会后悔的！”
　　沈霁昀还以为他能说出什么名言出来，一听完后立马笑出声来，附和道：“嗯，有道理。”
　　他看着江栎的上下眼皮在打架，轻拍了拍道：“睡吧，今天很累了。”
　　江栎嘀咕：“还不是怪你。”
　　他已经完全忘记了是谁先挑起来的火，不过沈霁昀也不在意，哄哄就好了：“嗯，下次不会了。”
　　江栎哼了哼，“明早起来你要是再睡了就跑，我就带你女儿远走高飞了。”
　　他可是很记仇的！！
