# 第35章 第 35 章 希望沈霁昀会喜欢他准备……

把沈霁昀的戒指拿到手后, 第二天等人一出门，他就约着苏时一起去了昨天的珠宝店，照着戒指的尺寸, 想着挑一个好看的戒指。虽然是他要求婚, 但挑婚戒还是要看他更喜欢哪一款。
　　苏时在一旁默默地看着，一会儿指一款，但江栎都摇头表示不满意。比起那种高调奢华的大钻石, 他还是喜欢简约的。
　　“这个呢，给我看看。”江栎指着柜台里的某一款道。
　　主钻是一颗圆润饱满的钻石, 经过特殊的工艺处理, 在灯光照耀下折射出璀璨的光芒, 四周用碎钻镶嵌, 衬托出钻石的耀眼。另一只则是用碎钻镶嵌了一圈, 款式较为简约。
　　反正好看的戒指是戴在他手上的, 这款深得他心，便毫不犹豫地买单付款。
　　买完戒指后, 江栎又开始头疼了, 要怎么把沈霁昀的这只戒指还回去呢？
　　苏时建议道：“要不然干脆别换了，你都给人家套脖子上了, 怪麻烦的。”
　　“你认真的吗？”江栎鄙夷道, 虽然也不是不行, 可万一在他求婚之前的这段时间, 沈霁昀发现了怎么办？
　　苏时点点头道：“你放心, 他发现不了一点。”
　　因为即便发现了，沈霁昀也会装成不知道的模样！
　　江栎撇嘴道：“行吧，那不换了。”
　　他心安理得地带着新买的戒指回家，却没想到原本应该在公司的人正坐在家里。
　　江栎吓了一跳, 走过去道：“你不是上班去了吗？”
　　“今天公司大楼停电，放假了。”沈霁昀把人捞进怀里，习惯性地在江栎嘴边嘬了一口道，“干嘛去了？”
　　“跟苏时出去逛街。”江栎脸红，转移话题道，“爸妈呢？”
　　沈霁昀：“爸今天难得休息，和妈出去了。”
　　自打上一次两人谈过一番后，沈母就是一副要和好不和好的态度，说是想重新体验一下年轻时的生活。沈父会了意，本想着找机会和沈母像以前一样找一下谈恋爱的时候的感觉，可惜一直没时间，今天可算是等到机会了。
　　江栎“哦”了一声，想从沈霁昀怀里出来，毕竟现在他外套兜里还装着戒指盒，要是一不小心掉出来他的精心准备就白费了！
　　“我上楼去换衣服。”江栎找借口开溜道。
　　沈霁昀松开手，突然意义不明地说了一句：“栎栎，我今天很闲。”
　　江栎愣了一下，没明白这话的意思，随口应了一声后就上楼去藏东西了。
　　沈霁昀有些无奈地笑了笑，江栎什么时候才能发现他自己偷偷摸摸做什么的时候一点都藏不住。
　　但是沈霁昀并不打算戳穿江栎，他还挺乐意见到江栎这副可爱的模样的。
　　江栎在房间里到处找，最后把戒指藏在了他的那半衣柜的角落里，还用一堆叠好的衣服挡在前面。做完这些，他满意地拍了拍手，只要不是特意来翻他的衣柜，肯定发现不了。
　　正准备出房间去找沈霁昀，对方却跟着他一起上来，江栎顿时心虚地瞟了一眼衣柜的方向，然后不动声色地挪动了一下位置到：“干嘛。”
　　“中午想吃什么？我给你做。”沈霁昀问。
　　江栎想，自从搬回来住以后，沈霁昀已经很久没有下过厨做饭了，于是开始点菜，恨不得把沈霁昀拿手的菜通通点一遍，最后还笑着挽上沈霁昀的胳膊道：“我帮你打下手。”
　　然后把人带离了房间，其中意思太明显，但也顾不聊这么多。
　　好在沈霁昀也没说什么，牵着江栎一起去厨房，从橱柜里找了件新的围裙给江栎。江栎在家只穿了件薄绒的家居服，宽松的家居服遮挡了隆起的孕肚，但系上围裙的带子后，就有些显现出来了。
　　他摸了摸肚子看向沈霁昀道：“女儿说她饿了，让你快点做饭。”
　　沈霁昀笑出声，从冰箱里拿了些食材出来，先把米饭蒸上后，就开始自顾自地准备食材，压根就没想让江栎上手帮忙。
　　江栎琢磨着，他站在这套着一件围裙，不做些什么也太没意思了，于是就到处晃一晃，沈霁昀切菜的时候他就站在旁边探头：“不需要我帮忙吗？”
　　“不用。”沈霁昀道，“你要是想做，等会你来炒菜？”
　　江栎立马点点头，等沈霁昀食材准备好后，站到炉灶边把锅里的水先擦干净。
　　沈霁昀把食材放到旁边的台子上，站在江栎身边，开火把油倒进锅里，随后指使江栎把食材倒进锅里。
　　江栎拿起食材往锅里一倒，食材上的水与锅里的热油一碰，顿时溅了一些起来，江栎一下子躲开，惊道：“好吓人。”
　　沈霁昀拿锅铲把锅里的食材搅动了一下，递给江栎道：“来吧，不会溅了。”
　　江栎用家居服的衣袖裹住手去拿锅铲，整个上半身离得远远的，幸好手长。
　　江栎全程就只拿着锅铲搅动食材，放调料都是沈霁昀来做。最后出锅时，江栎看了看成品道：“我真棒。”
　　沈霁昀附和：“是不错。”
　　江栎满意地把成品端到保温箱里放着，说什么都不想炒下一个菜了，于是自顾自取下围裙，站在边上督工。
　　沈霁昀一个人做饭的速度明显加快，炒了一个素菜之后又煮了汤，最后全部端上桌。
　　兴许是自己出了点力，江栎吃饭的时候感觉今天的这顿饭格外好吃，吃了满满的一碗米饭。
　　他给自己盛了一碗汤，小口喝的时候听见沈霁昀漫不经心地开口道：“栎栎，你的戒指呢？”
　　“啊？”江栎变得有些慌乱，他的戒指现在在沈霁昀那，而沈霁昀的那一只他用完之后本来想着还回去，也就没有戴回项链上。
　　他这件家居服的领口比较低，所以沈霁昀一早就发现了他脖子上空空如也，只是一直没有说。
　　“我放房间了。”江栎吞吞吐吐道。
　　沈霁昀微微笑着道：“没弄丢就行。”
　　“当然不会弄丢。”江栎小声嘀咕了一句，突然直觉沈霁昀突然提到戒指，奇奇怪怪的。
　　难道是知道了他的计划？不应该啊，他藏的挺好的，沈霁昀根本就无从发现嘛！还是说苏时那个大嘴巴，又把他出卖了？
　　“你最近跟苏时联系了吗？”江栎试探道。
　　沈霁昀：“没有，怎么了？”
　　江栎撇了撇嘴，总感觉很奇怪，但又说不上来。但他怕再问下去，自己会一不小心把事情透露出去，便闭上嘴安静喝汤。
　　吃完饭后，江栎躲回房间锁上门，打了个电话给苏时，慌乱道：“怎么办，感觉沈霁昀发现了什么！”
　　苏时道：“真的假的，你别自己吓自己。”
　　江栎也不知道是不是因为自己太紧张了，但是以防万一，拖得越久越容易被发现，还是赶紧提上日程好了。
　　“这样吧，你找个理由，约沈霁昀去那吃饭，然后我提前去准备。”江栎吩咐道。
　　“这么信任我！”苏时自豪道。
　　江栎威胁道：“要是说漏嘴了，我就把你踢出我的好友行列。”
　　苏时：“不会的，我保证一个字都不会说！”
　　他的语气很认真，江栎便放下心来，挂掉电话后没多久，沈霁昀就上来找他，发现他锁了门后敲门喊道：“栎栎。”
　　“来了。”江栎从地上爬起来去开门，“怎么啦？”
　　“苏时说下午有活动，晚上一起吃饭，去吗？”沈霁昀含笑问道。
　　江栎忽视掉沈霁昀这意味深长的笑容，摇了摇头道：“我不去，好累。”
　　“怎么了？不舒服？”沈霁昀搂住他关心道，“那我也不去了，在家里陪你。”
　　江栎猛地抬头，惊讶道：“不行！”
　　说完就有些后悔，他反应好像过激了，于是收了收情绪道：“苏时好不容易约你出去，你别不给他面子啊。”
　　沈霁昀扬起的嘴角都下不来，他揉了揉江栎的头道：“可我放心不下你一个人在家。”
　　江栎心里甜滋滋的，仰头亲了沈霁昀一下道：“没事，你放心出去吧。”
　　沈霁昀还是搂着他不放手，江栎眼看着时间一点点流逝，心里忍不住有些着急。他还要去现场布置，沈霁昀一直不出门，那他怎么办！
　　“你快去。”江栎轻轻地推了推沈霁昀，小声催促道。
　　沈霁昀松开他，似笑非笑地看着江栎，再次确认道：“那我去了？”
　　江栎果断地点了两下头，朝沈霁昀挥了挥手后，等人从卧室出去后又等了片刻，才探头出去，趴在二楼的栏杆上静静看着沈霁昀出门。
　　确认沈霁昀已经上车走远后，江栎火速换了衣服，打车去了他订好的餐厅。
　　苏时不在，很多东西只能他自己准备了。
　　他把U盘插到电脑上，把剪辑好的视频播了一遍，从小的时候关系还算良好的合照，到后来每一次聚会上被迫站在一起的合照，时光在他们身上同等变化，一起长大的好处大概也就是这些了。
　　视频的结尾，还有江栎自己录的一段真心话，那些当着沈霁昀的面不好意思说出来的话，都在视频里了。
　　检查完视频，江栎又把包间里粘的气球重新粘了一遍，这些气球太容易掉落下来了，希望别中途掉链子，打乱他的节奏。
　　万事俱备，只欠东风。
　　江栎眼看着时间差不多，给苏时发消息过去：[我OK了。]
　　半晌，苏时回复：[好嘞祖宗，马上把你对象带过来！]
　　江栎走到门口关上门，仔细地打量了一下自己的布置，希望沈霁昀会喜欢他准备的惊喜。
